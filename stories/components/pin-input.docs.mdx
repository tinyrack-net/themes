# PinInput

Multi-digit verification input with paste normalization and deterministic focus movement.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `data-tr-pin-input` | `PinInput` | Six digits by default | One input per digit. |
| Part | `tr-pin-input-digit` | Generated digit inputs | Numeric text input | First input advertises one-time-code autocomplete. |
| Behavior | `createPinInputManager` | Shared DOM manager | Input, paste, Backspace, Delete, arrows | Emits change and complete events. |
| State | `data-invalid`, native disabled | `invalid`, `disabled`, controlled or uncontrolled value | Explicit | Pair invalid state with visible error text. |
