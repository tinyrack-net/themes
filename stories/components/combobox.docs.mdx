# Combobox

Filterable selection with keyboard navigation, empty state, disabled options, and a positioned popup.

## Contract

| State | Contract |
| --- | --- |
| value | controlled `value` or uncontrolled `defaultValue` |
| popup | controlled `open` or uncontrolled `defaultOpen` |
| items | `items` on `Root`, matching `value` on each `Item` |
| parts | `Root`, `Input`, `Trigger`, `Portal`, `Positioner`, `Popup`, `List`, `Item`, `Empty` |
