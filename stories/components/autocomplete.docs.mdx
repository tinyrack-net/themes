# Autocomplete

Accepts free-form text while filtering and announcing suggestions from a known collection.

## Contract

| Axis | Behavior |
| --- | --- |
| value | the text value remains editable even when it does not match an item |
| input anatomy | InputGroup combines Input, Clear, Trigger, and Icon |
| popup anatomy | Status announces results; List contains groups, items, separators, and Empty feedback |

        <Autocomplete.Root items={['Rack Alpha', 'Rack Beta']} name="rack-search" onValueChange={setValue} required value={value}>

          <Autocomplete.Portal><Autocomplete.Positioner><Autocomplete.Popup><Autocomplete.Status /><Autocomplete.List>
            <Autocomplete.Item value="Rack Alpha">Rack Alpha</Autocomplete.Item>
            <Autocomplete.Item value="Rack Beta">Rack Beta</Autocomplete.Item>
            <Autocomplete.Empty>No matching racks</Autocomplete.Empty>
          </Autocomplete.List></Autocomplete.Popup></Autocomplete.Positioner></Autocomplete.Portal>
        </Autocomplete.Root>
        <Field.Error match>Enter a rack name.</Field.Error>
      </Field.Root>
      <Button type="submit">Continue</Button>
    </Form>
