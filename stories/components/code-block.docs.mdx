# CodeBlock

SSR-safe block code with client-side Shiki highlighting and a readable plain-text fallback.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `tr-code-block` on `<pre><code>` | `<CodeBlock>` | Shiki-backed block surface; SSR-safe | Keep code readable before hydration and when highlighting is unavailable. |
| Behavior | Generated token spans | `code` | `string` source text | Pass the exact copyable source string. |
| Behavior | `data-language` | `language` | Shiki bundled language; default unset | Provide a language when syntax coloring is useful. |
| Behavior | `data-highlighted="true"` | `theme` | Bundled Shiki theme; default `github-dark` | Use a theme that matches the surrounding product surface. |
| Behavior | `data-wrap="true"` | `wrap` | `boolean`; default `false` | Leave commands and source scrollable; wrap prose-like output deliberately. |
| Native | Native `<pre>` attributes | `HTMLAttributes<HTMLPreElement>` | Passed through | Preserve whitespace and copyable source text. |

## Usage

<ComponentExampleTabs
  id="code-block-basic"
  title="Highlighted block with plain fallback"
  description="CodeBlock renders readable source during SSR, then applies Shiki tokens in the browser when the language is supported."
  previewLayout="stretch"
  preview={(
    <CodeBlock code={'pnpm install\npnpm verify'} language="sh" />
  )}
  sources={[
    {
      label: 'React',
      language: 'tsx',
      code: String.raw`import '@tinyrack/ui/core/core.css';
import '@tinyrack/ui/components/code-block/code-block.css';
import { CodeBlock } from '@tinyrack/ui/components/code-block/react';

<CodeBlock
  code={'pnpm install\npnpm verify'}
  language="sh"
  theme="github-dark"
/>;`,
    },
    {
      label: 'HTML',
      language: 'html',
      code: String.raw`<pre class="tr-code-block" data-language="sh">
  <code>pnpm install
pnpm verify</code>
</pre>`,
    },
  ]}
/>

## Guidance

<ComponentGuidance
  whenToUse={[
    'Present multiline commands, configuration, logs, or source that readers may copy.',
    'Pass a supported language when syntax highlighting improves comprehension.',
  ]}
  avoid={[
    'Do not use a block for a short literal embedded in prose; use Code instead.',
    'Do not enable wrapping for source where indentation or aligned columns carry meaning.',
  ]}
  accessibility={[
    'Keep the pre and code structure so whitespace, keyboard selection, and copied text remain predictable.',
    'Treat syntax color as enhancement only; the plain SSR output must stay readable, and long unwrapped lines must scroll inside the block.',
  ]}
/>

## API

| Prop | Values | Note |
| --- | --- | --- |
| `code` | `string` | Source text to render and highlight. |
| `language` | Shiki bundled language | Sets `data-language`; omitted languages remain plain text. |
| `theme` | Shiki bundled theme | Default `github-dark`. |
| `wrap` | `boolean` | Adds `data-wrap="true"`. |
| `className` | `string` | Merged with `tr-code-block`. |
