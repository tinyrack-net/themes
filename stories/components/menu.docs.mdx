# Menu

Keyboard-navigable actions and links in a native top-layer popover.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `data-tr-menu` | `Menu` | One trigger and content | Use for compact action collections. |
| Part | `data-tr-menu-trigger` | `MenuTrigger` | Native button or `asChild` | Arrow Down or Up opens at an edge. |
| Part | `tr-menu-content`, `role="menu"` | `MenuContent` | Native popover | Top-layer position follows Popover. |
| Part | `tr-menu-item`, `role="menuitem"` | `MenuItem` | Button or link, optional disabled | Selection closes unless the select event is canceled. |
| Behavior | DOM manager | Shared DOM manager | Arrows, Home, End, typeahead | Roving focus skips disabled items. |
