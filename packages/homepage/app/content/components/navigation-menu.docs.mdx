# NavigationMenu

Site-level navigation with links and optional rich popup content.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Playground controls cover `label`, `disabled`, and the controlled `openSection` value reflected by real trigger interaction.

## API

Every compound part and prop/state type is exported from `@tinyrack/ui/components/navigation-menu`; styles remain opt-in through the matching CSS subpath.

NavigationMenu is intended for pointer-and-keyboard site navigation with enough horizontal room. At narrow breakpoints, swap the composition for a Menu or Drawer rather than clipping or visually reflowing the list without also changing its keyboard model.
