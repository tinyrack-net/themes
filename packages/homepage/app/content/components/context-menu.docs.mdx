# ContextMenu

A right-click menu with selection, groups, links, and submenus.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Playground controls cover `label`, `open`, `disabledItem`.

## API

Every compound part and prop/state type is exported from `@tinyrack/ui/components/context-menu`; styles remain opt-in through the matching CSS subpath. Render the trigger as a focusable control so keyboard users can use the platform context-menu command, and provide an explicit click/tap action that opens the same commands on touch devices.
