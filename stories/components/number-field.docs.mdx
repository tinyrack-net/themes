# NumberField

Numeric input with keyboard, stepper, scrub, and form behavior.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Storybook controls cover `label`, `value`, `disabled`.
