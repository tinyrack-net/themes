## Contract

The component owns only selection behavior. Locale route calculation, navigation, and storage stay in the consuming application.

## Examples

Resolve the target URL from a shared content key before changing the controlled locale.

## API

`value` and `onValueChange` are required. Each option may include a BCP 47 `language` for its rendered label.
