## Contract

Use TRNavigationMenu for a site header with a brand, a small number of direct destinations, and optional Product or Resources flyouts. The desktop composition keeps Base UI roving keyboard navigation; the mobile composition exposes the same destinations in a TRDrawer.

## API

The complete anatomy is `Root`, `List`, `Item`, `Trigger`, `Icon`, `Content`, `Portal`, `Backdrop`, `Positioner`, `Popup`, `Arrow`, `Viewport`, and `Link`. Every part, props type, and state type is a named export and a member of `TRNavigationMenu` where applicable.

- `Root`: use `value` with `onValueChange` for controlled open state, or `defaultValue` for uncontrolled state. `orientation` changes both roving-focus keys and layout; `delay` and `closeDelay` tune pointer intent.
- `Link`: set `active` for the current destination (`aria-current="page"`) and `closeOnClick` for links rendered inside flyout content.
- `Positioner`: `side`, `align`, offsets, collision padding, and `collisionAvoidance` control placement. The Tinyrack default shifts alignment and flips sides to keep the popup visible.
- `Portal`: optionally accepts a custom container. Place `Backdrop` and `Positioner` inside it; `Popup` owns the visual surface, while `Viewport` receives the active `Content`.
- Styling: import `@tinyrack/ui/components/navigation-menu.css`. Customize with `--tr-navigation-menu-color`, `--tr-navigation-menu-control-background`, `--tr-navigation-menu-popup-background`, `--tr-navigation-menu-popup-max-width`, and `--tr-navigation-menu-backdrop`; each falls back to semantic Tinyrack tokens.
- SSR: the composition is server-renderable and hydration-safe. Portal content mounts into its container after hydration.

TRNavigationMenu is intended for pointer-and-keyboard site navigation with enough horizontal room. Keep flyout groups task-oriented and preserve direct links for destinations that need no preview. At narrow breakpoints, swap the composition for a TRDrawer with the same destination set instead of compressing the horizontal keyboard model.
