## コントラクト

| 軸 | 値 | 既定値 |
| --- | --- | --- |
| `variant` | `default`, `outlined`, `elevated` | `default` |
| `padding` | `none`, `sm`, `md`, `lg` | `md` |
| parts | `Root`, `Header`, `Title`, `Description`, `Content`, `Footer` | compound API |

## API

| パート | 既定の要素 | 主な props |
| --- | --- | --- |
| `TRCard.Root` | `div` | `variant`, `padding`, `render` |
| `TRCard.Header` | `header` | ネイティブ header props |
| `TRCard.Title` | `h2` | 文脈に合う heading level のための `render` |
| `TRCard.Description` | `p` | ネイティブ paragraph props |
| `TRCard.Content` | `div` | ネイティブ div props |
| `TRCard.Footer` | `footer` | ネイティブ footer props |

`TRCard.Root` と `TRCard.Title` は wrapper node を追加しない Base UI の additive `render` contract をサポートします。すべてのパートは `className` をマージし、ネイティブ props と `ref` を転送し、consumer style を受け取ります。Card を構成するときは heading の文書順を守り、child を card edge まで広げる場合にのみ `padding="none"` を使用してください。
