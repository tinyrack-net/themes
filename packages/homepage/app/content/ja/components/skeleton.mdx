## コントラクト

`shape` は `text`、`rectangle`、`circle` を受け取り、既定値は `text` です。TRSkeleton は既定で shimmer し、静的な placeholder には `animate={false}` を設定します。サイズは layout または `style` から決まります。

## API

An unlabeled TRSkeleton has `aria-hidden="true"`; adding `aria-label`, `aria-labelledby`, or an explicit status role opts into one live region. The shimmer is disabled automatically when the user prefers reduced motion.

`--tinyrack-skeleton-fill` と `--tinyrack-skeleton-highlight` は theme-aware semantic token です。loading surface にコンポーネント単位の contrast 調整が必要な場合は `--tr-skeleton-fill` または `--tr-skeleton-highlight` を local override します。
