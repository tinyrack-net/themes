# Toolbar

Groups related controls with roving keyboard focus.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Storybook controls cover `label`, `disabled`, `orientation`, and `loopFocus`.

## Install

<ComponentInstall surfaces={[{ label: 'React', install: 'pnpm add @tinyrack/ui react react-dom', imports: ["import '@tinyrack/ui/core.css';", "import '@tinyrack/ui/components/toolbar.css';", "import { Toolbar } from '@tinyrack/ui/components/toolbar';"] }]} />
