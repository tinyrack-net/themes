## 계약

| 축 | 값 | 기본값 |
| --- | --- | --- |
| `variant` | `neutral`, `info`, `success`, `warning`, `danger` | `neutral` |
| `role` | `status`, `alert` 같은 native `div` role | 없음 |
| parts | `Root`, `Title`, `Description`, `Actions` | compound API |

## API

`TRAlert.Root`와 `TRAlert.Actions`는 native `div` props와 React 19 ref를 전달합니다. 알림의 긴급성은 애플리케이션 맥락이므로 `TRAlert.Root`는 live-region role을 자동으로 지정하지 않습니다. 기본 `div`, `p` 요소가 주변 의미에 맞지 않을 때 `TRAlert.Title`과 `TRAlert.Description`은 Base UI `render` contract를 지원합니다.
