### モーションの低減

Reduced motion は装飾的な移動を取り除きますが、状態変化やその意味は残します。

<ReducedMotionPreview />

## 実装

コンポーネントは semantic duration と easing variable を使います。コンポーネント境界に 1 つの transition を置き、reduced-motion 設定で装飾的な移動を除去します。

```css
.rack-layer {
  transition:
    opacity var(--tinyrack-duration-normal) var(--tinyrack-ease-out),
    transform var(--tinyrack-duration-normal) var(--tinyrack-ease-out);
}

@media (prefers-reduced-motion: reduce) {
  .rack-layer {
    transition-duration: 1ms;
    transform: none;
  }
}
```

システム全体の timing には `--tinyrack-duration-*` を使い、意図した local exception にだけコンポーネントレベルの `--tr-*` variable を使います。
