# AppShell

A responsive application frame with a static desktop sidebar and modal mobile navigation.

## Contract

`Main` owns the page's single main landmark. `Sidebar` is static on desktop and uses Base UI Drawer for mobile focus, dismissal, scroll lock, and focus return.

## API

`Root` accepts controlled or uncontrolled open state, `sm` or `lg` breakpoints, and `header-first` or `sidebar-first` layout. `Trigger` and `Close` require an accessible name. `Sidebar` contains a plain `ScrollArea` and mobile modal Drawer.
