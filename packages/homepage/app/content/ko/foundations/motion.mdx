### 동작 줄이기

Reduced motion은 장식적인 이동만 제거하며 상태 변화나 그 의미는 제거하지 않습니다.

<ReducedMotionPreview />

## 구현

컴포넌트는 semantic duration과 easing variable을 사용합니다. 컴포넌트 경계에 하나의 transition을 두고 reduced-motion 설정이 장식적 이동을 제거하게 하세요.

```css
.rack-layer {
  transition:
    opacity var(--tinyrack-duration-normal) var(--tinyrack-ease-out),
    transform var(--tinyrack-duration-normal) var(--tinyrack-ease-out);
}

@media (prefers-reduced-motion: reduce) {
  .rack-layer {
    transition-duration: 1ms;
    transform: none;
  }
}
```

시스템 전체 timing에는 `--tinyrack-duration-*`를 사용하고, 의도적인 local exception에만 컴포넌트 수준 `--tr-*` variable을 사용하세요.
