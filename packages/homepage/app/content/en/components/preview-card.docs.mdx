## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Playground controls cover `label`, controlled `open`, `delay`, `closeDelay`, `side`, and `align`.

`Trigger` waits `600ms` before opening and `300ms` before closing by default.
Override those values with `delay` and `closeDelay` only when the trigger-to-card
pointer path remains forgiving.

## API

Every compound part and prop/state type is exported from `@tinyrack/ui/components/preview-card`; styles remain opt-in through the matching CSS subpath. `Trigger` owns the anchor destination plus the `delay` and `closeDelay` timing props, while `Root` owns controlled or uncontrolled open state. Keep interactive preview content keyboard reachable and dismissible with Escape.

TRPreviewCard is a progressive enhancement for mouse hover and keyboard focus, not
a touch-only disclosure. The trigger remains a real link: tapping it follows its
`href`, so do not put required information or the only path to an action inside
the preview. Provide a separate button/dialog pattern when touch users must open
the supplemental surface without navigating.
