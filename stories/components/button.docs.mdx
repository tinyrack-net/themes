# Button

Native button behavior with a small Tinyrack visual contract.

## Contract

| Area | Values | Rule |
| --- | --- | --- |
| Size | `sm`, `md`, `lg` | Use `md` for normal actions. |
| Variant | `neutral`, `primary`, `danger` | Use `primary` once per action group. |
| Appearance | `solid`, `outline`, `ghost` | Choose emphasis without changing semantics. |
| Disabled | `boolean` | Uses the native disabled state. |
| IconButton | `label` required | Square action for icon-only controls. |

## Usage

```tsx
import '@tinyrack/ui/core/core.css';
import '@tinyrack/ui/components/button/button.css';
import { Button, IconButton } from '@tinyrack/ui/components/button/react';

<Button size="md" variant="primary" appearance="solid">
  Deploy
</Button>;

<IconButton label="Refresh rack">R</IconButton>;
```

```html
<button
  class="tr-btn"
  data-size="md"
  data-variant="primary"
  data-appearance="solid"
>
  Deploy
</button>
```

## API

| Prop | Values | Note |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | Default `md`. |
| `variant` | `neutral`, `primary`, `danger` | Default `neutral`. |
| `appearance` | `solid`, `outline`, `ghost` | Default `solid`. |
| `label` | `string` | Required for `IconButton` accessible names. |
| `disabled` | `boolean` | Native disabled state. |
| `type` | `button`, `submit`, `reset` | Default `button`. |
