## 필수 의존성 설치

Tinyrack UI는 React 19와 Tailwind CSS 4를 사용합니다. UI 패키지, React,
Tailwind CSS와 컴포넌트 및 예시의 아이콘에 사용하는 Lucide React를 함께
설치하세요.

```shellscript
pnpm add @tinyrack/ui tailwindcss react react-dom lucide-react
```

Tinyrack UI는 Tailwind CSS 4.3 이상, 5 미만을 지원합니다.

## 기본 스타일 불러오기

애플리케이션의 전역 진입점에서 기본 스타일시트를 한 번 불러옵니다.

```tsx
import '@tinyrack/ui/core.css';
```

컴포넌트 스타일에서 Tinyrack 토큰과 Tailwind variant를 사용할 수 있도록
`core.css`를 모든 컴포넌트 스타일보다 먼저 불러와야 합니다.

## 컴포넌트 스타일 불러오기

컴포넌트 스타일은 JavaScript와 별도로 배포되며 자동으로 포함되지 않습니다.
애플리케이션에서 사용하는 각 컴포넌트의 스타일시트를 불러오세요.

예를 들어 Button을 사용한다면 다음과 같이 불러옵니다.

```tsx
import '@tinyrack/ui/core.css';
import '@tinyrack/ui/components/button.css';
```

추가 컴포넌트의 스타일은 `core.css` 다음에 나열합니다.

```tsx
import '@tinyrack/ui/core.css';
import '@tinyrack/ui/components/button.css';
import '@tinyrack/ui/components/input.css';
import '@tinyrack/ui/components/dialog.css';
```

모든 컴포넌트 스타일을 한꺼번에 불러오기보다 실제 사용하는 컴포넌트의
스타일만 불러오는 것을 권장합니다.

## 컴포넌트 사용하기

컴포넌트는 각각의 public subpath에서 가져옵니다.

```tsx
import { TRButton } from '@tinyrack/ui/components/button';

export function DeployButton() {
  return <TRButton variant="primary">변경 사항 배포</TRButton>;
}
```
