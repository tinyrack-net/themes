# Disclosure

Native expandable content with no JavaScript requirement.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `details.tr-disclosure` | `Disclosure` | Native closed state | Preserve native `open` and `toggle`. |
| Part | `summary.tr-disclosure-summary` | `DisclosureSummary` | Required first summary | Use a concise visible label. |
| Part | `tr-disclosure-content` | `DisclosureContent` | Flow content | Keep forms and headings semantically valid. |
