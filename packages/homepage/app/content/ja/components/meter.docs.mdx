## コントラクト

この Tinyrack wrapper は Base UI の動作、React 19 refs、ネイティブ props、events、`render` contract を保ちます。Playground control は不正な範囲を正規化し、`label`、`min`、`max`、`value`、number formatting、すべての明示的な semantic variant に対応します。

## API

現在の `min`/`max` 範囲内に `value` を保ち、表示単位には `format`、文脈に応じた通知には `getAriaValueText` を使います。`neutral`、`info`、`success`、`warning`、`danger` は明示的で、Tinyrack は暗黙の low/high threshold API を追加しません。
