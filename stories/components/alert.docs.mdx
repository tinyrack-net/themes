# Alert

Persistent status messaging with semantic titles, descriptions, and optional actions.

## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `variant` | `neutral`, `info`, `success`, `warning`, `danger` | `neutral` |
| parts | `Root`, `Title`, `Description`, `Actions` | compound API |

## Install

<ComponentInstall surfaces={[{ label: 'React', install: 'pnpm add @tinyrack/ui react react-dom', imports: ["import '@tinyrack/ui/core.css';", "import '@tinyrack/ui/components/alert.css';", "import { Alert } from '@tinyrack/ui/components/alert';"] }]} />

## API

`Alert.Root` forwards native `div` props and React 19 `ref`; it does not assign a live-region role automatically because announcement urgency is application context.
