# Divider

Semantic separators for horizontal sections and vertical control groups.

## Contract

`orientation` accepts `horizontal` or `vertical` and defaults to `horizontal`.

## API

Vertical dividers set separator semantics and `aria-orientation="vertical"` automatically.
