## 계약

`aria-label` 또는 `aria-labelledby`를 제공하세요. appearance, variant, size, loading, disabled, ref, event는 TRButton과 동일합니다.

const sizes = ['sm', 'md', 'lg'] as const;

export function IconButtonMatrix() {
  return appearances.map((appearance) => <div key={appearance}>
    {variants.flatMap((variant) => sizes.map((size) => <TRIconButton
      appearance={appearance}
      aria-label={appearance + ' ' + variant + ' ' + size}
      key={variant + '-' + size}
      uiSize={size}
      variant={variant}
    >
      <SettingsIcon aria-hidden="true" />
    </TRIconButton>))}
  </div>);
}` }]}

## API

`TRIconButton`은 TRButton contract를 따르지만 정확히 하나의 접근 가능한 naming prop이 필요합니다. CSS subpath에는 TRButton CSS가 포함됩니다.
