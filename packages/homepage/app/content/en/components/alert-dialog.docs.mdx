## API

Every compound part and prop/state type is exported from `@tinyrack/ui/components/alert-dialog`; styles remain opt-in through the matching CSS subpath. Group confirmation controls in `tr-alert-dialog-actions`; they wrap into full readable actions on narrow viewports. Use a secondary Button for Cancel and a danger Button for the destructive confirmation.
