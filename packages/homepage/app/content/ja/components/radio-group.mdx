## コントラクト

| 軸 | 動作 |
| --- | --- |
| value | uncontrolled with `defaultValue`, controlled with `value` and `onValueChange` |
| keyboard | arrow keys move selection; Tab enters and leaves the group once |
| form | `name` and `required` apply to the group; individual radios provide values |

## API

`TRRadioGroup` と prop/state type は `@tinyrack/ui/components/radio-group` から export されます。`TRRadio.Root` と `TRRadio.Indicator` で構成します。root は ref、ネイティブ props、state に基づく `className`、`style`、`render` を保ち、矢印キーで選択を移動し、Tab はグループへ 1 回入って 1 回出ます。
