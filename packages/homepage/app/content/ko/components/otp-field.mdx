## 계약

| 축 | 동작 |
| --- | --- |
| value | uncontrolled with `defaultValue`, controlled with `value` and `onValueChange` |
| input | supports typing, full-code paste, autofill, and one native form value |
| length | `length` controls the rendered inputs and native required validity |

    <TRForm noValidate onSubmit={(event) => {

      event.currentTarget.checkValidity();
      setAttempted(true);

      <TRButton type="submit">Verify</TRButton>

## API

`TROTPField.Root`, `TROTPField.Input`, `TROTPField.Separator` namespace member를 사용하세요. 같은 subpath는 접두사가 붙은 `TROTPFieldRoot`, `TROTPFieldInput`, `TROTPFieldSeparator` binding과 prop/state type도 export하지만 bare `Root`, `TRInput`, `TRSeparator` binding은 export하지 않습니다.
