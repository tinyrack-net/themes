## コントラクト

データソースは `onSearch(query, signal)` が管理します。TRDialog は Ctrl/⌘+K、読み込み中、空の結果、fallback、ハイライト、選択、フォーカス、リクエストのキャンセルを管理します。

## 例

主要な検索インデックスを利用できず、結果が少なくなったことを説明するには `fallback` を設定します。

## API

表示されるすべてのメッセージをローカライズできます。`fallback` は主要な Pagefind ソースを利用できないことを伝えます。
