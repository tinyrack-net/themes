## コントラクト

このコンポーネントは storage を読み取ったり document を変更したりしません。アプリケーションアダプターで `value` を渡し、`onValueChange` を保存します。

<ComponentInstall surfaces={[{ label: 'React', install: 'pnpm add @tinyrack/ui tailwindcss react react-dom', imports: ["import '@tinyrack/ui/core.css';", "import '@tinyrack/ui/components/color-scheme-toggle.css';", "import { TRColorSchemeToggle } from '@tinyrack/ui/components/color-scheme-toggle';"] }]} />

## 使用方法

<ComponentExampleTabs id="color-scheme-toggle-basic" title="制御付き scheme" preview={<Stories.ColorSchemeTogglePreview value="light" />} sources={[{ label: 'React', language: 'tsx', code: String.raw`<TRColorSchemeToggle value={scheme} onValueChange={setScheme} />` }]} />

## 例

選択した scheme をアプリケーションアダプターに保存し、`value` で再び渡します。

## API

`value` は `light | dark` です。label、size、disabled 状態、class、button ref を設定できます。
