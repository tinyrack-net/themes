# Skeleton

A loading placeholder that preserves layout shape while content is unavailable.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `tr-skeleton` on `<div>` | `Skeleton` | `aria-hidden="true"` by default | Pair with parent `aria-busy` while loading. |
| Shape | `data-shape` | `shape` | `text`, `rectangle`, `circle`; default `text` | Match the placeholder to the content it replaces. |
| Motion | `data-animate` | `animate` | `true`, `false`; default `true` | Respect `prefers-reduced-motion` or disable animation explicitly. |
| Native | Native `<div>` attributes | `HTMLAttributes<HTMLDivElement>` | Passed through | Set dimensions with layout utilities or inline styles when needed. |

## Guidance

<ComponentGuidance
  whenToUse={[
    'Use Skeleton when the eventual content shape is known and preserving layout reduces shift.',
    'Use a static skeleton when motion does not add useful feedback.',
  ]}
  avoid={[
    'Do not expose placeholder text to assistive technology as if it were real content.',
    'Do not use Skeleton for an operation whose progress should be communicated with Progress.',
  ]}
  accessibility={[
    'Keep the default aria-hidden behavior and mark the loading region with aria-busy.',
    'Respect prefers-reduced-motion and allow callers to disable animation.',
  ]}
/>
