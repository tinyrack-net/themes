## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `variant` | `default`, `outlined`, `elevated` | `default` |
| `padding` | `none`, `sm`, `md`, `lg` | `md` |
| parts | `Root`, `Header`, `Title`, `Description`, `Content`, `Footer` | compound API |

## API

`TRCard.Root` and `TRCard.Title` support Base UI's additive `render` contract. Every part forwards native props and `ref`; keep headings in document order when composing cards.
