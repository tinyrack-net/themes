# Autocomplete

Free-form text input with filtered suggestions and keyboard navigation.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Storybook controls cover `placeholder`, `open`, `disabled`.
