## Contract

`Root`, `Header`, `Brand`, `Actions`, `Sidebar`, `Main`, and `Outline` compose a responsive shell. `docs` provides desktop navigation and a focus-managed mobile drawer, `splash` keeps only the global header, and `standalone` removes both navigation regions. Router state arrives through props only.

## API

`Root` requires `currentPath` and `locationKey`. Pass `hash` for contained heading scrolling, `navigationKind` as `POP`, `PUSH`, or `REPLACE` for scroll restoration, and `pendingPath` to expose pending progress and `aria-busy`. Use `layout` to select `docs`, `splash`, or `standalone`.

`Header`, `Brand`, `Actions`, `Sidebar`, and `Outline` preserve their native props, class names, styles, and refs. `Main` additionally accepts `contentClassName` for its inner content and `viewportLabel` for the scroll region's accessible name.

Set `--tr-docs-shell-block-size` when the shell is embedded in a bounded preview or application region; it defaults to `100dvh`.
