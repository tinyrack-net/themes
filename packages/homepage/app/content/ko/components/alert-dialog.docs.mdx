## 계약

이 Tinyrack wrapper는 Base UI 동작, React 19 refs, native props, events, `render` contract를 보존합니다. Playground control은 `label`, `open`, `disabled`, 공유 `handleMode` 연결을 다룹니다.

  const [result, setResult] = useState('Rack not deleted');

  return <div>
    <TRAlertDialog.Root>
      <TRAlertDialog.Trigger>Delete rack</TRAlertDialog.Trigger>
      <TRAlertDialog.Portal>
        <TRAlertDialog.Backdrop />
        <TRAlertDialog.Viewport>
          <TRAlertDialog.Popup>
            <TRAlertDialog.Title>Delete rack?</TRAlertDialog.Title>
            <TRAlertDialog.Description>This action cannot be undone.</TRAlertDialog.Description>
            <div className="tr-alert-dialog-actions">
              <TRAlertDialog.Close render={<TRButton variant="secondary" />}>Cancel</TRAlertDialog.Close>
              <TRAlertDialog.Close render={<TRButton variant="danger" />} onClick={() => setResult('Rack deleted')}>
                Delete rack
              </TRAlertDialog.Close>
            </div>
          </TRAlertDialog.Popup>
        </TRAlertDialog.Viewport>
      </TRAlertDialog.Portal>
    </TRAlertDialog.Root>

  </div>;

## API

모든 compound part와 prop/state type은 `@tinyrack/ui/components/alert-dialog`에서 export됩니다. style은 대응하는 CSS subpath를 통해 opt-in으로 유지됩니다. 확인 컨트롤은 `tr-alert-dialog-actions`로 그룹화하면 좁은 viewport에서 읽기 쉬운 액션으로 줄바꿈됩니다. Cancel에는 secondary TRButton을, 파괴적 확인에는 danger TRButton을 사용하세요.
