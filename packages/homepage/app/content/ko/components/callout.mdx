## 핵심 속성

Callout은 보조 안내를 문서의 주요 흐름과 구분해요. `note`는 배경 정보, `tip`은 선택적 조언, `caution`은 피할 수 있는 위험, `danger`는 데이터 손실처럼 심각한 피해를 일으킬 수 있는 작업에 사용하세요. 기본적으로 정적 콘텐츠이며 실시간 알림이 아니에요.

## API

### 콘텐츠와 variant

| Prop | 타입 | 기본값 | 용도 |
| --- | --- | --- | --- |
| `children` | `ReactNode` | 필수 | 인라인 또는 블록 수준의 안내 콘텐츠예요. |
| `variant` | `'note' \| 'tip' \| 'caution' \| 'danger'` | `'note'` | 의미론적 제목과 시각적 표현을 선택해요. |
| `title` | `ReactNode` | 변형별 제목 | 지역화를 포함해 기본 영어 제목을 교체해요. |

### HTML과 live region

`TRCallout`은 기본 `div` props, 이벤트, `className`, `style`, React 19 `ref`도 전달해요. 동적으로 삽입되는 긴급 또는 일반 알림에는 `role="alert"`나 `role="status"`를 명시하세요. 정적 문서 안내에는 실시간 영역 역할을 추가하지 마세요.
