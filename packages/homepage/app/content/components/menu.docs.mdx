# Menu

Keyboard-navigable action lists with separators, disabled items, positioning, and dismissal.

## Contract

Use `Root/Trigger/Portal/Positioner/Popup` for the surface and compose `Item` or `Separator` inside it. A nested menu uses `SubmenuRoot` and `SubmenuTrigger` around its own `Portal/Positioner/Popup` surface.
