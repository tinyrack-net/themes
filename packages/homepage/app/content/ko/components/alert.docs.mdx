## API

`TRAlert.Root`는 native `div` props와 React 19 `ref`를 전달합니다. 알림의 긴급성은 애플리케이션 맥락이므로 live-region role을 자동으로 지정하지 않습니다. 기본 `div`, `p` 요소가 주변 의미에 맞지 않을 때 `TRAlert.Title`과 `TRAlert.Description`은 Base UI `render` contract를 지원합니다.
