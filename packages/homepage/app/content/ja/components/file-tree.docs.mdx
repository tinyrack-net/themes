## コントラクト

`FileTree` は Markdown の unordered list を children として受け取ります。ネストした list はディレクトリになり、`/` で終わる list item は空のディレクトリになります。ディレクトリはプラットフォームの `details` disclosure を使います。

## 例

別の Markdown list をネストしてディレクトリを作成します。`...` または `…` は placeholder として使用でき、`**bold**` や `` `code` `` などの inline Markdown も保持されます。

## API

`FileTree` は native `ul` props と React 19 refs を転送します。ディレクトリはデフォルトで開いており、個別に切り替えられます。
