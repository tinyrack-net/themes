# Radio

<div className="tr-doc-lead">
  Mutually exclusive native choices with vertical or horizontal group layout.
</div>

## Contract

| Area | Values | Rule |
| --- | --- | --- |
| Size | `sm`, `md`, `lg` | Shares choice spacing with `Checkbox` and `Switch`. |
| Orientation | `vertical`, `horizontal` | Applied by `RadioGroup`. |
| Input | native radio | Keep radios grouped by shared `name`. |

## Usage

```tsx
import '@tinyrack/ui/components/form/form.css';
import { Radio, RadioGroup } from '@tinyrack/ui/components/form/react';

<RadioGroup aria-label="Rack role" orientation="vertical">
  <Radio defaultChecked name="rack-role" value="primary">Primary</Radio>
  <Radio name="rack-role" value="replica">Replica</Radio>
</RadioGroup>;
```

```html
<fieldset class="tr-radio-group" data-orientation="vertical">
  <label class="tr-radio" data-size="md">
    <input class="tr-radio-input" type="radio" name="rack-role" checked />
    <span aria-hidden="true" class="tr-radio-control"></span>
    <span class="tr-radio-label">Primary</span>
  </label>
</fieldset>
```
