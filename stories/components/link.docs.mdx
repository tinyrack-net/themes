# Link

Navigation with semantic color and underline behavior while preserving native anchor capabilities.

## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `variant` | `default`, `muted`, `primary`, `danger` | `default` |
| `underline` | `always`, `hover`, `none` | `hover` |

## API

Use `Link` for navigation and `Button` for commands; `target`, `rel`, `download`, native anchor props, and `ref` are forwarded unchanged.
