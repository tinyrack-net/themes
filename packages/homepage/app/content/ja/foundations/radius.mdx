## 実装

Radius の値は public TypeScript metadata と runtime CSS token です。コンポーネント CSS は semantic radius role を使い、アプリケーション layout は prefix 付き Tailwind utility を使えます。

```tsx
<TRCard.Root>
  <TRButton>デプロイ</TRButton>
  <TRBadge>準備完了</TRBadge>
</TRCard.Root>
```

```ts
import { tinyrackRadii } from '@tinyrack/ui/core';

const panelRadius = tinyrackRadii.lg;
const statusRadius = tinyrackRadii.full;
```

CSS では `--tinyrack-radius-*` を、Tailwind markup では `rounded-tinyrack-*` utility を使います。
