## 계약

이 Tinyrack wrapper는 Base UI 동작, React 19 ref, native props, event, `render` contract를 보존합니다. Playground control은 `disabled`, `orientation`, `loopFocus`, 접근 가능한 `label`을 다룹니다.

## API

모든 compound part와 prop/state type은 `@tinyrack/ui/components/menubar`에서 export됩니다. style은 대응 CSS subpath를 통한 opt-in으로 유지됩니다.
