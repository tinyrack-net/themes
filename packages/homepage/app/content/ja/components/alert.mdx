## コントラクト

| 軸 | 値 | 既定値 |
| --- | --- | --- |
| `variant` | `neutral`, `info`, `success`, `warning`, `danger` | `neutral` |
| `role` | `status`、`alert` などの native `div` role | なし |
| parts | `Root`, `Title`, `Description`, `Actions` | compound API |

## API

`TRAlert.Root` と `TRAlert.Actions` はネイティブ `div` props と React 19 ref を転送します。通知の緊急度はアプリケーションの文脈で決まるため、`TRAlert.Root` は live-region role を自動付与しません。既定の `div` や `p` が周囲の意味に合わない場合、`TRAlert.Title` と `TRAlert.Description` は Base UI の `render` contract に対応します。
