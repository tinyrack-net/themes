## コントラクト

| Axis | Contract |
| --- | --- |
| selection | single by default; opt into `multiple` |
| state | controlled `value` or uncontrolled `defaultValue`; both use string arrays and default to `[]` |
| orientation | horizontal by default; vertical when requested |
| keyboard | arrow focus loops by default; set `loopFocus={false}` for bounded focus |
| availability | enabled by default; disable the group or an individual item |

    </div>

## API

TRToggleGroup は Base UI の state、arrow-key focus、orientation、ref、ネイティブ props、event、`render` を保ちます。安定した item `value` を用意し、制御付き・非制御 selection を string array として保ちます。horizontal では Left/Right、vertical では Up/Down arrow を使い、`loopFocus={false}` は group の端で停止します。Group の `disabled` はすべての item に適用されますが、個別の TRToggle は独立して無効化できます。group にアクセシブルな名前を付けます。selection が名前付き form field である必要がある場合は TRCheckboxGroup または TRRadioGroup を使います。
