# ScrollArea

A custom scrollbar surface that preserves native scrolling.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native scrolling, and the `render` contract. Playground controls cover vertical, horizontal, and two-axis content.

## API

`Root` establishes the visible size and overflow-edge threshold. Its default `surface` variant keeps the existing border and padding contract; `plain` removes both for shells and embedded canvases. `Viewport` owns native wheel, touch, trackpad, and keyboard scrolling, so give a focusable viewport an accessible name. `Content` holds the scrollable layout; add one `Scrollbar` and `Thumb` for each required axis, and render `Corner` only where both tracks meet. `Scrollbar` also supports `keepMounted`; every part preserves refs, native props, and the Base UI `render` contract.
