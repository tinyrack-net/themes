## API

TRSpinner는 기본적으로 `role="status"`를 노출하므로 비어 있지 않은 작업별 `label`을 사용하세요. 빈 label은 `Loading`으로 대체됩니다. 이미 이름이 있는 loading control이나 status region 안에서는 `decorative`를 설정해 하나의 element만 알리게 하세요. 작업이 진행 중일 때만 spinner를 mount하고, 적절한 경우 업데이트되는 region에 `aria-busy="true"`를 설정한 뒤 그 region에서 완료를 알리세요. 사용자가 reduced motion을 요청하면 회전이 멈춥니다.

| CSS token | 기본값 | 용도 |
| --- | --- | --- |
| `--tr-spinner-size` | size recipe | Indicator diameter |
| `--tr-spinner-stroke-width` | `--tinyrack-spinner-stroke-width` | Ring thickness |
| `--tr-spinner-color` | semantic variant color | Active and track color |
| `--tinyrack-spinner-size-sm` | foundation metric | Small diameter |
| `--tinyrack-spinner-size-md` | foundation metric | Medium diameter |
| `--tinyrack-spinner-size-lg` | foundation metric | Large diameter |
| `--tinyrack-spinner-track-opacity` | foundation metric | Inactive ring opacity |
