## 계약

TRTextarea는 native ref, name, FormData, reset, disabled, read-only, required, invalid 상태를 그대로 보존합니다.

밀도 높은 화면에서는 `uiSize="sm"`을, 더 큰 읽기 및 편집 영역이 필요하면 `uiSize="lg"`를 사용하세요. 내용이 표시 영역을 넘으면 native scroll이 동작하며, 가로 layout overflow를 방지하도록 세로 방향으로만 크기를 조절할 수 있습니다.

function MaintenanceForm() {

      <TRButton type="submit">Submit</TRButton>
      <TRButton appearance="outline" type="reset">Reset</TRButton>
      <output aria-live="polite">{result}</output>
    </TRForm>
  );
}` }]}

<ComponentExampleTabs
  id="textarea-validation"
  title="필수 submit과 복구"
  description="비어 있는 상태로 submit해 invalid textarea로 focus를 이동한 뒤 reason을 입력하고 다시 submit하세요."
  preview={<Stories.TextareaValidationPreview />}
  sources={[{ label: 'React', language: 'tsx', code: String.raw`import '@tinyrack/ui/components/button.css';
import '@tinyrack/ui/components/field.css';
import '@tinyrack/ui/components/form.css';
import { TRButton } from '@tinyrack/ui/components/button';
import { TRField } from '@tinyrack/ui/components/field';
import { TRForm } from '@tinyrack/ui/components/form';
import { useState } from 'react';

function ChangeReasonForm() {
  const [attempted, setAttempted] = useState(false);
  const [submitted, setSubmitted] = useState(false);
  const [value, setValue] = useState('');
  const invalid = attempted && value.trim().length === 0;

  return (
    <TRForm noValidate onSubmit={(event) => {
      event.preventDefault();
      setAttempted(true);
      setSubmitted(value.trim().length > 0);
      if (value.trim().length === 0) {
        const reason = event.currentTarget.elements.namedItem('reason');
        if (reason instanceof HTMLTextAreaElement) reason.focus();
      }
    }}>
      <TRField.Root invalid={invalid}>
        <TRField.Label htmlFor="reason">Change reason</TRField.Label>
        <TRTextarea aria-invalid={invalid} id="reason" name="reason" onChange={(event) => {
          setValue(event.currentTarget.value);
          setSubmitted(false);
        }} required value={value} />
        <TRField.Error match>{invalid ? 'Add a reason before submitting.' : null}</TRField.Error>
      </TRField.Root>
      <TRButton type="submit">Submit change</TRButton>
      <output aria-live="polite">{submitted ? 'Change submitted.' : ''}</output>
    </TRForm>
  );
}` }]}
/>

## API

`TRTextarea`는 모든 native textarea prop, ref, event, class, style을 전달합니다. 기본 resize 방향은 vertical이며 native validity가 기준으로 유지됩니다.

| Prop | Type | 기본값 | 설명 |
| --- | --- | --- | --- |
| `uiSize` | `'sm' \| 'md' \| 'lg'` | `'md'` | 해당 control의 typography, padding, minimum block size를 적용합니다. |
| Native props | `ComponentPropsWithRef<'textarea'>` | 해당 없음 | `value`, `defaultValue`, `rows`, `name`, validation attribute, event, style, `ref`를 포함합니다. |
