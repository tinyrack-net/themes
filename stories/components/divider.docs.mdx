# Divider

A semantic separator for repeated rows, sections, and compact two-column operational layouts.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `tr-divider` on `<hr>` | `Divider` | Required separator | Use between related groups, not as decoration in every gap. |
| Orientation | `data-orientation` | `orientation` | `horizontal`, `vertical`; default `horizontal` | Vertical dividers need a parent with a meaningful height. |
| Native | Native `<hr>` attributes | `HTMLAttributes<HTMLHRElement>` | Passed through | Vertical mode exposes `aria-orientation="vertical"`. |

## Guidance

<ComponentGuidance
  whenToUse={[
    'Use Divider when a boundary needs to remain visible across dense operational content.',
    'Use horizontal mode for stacked sections and vertical mode inside a row or toolbar.',
  ]}
  avoid={[
    'Do not use Divider as a substitute for normal spacing.',
    'Do not use vertical mode without a parent height that makes the separator meaningful.',
  ]}
  accessibility={[
    'Keep the native separator semantics and provide a label only when the separator itself needs explanation.',
    'Preserve the generated vertical aria-orientation attribute in custom markup.',
  ]}
/>
