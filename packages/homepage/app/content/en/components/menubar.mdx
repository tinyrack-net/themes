## Contract

`TRMenubar` is the single menubar root. It composes `TRMenu.Root` children and preserves Base UI keyboard behavior, React 19 refs, native props and events, and the `render` contract. Use `orientation` to select horizontal Left/Right or vertical Up/Down roving focus, `loopFocus` to control wrapping, and `disabled` to disable every child menu.

## API

`@tinyrack/ui/components/menubar` exports the `TRMenubar` root with `TRMenubarProps` and `TRMenubarState`. Menu anatomy such as `TRMenu.Root`, `TRMenu.Trigger`, `TRMenu.Portal`, `TRMenu.Positioner`, and `TRMenu.Popup` is exported separately from `@tinyrack/ui/components/menu`. The menubar CSS subpath includes the styles required by its composed menu parts.
