## 계약

`TRColorSchemeToggle`은 제어형 2상태 버튼이에요. 아이콘과 접근 가능한 이름은 다음 동작을 설명해요. `light`에서는 `dark`를, `dark`에서는 `light`를 제공해요. 컴포넌트는 `localStorage`를 읽거나 `system` 설정을 조회하거나 문서를 변경하지 않아요. 이 작업은 애플리케이션 어댑터에서 처리하고 저장하세요.

## API

| Prop | Type | Default | 설명 |
| --- | --- | --- | --- |
| `value` | `"light" \| "dark"` | 필수 | 현재 해석된 색상 모드예요. |
| `onValueChange` | `(value: TRColorScheme) => void` | 필수 | 취소되지 않은 활성화 후 반대 색상 모드를 전달해요. |
| `darkLabel` | `string` | `"Use dark color scheme"` | 다음 값이 `dark`일 때의 접근 가능한 동작 이름이에요. |
| `lightLabel` | `string` | `"Use light color scheme"` | 다음 값이 `light`일 때의 접근 가능한 동작 이름이에요. |
| `aria-labelledby` | `string` | — | 생성된 레이블 대신 외부의 접근 가능한 이름을 사용해요. |
| `disabled` | `boolean` | `false` | 포커스와 활성화를 막아요. |
| `uiSize` | `"sm" \| "md" \| "lg"` | `"md"` | 상속된 IconButton 크기예요. |
| `ref` | `Ref<HTMLButtonElement>` | — | 버튼으로 전달되는 ref예요. |
| `onClick` | `MouseEventHandler<HTMLButtonElement>` | — | `onValueChange`보다 먼저 실행되며 `preventDefault()`로 값 변경을 취소해요. |

그 밖의 안전한 `TRIconButton` 버튼 속성, 네이티브 이벤트, `className`, `style`, 사용자 지정 토큰을 상속해요. 시각적 `appearance`는 `ghost`로, 컨트롤은 네이티브 버튼으로 유지해요.
