## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `variant` | `neutral`, `info`, `success`, `warning`, `danger` | `neutral` |
| `size` | `sm`, `md`, `lg` | `md` |

## API

`TRBadge` forwards native `span` props, events, styles, and React 19 `ref`. Keep status text visible; an icon alone needs an application-provided accessible label.
