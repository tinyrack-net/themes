## 계약

`TRMenubar`는 단일 menubar root입니다. `TRMenu.Root` 자식을 조합하며 Base UI keyboard 동작, React 19 ref, native prop과 event, `render` contract를 보존합니다. `orientation`은 가로 Left/Right 또는 세로 Up/Down roving focus를 선택하고, `loopFocus`는 끝에서의 순환을 제어하며, `disabled`는 모든 하위 menu를 비활성화합니다.

## API

`@tinyrack/ui/components/menubar`는 `TRMenubar` root와 `TRMenubarProps`, `TRMenubarState`를 export합니다. `TRMenu.Root`, `TRMenu.Trigger`, `TRMenu.Portal`, `TRMenu.Positioner`, `TRMenu.Popup` 같은 menu anatomy는 `@tinyrack/ui/components/menu`에서 별도로 export됩니다. menubar CSS subpath에는 조합된 menu part에 필요한 style이 포함됩니다.
