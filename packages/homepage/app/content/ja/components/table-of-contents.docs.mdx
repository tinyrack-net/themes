## コントラクト

h2/h3 の見出しデータと現在の見出しを渡します。`renderLink` は router link を注入できますが、ネイティブの hash navigation が fallback として残ります。

## API

Depth は `2 | 3` に制限されます。同じデータからデスクトップレールとモバイルのネイティブ `details` disclosure をレンダリングします。
