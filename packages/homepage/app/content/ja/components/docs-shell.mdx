## コントラクト

`Root`、`Header`、`Brand`、`Actions`、`Sidebar`、`Main`、`Outline` でレスポンシブな shell を構成します。`docs` と `splash` layout では `Header` が sidebar と content の上でビューポート全体に広がります。router 状態は props だけで渡されます。

## 例

グローバル header を残す製品ランディングページには `splash`、ナビゲーション chrome のない埋め込み API リファレンスには `standalone` を使います。

## API

`layout` は `docs`、`splash`、`standalone` に対応し、モバイル disclosure と scroll restoration は UI 内部に残ります。
