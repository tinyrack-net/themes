## コントラクト

| 軸 | 動作 |
| --- | --- |
| ownership | selection is owned by the nearest `TRRadioGroup` |
| identity | every item needs a stable `value`; labels should target a unique `id` |
| state | selected, unselected, read-only, and disabled |

密度の高いコントロールサーフェスで小さな radio が必要な場合は `uiSize="sm"` を使います。

## API

`Root`、`Indicator` と prop/state type は `@tinyrack/ui/components/radio` から export されます。`TRRadioGroup` 内だけで使います。完全な label row を少なくとも 24px の高さに保ち、表示テキストで radio glyph より大きな pointer target を作ります。
