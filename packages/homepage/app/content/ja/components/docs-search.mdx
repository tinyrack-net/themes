## Contract

`TRDocsSearch.Trigger` は単独のボタン、`TRDocsSearch.Dialog` は制御されたポータル検索画面です。Dialog は Ctrl/⌘+K、読み込み中、空、エラー、fallback、キーボード選択、フォーカス復帰、古いリクエストのキャンセルを管理します。データソースは `onSearch(query, signal)` が管理し、キャンセル可能な処理には受け取った `AbortSignal` を渡してください。

## Examples

- フォーカスは combobox に留まり、`ArrowDown` と `ArrowUp` でアクティブな結果を移動します。`Enter` は結果を選択し、`Escape` は dialog を閉じます。
- `enableShortcut={false}` はグローバルショートカットを無効にします。埋め込み preview や disabled trigger ですべての起動経路を止める場合に使用します。
- `compact` はアクセシブルな名前を保ちながら表示ラベルとショートカットを隠します。`uiSize="sm" | "md" | "lg"` の各サイズで compact trigger は正方形を保ちます。
- Dialog は `document.body` にポータルされます。閉じた後に trigger へフォーカスを戻すには `returnFocusRef` を渡します。
- 閉じた dialog と trigger は SSR セーフで、開くまではポータルを描画せずに hydration されます。
- 主要な検索インデックスを利用できない場合は `fallback` を設定し、縮小されたローカル結果が表示される理由を伝えます。検索リクエストが reject された場合は、Usage ソースで示すローカライズ済みの `messages.error` 状態を使用します。

## API

| エクスポート | 主な props | 用途 |
| --- | --- | --- |
| `TRDocsSearch` | `Trigger`, `Dialog` | 複合 namespace です。 |
| `TRDocsSearch.Trigger` | `compact`, `label`, `shortcutLabel`, `uiSize`、button props、`ref` | 表示型または compact の検索ボタンです。`disabled` はボタン操作を防ぎます。 |
| `TRDocsSearch.Dialog` | `open`, `onOpenChange`, `onSearch`, `onSelect`, `messages`, `fallback`, `enableShortcut`, `popupRef`, `returnFocusRef` | 制御された検索 dialog と結果一覧です。 |
| `TRDocsSearchResult` | `id`, `url`, `title`, `section`, `excerpt`、任意の一致範囲 | Dialog が描画する検索結果です。一致範囲の終了 offset は含まれません。 |
| `TRDocsSearchMessages` | `title`, `trigger`, `placeholder`, `close`, `results`, `idle`, `loading`, `empty`, `error`, `fallback` | 表示文言とアクセシブルな文言をすべてローカライズします。 |

`onSearch` は `(query: string, signal: AbortSignal)` を受け取り、readonly な結果の promise を返します。Reject された場合は `messages.error` を表示し、キャンセルまたは置換されたリクエストが新しい結果を上書きすることはありません。
