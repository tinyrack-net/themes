### モーションの低減

モーションの低減は装飾的な移動だけを取り除き、状態変化やその意味は残します。

<ReducedMotionPreview />

## 実装

コンポーネントは意味に基づく持続時間とイージング変数を使います。コンポーネント境界に 1 つのトランジションを置き、モーション低減設定で装飾的な移動を除去します。

```css
.rack-layer {
  transition:
    opacity var(--tinyrack-duration-normal) var(--tinyrack-ease-out),
    transform var(--tinyrack-duration-normal) var(--tinyrack-ease-out);
}

@media (prefers-reduced-motion: reduce) {
  .rack-layer {
    transition-duration: 1ms;
    transform: none;
  }
}
```

システム全体の時間には `--tinyrack-duration-*` を使い、意図した局所的な例外にだけコンポーネントレベルの `--tr-*` 変数を使います。
