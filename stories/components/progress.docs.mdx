# Progress

Determinate and indeterminate task progress with size and semantic status axes.

## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `variant` | `neutral`, `info`, `success`, `warning`, `danger` | `neutral` |
| `size` | `sm`, `md`, `lg` | `md` |
| parts | `Root`, `Label`, `Track`, `Indicator`, `Value` | compound API |

## Install

<ComponentInstall surfaces={[{ label: 'React', install: 'pnpm add @tinyrack/ui react react-dom', imports: ["import '@tinyrack/ui/core.css';", "import '@tinyrack/ui/components/progress.css';", "import { Progress } from '@tinyrack/ui/components/progress';"] }]} />
