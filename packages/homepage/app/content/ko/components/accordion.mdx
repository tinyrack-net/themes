## 계약

| 축 | 값 | 기본값 |
| --- | --- | --- |
| selection | controlled `value` or uncontrolled `defaultValue` | `[]` |
| expansion | Root `multiple` | single |
| availability | Root `disabled` or Item `disabled` | interactive |
| lifecycle | unmount, `keepMounted`, or `hiddenUntilFound` | unmount |

## API

### 구조

```tsx
<TRAccordion.Root>
  <TRAccordion.Item value="network">
    <TRAccordion.Header>
      <TRAccordion.Trigger>네트워크</TRAccordion.Trigger>
    </TRAccordion.Header>
    <TRAccordion.Panel>온라인</TRAccordion.Panel>
  </TRAccordion.Item>
</TRAccordion.Root>
```

전체 구조는 `Root`, `Item`, `Header`, `Trigger`, `Panel`입니다. 각 part는 `TRAccordion.Part`와 named `TRAccordionPart` export로 제공되며, 대응하는 `TRAccordionPartProps`와 `TRAccordionPartState` type도 제공합니다.

### Root props

| Prop | Type | 기본값 | 용도 |
| --- | --- | --- | --- |
| `value` | `Value[]` | - | 열린 item value를 제어합니다. |
| `defaultValue` | `Value[]` | `[]` | 비제어형 상태에서 처음 열 item을 지정합니다. |
| `onValueChange` | `(value, details) => void` | - | 다음 value와 Base UI event details를 전달합니다. |
| `multiple` | `boolean` | `false` | 여러 item을 동시에 열어 둡니다. |
| `disabled` | `boolean` | `false` | 모든 item의 상호작용을 막습니다. |
| `keepMounted` | `boolean` | `false` | 닫힌 panel을 `hidden` 상태로 DOM에 유지합니다. |
| `hiddenUntilFound` | `boolean` | `false` | 닫힌 panel을 `hidden="until-found"`로 검색 가능하게 유지하며 `keepMounted`보다 우선합니다. |
| `render` | `ReactElement \| function` | `<div>` | 동작과 ref 병합을 유지하면서 root element를 바꿉니다. |

`Item`은 제어형 또는 기본 확장에 안정적인 `value`가 필요하며 `disabled`와 `onOpenChange`를 받습니다. `Header`는 기본적으로 `h3`를 렌더링하므로 주변 heading level에 맞춰 `render`를 사용하세요. `Trigger`는 button, `Panel`은 연결된 region을 렌더링합니다. 모든 part는 `className`, `style`, native attribute, event handler, ref, Base UI의 state 기반 `render` contract를 보존합니다.

Enter 또는 Space는 focus를 유지한 채 trigger를 전환합니다. Tab과 Shift+Tab은 일반 document 순서를 따릅니다. Disabled trigger도 `aria-disabled="true"`로 focus를 받지만 전환되지는 않습니다. 더 이상 권장하지 않는 화살표 키 roving-focus props는 focus 동작에 영향을 주지 않습니다.

### 모양

| Token | Fallback | 용도 |
| --- | --- | --- |
| `--tr-accordion-background` | `--tinyrack-surface` | Root surface. |
| `--tr-accordion-radius` | `--tinyrack-radius-md` | 바깥쪽 radius. |
| `--tr-accordion-content-background` | `transparent` | 열린 panel surface. |
| `--tr-accordion-disabled-background` | `--tinyrack-surface-muted` | Disabled item surface. |
| `--tr-accordion-disabled-color` | `--tinyrack-text-muted` | Disabled item text. |
| `--tr-collapsible-duration` | `--tinyrack-duration-normal` | Collapsible에서 상속하는 panel transition duration. |

`@tinyrack/ui/components/accordion.css`를 명시적으로 import하세요. Foundation token이 light/dark 색상, border, focus indicator, typography, spacing, reduced-motion 동작을 제공합니다.
