# CopyButton

A reusable copy action with Clipboard API fallback and accessible status announcements.

## Contract

The control reports `idle`, `copied`, or `unavailable` through `data-copy-status` and resets after two seconds by default.

## API

`value` is copied with Clipboard API and an offscreen selection fallback. Labels, `resetDelay`, status changes, and all normal Button appearance, variant, size, loading, and event props are supported.
