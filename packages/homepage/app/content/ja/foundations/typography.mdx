role recipe は font size、line height、tracking、weight を定義します。コンポーネントは数値を繰り返さず、同じ名前付き weight を使います。

## 実装

Core CSS は IBM Plex Sans font variable と prefix 付きの size、line-height、tracking utility を公開します。

```css
@import "tailwindcss";
@import "@tinyrack/ui/core.css";
```

```html
<header>
  <span class="text-tinyrack-xs leading-tinyrack-xs tracking-tinyrack-lg uppercase">Rack A</span>
  <h1 class="font-tinyrack-heading text-tinyrack-3xl leading-tinyrack-sm">12 nodes online</h1>
  <p class="text-tinyrack-md leading-tinyrack-md">The cluster is healthy.</p>
</header>
```

```ts
import { tinyrackTypography } from '@tinyrack/ui/core';

const headingRecipe = tinyrackTypography.textStyle.headingLg;
const bodySize = tinyrackTypography.fontSize.md;
```
