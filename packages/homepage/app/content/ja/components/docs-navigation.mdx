## 主なプロパティ

`TRDocsNavigation` は、名前付きのナビゲーションランドマーク内に再帰的な `group`、`page`、`link` 項目をレンダリングします。現在または保留中のパスを含むグループは、ルート状態の変更後も自動的に開きます。ほかのグループはポインターまたはキーボードで開閉できます。

ルート照合はアダプター側で行います。コンポーネントは、渡された `currentPath` と `pendingPath` の文字列を完全一致で比較します。

## アクセシビリティ

ルートは `label` で名前を付けた `<nav>` です。現在のページには `aria-current="page"`、保留中のリンクには `data-pending` と装飾用スピナーが設定されます。グループのトリガーは Collapsible が提供するネイティブボタンで、Enter キー、Space キー、通常の Tab ナビゲーションに対応します。項目のラベルは明確にし、既定の `Documentation` が適切でない場合は `label` をローカライズしてください。

## API

### `TRDocsNavigation`

| プロパティ | 型 | 既定値 | 用途 |
| --- | --- | --- | --- |
| `items` | `readonly TRDocsNavigationItem[]` | 必須 | 再帰的なナビゲーションデータです。 |
| `currentPath` | `string` | 必須 | 現在のページとして示す完全一致のパスです。 |
| `pendingPath` | `string` | `undefined` | 保留中として示す遷移先の完全一致パスです。 |
| `defaultGroupsOpen` | `boolean` | `false` | 現在または保留中のパスを含まないグループも初期状態で開きます。 |
| `label` | `string` | `'Documentation'` | ナビゲーションランドマークのアクセシブルな名前です。 |
| `renderLink` | `(item, state) => ReactElement` | ネイティブのアンカー | フレームワークのルーターリンクを注入します。`state` は `active` と `pending` を含みます。 |
| `onNavigate` | `(item) => void` | `undefined` | `page` または `link` の操作後に実行します。モバイルドロワーを閉じる用途などに使えます。 |

`TRDocsNavigation` はネイティブの `nav` プロパティ、`className`、`style`、イベント、データ属性、React 19 の `ref` も転送します。

### `TRDocsNavigationItem`

`page` 項目は `label`、`path`、`type: 'page'` を持ちます。`link` 項目には任意の `external` が加わります。外部リンクに `target` や専用表示が必要な場合は、`renderLink` でこのメタデータを使用してください。`group` 項目は `label`、`type: 'group'`、再帰的な `children` を持ちます。

公開されるデータ型は `TRDocsNavigationItem`、`TRDocsNavigationGroup`、`TRDocsNavigationPage`、`TRDocsNavigationLink` です。`TRDocsNavigationLinkState` は `renderLink` に渡す `active` と `pending` の真偽値を含み、`TRDocsNavigationProps` はコンポーネントの完全なプロパティ型です。

## スタイル

コアトークンの後にコンポーネントのスタイルシートをインポートしてください。`--tr-docs-navigation-gap`、`--tr-docs-navigation-item-gap`、`--tr-docs-navigation-group-color`、`--tr-docs-navigation-link-color`、`--tr-docs-navigation-link-hover-background`、`--tr-docs-navigation-link-hover-color`、`--tr-docs-navigation-link-active-border-color` を上書きできます。長いラベルは狭いサイドバーやモバイルドロワー内で折り返されます。

## SSR

サーバー出力には現在のパスと初期状態で開く祖先グループが含まれます。同じ `items`、`currentPath`、`pendingPath` でハイドレートしてください。ハイドレーション後のルート変更では、ナビゲーションツリーを置き換えずに新しく該当するグループを開きます。
