## コントラクト

| 軸 | 値 | 既定値 |
| --- | --- | --- |
| `variant` | `neutral`, `info`, `success`, `warning`, `danger` | toast `type` or `neutral` |
| `position` | block start/end × inline start/center/end | `block-end-inline-end` |
| parts | `Provider`, `Portal`, `Viewport`, `Root`, `Content`, `Title`, `Description`, `Action`, `閉じる`, `Positioner`, `Arrow` | compound API |

<ComponentExampleTabs
  id="toast-lifecycle"
  title="アクション・更新・promise・queue・timeout"
  description="時間付き toast を表示してその場で更新し、promise 状態を対応付けるか、provider limit 2 の後ろに notification を 4 件 enqueue します。"
  preview={<TRToast.Provider limit={2}><ToastLifecycleDemo /></TRToast.Provider>}
  sources={[{ label: 'React', language: 'tsx', code: String.raw`function ToastLifecycleDemo() {
  const manager = useToastManager();
  const toastId = useRef(null);

  return <>
    <TRButton onClick={() => {
      toastId.current = manager.add({
        title: 'Rack operation',
        description: 'This toast closes in five seconds.',
        timeout: 5000,
      });
    }}>Show timed toast</TRButton>
    <TRButton onClick={() => {
      if (toastId.current === null) return;
      manager.update(toastId.current, {
        title: 'Rack operation updated',
        type: 'success',
      });
    }}>Update toast</TRButton>
    <TRButton onClick={() => manager.promise(Promise.resolve('Rack A'), {
      loading: { title: 'Deploying rack', type: 'info' },
      success: (rack) => ({ title: rack + ' deployed', type: 'success' }),
      error: { title: 'Deployment failed', type: 'danger' },
    })}>Run promise</TRButton>
    <TRButton onClick={() => {
      for (let index = 1; index <= 4; index += 1) {
        manager.add({
          description: 'Queued with provider limit 2.',
          title: 'Queued toast ' + index,
          type: 'neutral',
        });
      }
    }}>Queue four</TRButton>
    <TRButton onClick={() => manager.close()}>Reset toasts</TRButton>
    <TRToast.Portal>
      <TRToast.Viewport aria-label="Lifecycle notifications">
        {manager.toasts.map((toast) => (
          <TRToast.Root key={toast.id} toast={toast}>
            <TRToast.Content><TRToast.Title>{toast.title}</TRToast.Title><TRToast.Description>{toast.description}</TRToast.Description></TRToast.Content>
            <TRToast.Action onClick={() => manager.close(toast.id)}>Undo</TRToast.Action>
            <TRToast.Close aria-label="Dismiss notification">×</TRToast.Close>
          </TRToast.Root>
        ))}
      </TRToast.Viewport>
    </TRToast.Portal>
  </>;
}

<TRToast.Provider limit={2}><ToastLifecycleDemo /></TRToast.Provider>` }]}

## API

`Provider` 内で toast manager を作成または利用します。Base UI が queue、timer、swipe gesture、focus、live-region の意味を担当します。構成した layout には `Content`、`Positioner`、`Arrow` を使え、同時にレンダリングする各 `Viewport` に異なる `aria-label` を付けます。
