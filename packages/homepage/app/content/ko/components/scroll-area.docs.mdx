## API

`Root`는 보이는 크기와 overflow edge threshold를 설정합니다. 기본 `surface` variant는 기존 border와 padding contract를 유지하고 `plain`은 shell과 embedded canvas를 위해 둘 다 제거합니다. `autoHide` 기본값은 `false`이며 활성화하면 idle track이 숨겨졌다가 hover, scroll, focus에서 다시 나타납니다. `Viewport`는 native wheel, touch, trackpad, keyboard scroll을 담당하므로 focus 가능한 viewport에 접근 가능한 이름을 제공하세요. `Content`가 scrollable layout을 담고, 필요한 각 axis에 `Scrollbar`와 `Thumb`를 하나씩 추가하며 두 track이 만나는 곳에만 `Corner`를 렌더링합니다. `Scrollbar`는 `keepMounted`도 지원하며 모든 part는 ref, native props, Base UI `render` contract를 보존합니다.
