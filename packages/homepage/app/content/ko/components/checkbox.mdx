## 계약

| Axis | Values |
| --- | --- |
| state | unchecked, checked, `indeterminate` |
| availability | enabled, `readOnly`, `disabled` |
| size | `sm`, `md`, `lg` |
| form | `name`, `value`, `uncheckedValue`, `form`, `required`, reset, native validity |

보이는 label이나 접근 가능한 이름을 제공하세요. 포커스된 checkbox는 Space로 전환하며 Enter는 native checkbox처럼 전환하지 않습니다. 값을 포커스 및 제출 가능하게 유지하려면 `readOnly`, 사용할 수 없고 제출에서도 제외하려면 `disabled`를 사용하세요.

      </TRField.Root>
      <TRButton type="submit">Continue</TRButton>
    </TRForm>

## API

### `TRCheckbox.Root`

| Prop | Type / 기본값 | 용도 |
| --- | --- | --- |
| `checked` / `defaultChecked` | `boolean` / `undefined`, `false` | controlled 또는 uncontrolled 선택 상태 |
| `indeterminate` | `boolean` / `false` | `aria-checked="mixed"`를 노출합니다. 사용자가 mixed 상태를 해소하면 `onCheckedChange`에서 지우세요. |
| `onCheckedChange` | `(checked, details) => void` | 변경 시도를 알립니다. controlled 사용자는 `checked`를 갱신합니다. |
| `disabled` / `readOnly` | `boolean` / `false` | 사용할 수 없게 하거나, 포커스 가능하지만 변경할 수 없게 합니다. |
| `required` | `boolean` / `false` | native constraint validation에 참여합니다. |
| `name`, `value`, `uncheckedValue`, `form` | `string` | 외부 form과 명시적 off 값을 포함한 native form 직렬화를 설정합니다. |
| `inputRef` / `ref` | `Ref<HTMLInputElement>` / `Ref<HTMLElement>` | 숨겨진 native input 또는 시각적 root에 접근합니다. |
| `uiSize` | `'sm' \| 'md' \| 'lg'` / `'md'` | 시각적 컨트롤 크기를 설정합니다. |
| `render` | Base UI render prop | 동작과 상태를 유지하며 root host를 교체합니다. |

Uncontrolled root는 소유 form이 reset되면 `defaultChecked`로 복원됩니다. Controlled root는 계속 `checked`의 제어를 받습니다.

### `TRCheckbox.Indicator`

Indicator는 checked 또는 indeterminate일 때 렌더링됩니다. transition 스타일에는 `keepMounted`, mixed glyph가 실제 상태를 따라야 할 때는 `render(props, state)`와 `state.indeterminate`를 사용하세요. Root와 Indicator는 render callback, 함수형 `className`/`style`, 그리고 `data-checked`, `data-unchecked`, `data-indeterminate`, `data-disabled`, `data-readonly`, `data-invalid` 속성으로 Base UI 상태를 노출합니다.

`TRCheckboxRoot`, `TRCheckboxIndicator`, `TRCheckboxRootProps`, `TRCheckboxIndicatorProps`, `TRCheckboxUiSize`, `TRCheckboxRootState`, `TRCheckboxIndicatorState`는 `@tinyrack/ui/components/checkbox`에서 export됩니다. CSS는 opt-in입니다.
