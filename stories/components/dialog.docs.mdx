# Dialog

Focus-trapped dialog surfaces with four sizes and five placements.

## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `size` on `Popup` | `sm`, `md`, `lg`, `full` | `md` |
| `placement` on `Popup` | `middle`, `top`, `bottom`, `start`, `end` | `middle` |
| `modal` on `Root` | focus trap and scroll lock when true | `true` |
| state | controlled `open` or uncontrolled `defaultOpen` | closed |

<ComponentExampleTabs
  id="dialog-task-form"
  title="Long modal task with a form"
  description="The modal traps focus while open, keeps long content inside the viewport, submits a real form, and restores focus to Open dialog after dismissal."
  preview={<Stories.DialogExample description={'Review the deployment name, environment, and rollback notes before saving. The result is announced below the form without closing the dialog.'} modal={true} open={false} placement={'middle'} size={'lg'} title={'Edit deployment'} />}

  const inputId = useId();
  const [result, setResult] = useState('No changes saved');

  return <Dialog.Root modal>
    <Dialog.Trigger>Open dialog</Dialog.Trigger>
    <Dialog.Portal>
      <Dialog.Backdrop />
      <Dialog.Popup placement="middle" size="lg">
        <Dialog.Title>Edit deployment</Dialog.Title>
        <Dialog.Description>
          Review the deployment name, environment, and rollback notes before saving.
        </Dialog.Description>
        <form onSubmit={(event) => {
          event.preventDefault();
          setResult('Saved ' + new FormData(event.currentTarget).get('rack'));
        }}>
          <label htmlFor={inputId}>Rack name</label>
          <input defaultValue="rack-alpha" id={inputId} name="rack" required />
          <button type="submit">Save changes</button>
        </form>
        <output aria-live="polite">{result}</output>
        <Dialog.Close>Cancel</Dialog.Close>
      </Dialog.Popup>
    </Dialog.Portal>
  </Dialog.Root>;
}` }]}
/>

## API

Base UI owns focus trapping, scroll lock, portal, backdrop interaction, restoration, and Escape dismissal.
