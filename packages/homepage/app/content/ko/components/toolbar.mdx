## 계약

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Playground controls cover `label`, `disabled`, `orientation`, and `loopFocus`.

## API

모든 compound part와 prop/state type은 `@tinyrack/ui/components/toolbar`에서 export됩니다. style은 대응 CSS subpath를 통한 opt-in으로 유지됩니다. `TRToolbar.Root`는 loop가 켜진 horizontal arrow-key focus가 기본이며 vertical toolbar는 Up/Down을 사용합니다. `TRToolbar.Group`은 관련 control을 이름 짓고 `TRToolbar.Input`은 일반 text-editing key를 유지하며 disabled group/item은 activation을 막습니다. horizontal toolbar에서는 `TRToolbar.Input`을 최대 하나만 사용하고 마지막에 배치해 text-editing key가 뒤의 toolbar item을 막지 않게 하세요. focus 가능한 disabled `TRToolbar.Button`은 사용 불가 상태가 보이게 유지하고 모든 `TRToolbar`와 중첩 `Group`에 접근 가능한 이름을 제공하세요.
