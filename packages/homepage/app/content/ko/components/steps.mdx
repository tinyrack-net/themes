## 계약

`Root`는 순서 있는 list를 렌더링하고 `Item`은 list item을 렌더링합니다. 작성한 MDX children은 그대로 유지됩니다.

각 단계 안에 heading, paragraph, code block을 직접 배치하세요.

## API

두 part는 native ordered-list props, list-item props, React 19 refs를 전달합니다.
