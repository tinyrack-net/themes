# ScrollArea

A custom scrollbar surface that preserves native scrolling.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native scrolling, and the `render` contract. Storybook controls cover vertical, horizontal, and two-axis content.

## Install

<ComponentInstall surfaces={[{ label: 'React', install: 'pnpm add @tinyrack/ui react react-dom', imports: ["import '@tinyrack/ui/core.css';", "import '@tinyrack/ui/components/scroll-area.css';", "import { ScrollArea } from '@tinyrack/ui/components/scroll-area';"] }]} />

## API

Keep native wheel, touch, trackpad, and keyboard scrolling on `Viewport`; scrollbars are visual controls for the same native scroll position.
