# Accordion

Keyboard-navigable sections for progressively disclosing related content.

## Contract

| Axis | Values | Default |
| --- | --- | --- |
| selection | controlled `value` or uncontrolled `defaultValue` | collapsed |
| expansion | single or `multiple` | single |
| state | item `disabled` | interactive |
