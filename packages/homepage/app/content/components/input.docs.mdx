# Input

A tokenized native input that preserves labels, refs, validity, and form submission.

## Contract

Controls expose `label`, `value`, `placeholder`, `disabled`, `readOnly`, and `required`. Use a visible `<label>` and stable `id`; use `aria-describedby` for errors or instructions.

## API

`Input` forwards native input props, refs, events, `className`, `style`, and Base UI's `render` contract. Prefer `onValueChange(value, details)` when the next string value is what the state model needs; use native `onChange` when the event object itself matters. The `render` prop can replace the rendered element while Tinyrack merges behavior, state, ref, and `.tr-input` styling onto it.
