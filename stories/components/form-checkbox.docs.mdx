# Checkbox

Boolean native input with a custom visual control and preserved input semantics.

## Contract

| Surface | API | Values | Default | Rule |
| --- | --- | --- | --- | --- |
| Wrapper class | `tr-checkbox` | label element | required | Wraps the native input and visible label. |
| Input class | `tr-checkbox-input` | native checkbox | required | The native input remains focusable. |
| Control class | `tr-checkbox-control` | decorative span | required | Mark with `aria-hidden="true"`. |
| Label class | `tr-checkbox-label` | span text | optional | Omitted when children are undefined. |
| Size attribute | `data-size` | `sm`, `md`, `lg` | `md` | Shared with `Radio` and `Switch`. |
| Invalid attribute | `data-invalid` | `true` | unset | Mirrors `aria-invalid` when `invalid` is set. |

## Install

| Surface | Install | Import |
| --- | --- | --- |
| CSS / HTML | `pnpm add @tinyrack/ui tailwindcss` | `@tinyrack/ui/core/core.css` and `@tinyrack/ui/components/form/form.css` |
| React | `pnpm add @tinyrack/ui react react-dom` | `@tinyrack/ui/components/form/react` |
| React MDX | `pnpm add @tinyrack/ui @mdx-js/react` | `@tinyrack/ui/mdx/react` maps task-list checkboxes. |
| Astro MDX | `pnpm add @tinyrack/ui astro @astrojs/mdx` | `@tinyrack/ui/mdx/astro` maps task-list checkboxes. |

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/form/form.css";
```

## API

| Prop | Values | Note |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | Default `md`; sets `data-size`. |
| `invalid` | `boolean` | Sets `aria-invalid` on input and `data-invalid` on wrapper. |
| `children` | `ReactNode` | Renders `tr-checkbox-label` when present. |
| native checkbox props | `InputHTMLAttributes<HTMLInputElement>` except `children`, `size`, `type` | Passed through to the native checkbox input. |
