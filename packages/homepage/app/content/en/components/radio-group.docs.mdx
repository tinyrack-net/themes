## Contract

| Axis | Behavior |
| --- | --- |
| value | uncontrolled with `defaultValue`, controlled with `value` and `onValueChange` |
| keyboard | arrow keys move selection; Tab enters and leaves the group once |
| form | `name` and `required` apply to the group; individual radios provide values |

## API

`TRRadioGroup` and its prop and state types are exported from `@tinyrack/ui/components/radio-group`; compose it with `TRRadio.Root` and `TRRadio.Indicator`. The root preserves refs, native props, state-driven `className`, `style`, and `render`; arrow keys move selection while Tab enters and exits the group once.
