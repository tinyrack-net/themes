# Tinyrack UI

A React-only component system for compact operational interfaces. Accessible
interactive behavior is built on Base UI, while Tinyrack owns the visual tokens,
component defaults, and public compound APIs.

## Start in five minutes

<ComponentInstall
  surfaces={[
    {
      label: 'React',
      install: 'pnpm add @tinyrack/ui react react-dom',
      imports: [
        "import '@tinyrack/ui/core.css';",
        "import '@tinyrack/ui/components/button.css';",
        "import { Button } from '@tinyrack/ui/components/button';",
      ],
      note: 'Import each component and its CSS from the same suffix-free component subpath.',
    },
  ]}
/>

```tsx
import '@tinyrack/ui/core.css';
import '@tinyrack/ui/components/badge.css';
import '@tinyrack/ui/components/button.css';
import { Badge } from '@tinyrack/ui/components/badge';
import { Button } from '@tinyrack/ui/components/button';
```

## Public package map

| Surface | Import |
| --- | --- |
| Component JavaScript | `@tinyrack/ui/components/<component>` |
| Component CSS | `@tinyrack/ui/components/<component>.css` |
| Foundation tokens | `@tinyrack/ui/core` |
| Foundation CSS | `@tinyrack/ui/core.css` |
| React MDX map | `@tinyrack/ui/mdx` |
| React MDX CSS | `@tinyrack/ui/mdx.css` |

There is no root component barrel, `/react` or `/dom` suffix, Astro renderer, or
framework-neutral DOM manager API.
