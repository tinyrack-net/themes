# Menu

Keyboard-navigable action lists with separators, disabled items, positioning, and dismissal.

## Contract

Use `Root/Trigger/Portal/Positioner/Popup` for the surface and compose `Item`, `Separator`, or `Submenu` inside it.

## Install

<ComponentInstall surfaces={[{ label: 'React', install: 'pnpm add @tinyrack/ui react react-dom', imports: ["import '@tinyrack/ui/core.css';", "import '@tinyrack/ui/components/menu.css';", "import { Menu } from '@tinyrack/ui/components/menu';"] }]} />
