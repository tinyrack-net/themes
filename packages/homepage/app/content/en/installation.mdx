Set up Tinyrack UI in a React 19 application with Tailwind CSS 4.3 or later and
render a themed primary action. You need an existing Vite React application and
Node.js 24 or later.

## 3. Import styles in order

Import `core.css` once in the global entry point, before every component
stylesheet. Component JavaScript does not load its CSS automatically.

## 4. Select a theme

Set `data-theme` on the document root. `rack-blue` is a built-in product theme.

<GettingStartedCode label="Theme selection" snippet="theme" />

## 5. Render a primary action

Import components from their public subpaths and express action meaning with

## Check the result

Start the Vite development server. The button should use the blue theme, show
the primary treatment, and retain a visible keyboard focus indicator. If it is
unstyled, check that `core.css` comes before `button.css`.

Continue with [Foundations](/en/foundations/) to understand the system rules, or
browse [Components](/en/components/) to build the interface.
