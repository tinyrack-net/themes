# Input

A tokenized native input with Base UI render and ref behavior.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Storybook controls cover `placeholder`, `disabled`, `readOnly`, `required`.
