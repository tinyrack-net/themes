## 계약

이 Tinyrack wrapper는 Base UI 동작, React 19 ref, native props, event, `render` contract를 보존합니다. 방향 전환은 `swipeDirection`을 따르고 swipe gesture와 동기화되며 reduced-motion 설정을 존중합니다. Playground control은 `label`, `open`, `modal`, `swipeDirection`, 제어형 `activeSnapPoint` mapping을 다룹니다.

## API

모든 compound part와 prop/state type은 `@tinyrack/ui/components/drawer`에서 export됩니다. style은 대응하는 CSS subpath를 통한 opt-in으로 유지됩니다.
