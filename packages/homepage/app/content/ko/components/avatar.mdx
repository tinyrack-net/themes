## API

- `TRAvatar.Root`는 Base UI root props, `className`, `style`, `render`, React 19 `ref`를 전달합니다. `uiSize` 기본값은 `md`, `shape` 기본값은 `circle`입니다.
- `TRAvatar.Image`는 `render`, `ref`, `onLoadingStatusChange`를 포함한 native image와 Base UI props를 전달합니다. ID 이미지에는 의미 있는 `alt` 텍스트를 사용하세요.
- `TRAvatar.Fallback`은 Image가 없거나 실패했을 때 표시됩니다. 콘텐츠는 짧고 알아보기 쉽게 유지하고, Image 로딩 중 깜박임을 피하려면 `delay`를 밀리초 단위로 설정하세요.
- 컴포넌트 범위 스타일에는 `--tr-avatar-size`, `--tr-avatar-radius`, `--tr-avatar-background`, `--tr-avatar-border`, `--tr-avatar-color`, `--tr-avatar-font-size`, `--tr-avatar-font-weight`를 재정의하세요.
