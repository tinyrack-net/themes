# Select

Chooses one value from a known collection and exposes a button-triggered, keyboard-navigable popup.

## Contract

| Axis | Behavior |
| --- | --- |
| value | uncontrolled with `defaultValue`, controlled with `value` and `onValueChange` |
| disclosure | `Trigger` owns focus; `Portal`, `Positioner`, and `Popup` own the floating layer |
| collection | groups, labels, separators, disabled items, text, and indicators describe the menu anatomy |

## API

The complete trigger, popup, collection, item, arrow, backdrop, and scroll-arrow anatomy is exported from `@tinyrack/ui/components/select` with corresponding prop and state types. Give `Trigger` an accessible name, keep `value` and `open` controlled independently when needed, and use `disabled` for unavailable controls versus `readOnly` for focusable values that cannot change.
