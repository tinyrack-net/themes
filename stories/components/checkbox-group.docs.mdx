# Checkbox Group

Coordinates a set of independently toggleable checkbox values and serializes every checked value under one form name.

## Contract

| Axis | Behavior |
| --- | --- |
| value | uncontrolled with `defaultValue`, controlled with `value` and `onValueChange` |
| form | give each checkbox the same `name`; every checked item contributes its `value` |
| availability | group-level `disabled` cascades; individual checkboxes can be `readOnly` |

  return (
    <Form onSubmit={(event) => {
      event.preventDefault();
      const values = new FormData(event.currentTarget).getAll('channels');
      setSubmitted(values.join(', ') || 'none');
    }}>
      <CheckboxGroup defaultValue={['email']}>
        <label><Checkbox.Root name="channels" value="email"><Checkbox.Indicator>✓</Checkbox.Indicator></Checkbox.Root> Email</label>
        <label><Checkbox.Root name="channels" value="sms"><Checkbox.Indicator>✓</Checkbox.Indicator></Checkbox.Root> SMS</label>
      </CheckboxGroup>
      <Button type="submit">Save channels</Button>
      <output aria-live="polite">{submitted}</output>
    </Form>
  );
}` }]}

## API

`CheckboxGroup` and its prop and state types are exported from `@tinyrack/ui/components/checkbox-group`; render `Checkbox.Root` children with distinct `value` props.
