## Contract

| Axis | Behavior |
| --- | --- |
| state | uncontrolled with `defaultChecked`, controlled with `checked` and `onCheckedChange` |
| semantics | pair the switch with a visible label; use a checkbox when submitting a batch choice |
| form | supports `form`, `name`, `value`, `uncheckedValue`, and `required` through its hidden native input |

## API

`TRSwitch`, `TRSwitchRoot`, `TRSwitchThumb`, `TRSwitchRootProps`, `TRSwitchThumbProps`, `TRSwitchRootState`, and `TRSwitchThumbState` are exported from `@tinyrack/ui/components/switch`; CSS remains opt-in. Associate visible label text with `Root`, use Space to toggle, and choose `disabled` for unavailable controls versus `readOnly` for focusable values that cannot change. `form`, `name`, `value`, `uncheckedValue`, and `required` are carried by the hidden native input for form submission, reset, and validation.
