## コントラクト

`TRField` は Base UI Field の全 anatomy をラップします。`Label`、`Description`、表示中の `Error` は control に自動関連付けされます。Root の `name` と `disabled` は Control の同名 prop より優先されます。

| Part | 責務 |
| --- | --- |
| `Root` | name、利用可否、validation timing、dirty/touched/filled/focused state |
| `Label` | accessible name と click-to-focus target |
| `Control` | native input value、event、validity、form data、reset |
| `Description` | `aria-describedby` に含まれる常時表示の説明 |
| `Error` | native validity または外部 state に対応する復旧メッセージ |
| `Validity` | value、errors、native `ValidityState` を読む render prop |
| `Item` | collection 内の checkbox/radio と label/description の配置 |

値を focus 可能なまま submit する場合は Control の `readOnly`、利用不可かつ form data から除外する場合は Root の `disabled` を使います。標準 part の組み合わせでは `aria-labelledby` と `aria-describedby` を手動設定しません。

## API

### `TRField.Root`

| Prop | 型 / default | 用途 |
| --- | --- | --- |
| `name` | `string` | field を命名し `Control.name` より優先 |
| `disabled` | `boolean` / `false` | control を無効化し全 part に `data-disabled` を付与 |
| `invalid`, `dirty`, `touched` | `boolean` | 外部管理の field state |
| `validate` | sync/async validator | error、error array、または `null` を返す |
| `validationMode` | `'onSubmit' \| 'onBlur' \| 'onChange'` / `'onSubmit'` | validation timing |
| `validationDebounceTime` | `number` / `0` | onChange validation の debounce（ms） |
| `actionsRef` | `RefObject<TRFieldRootActions>` | `validate()` を公開 |
| `uiSize` | `'sm' \| 'md' \| 'lg'` / `'md'` | visual control size |

`Control` は native input props、`value` / `defaultValue`、`onValueChange(value, details)` を受け取ります。`Label.nativeLabel={false}` は `render` が非 label host を使う場合に指定します。`Error.match` は `true` または `valueMissing` などの `ValidityState` key、`Item.disabled` は単一 option の無効化、`Validity` は `validity`、`value`、`initialValue`、`error`、`errors` を提供します。

全 host part は ref、native attribute/event、関数形式の `className` / `style`、Base UI `render` を保持します。namespace、7 named part、各 props/state type、`TRFieldUiSize`、`TRFieldRootActions`、`TRFieldValidityData`、Control change-event type はすべて個別 export されます。CSS は opt-in です。
