## コントラクト

`Root`、`Directory`、`File` は MDX children とネイティブリストの意味を保ちます。ディレクトリはプラットフォームの `details` disclosure を使います。

## 例

有効なリスト構造を保つには、ディレクトリ内に別の `Root` をネストします。

## API

すべての part はネイティブ props と React 19 refs を転送します。`defaultOpen` はディレクトリ disclosure の初期状態を制御します。
