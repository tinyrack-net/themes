## コントラクト

| 軸 | 動作 |
| --- | --- |
| state | input text, selected value, and open state are separate controllable axes |
| input anatomy | Label → InputGroup → optional InputAdornment + Input + Clear + Trigger; Icon belongs inside Trigger |
| multiple anatomy | Chips → Value → Chip + ChipRemove; grid collections compose Collection → Row → Item |
| popup anatomy | Portal → Positioner → Popup → Status + List; groups, indicators, separators, and Empty cover collection feedback |

## API

The complete label, input, popup, multiple-chip, row, collection, group, item-indicator, status, empty-state, and arrow anatomy is exported from `@tinyrack/ui/components/combobox` with matching generic prop and state types. Use `InputAdornment` for decorative content beside the input; `Icon` is the popup indicator and belongs inside `Trigger`. Use a native `label` or `Field.Label` for an editable `Combobox.Input`; `Combobox.Label` specifically names a trigger-only select anatomy.
