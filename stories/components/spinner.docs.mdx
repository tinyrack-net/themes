# Spinner

Compact progress feedback for work without a measurable completion value.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `tr-spinner` | `Spinner` | Native status or decorative span | Label standalone progress; hide embedded decoration. |
| Size | `data-size` | `size` | `sm`, `md`, `lg`; default `md` | Match the surrounding control density. |
| Variant | `data-variant` | `variant` | `currentColor`, `neutral`, `primary`, `danger`; default `currentColor` | Prefer current color inside a Button. |
