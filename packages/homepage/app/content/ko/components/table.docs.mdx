## API

`Table.Root`는 native table을 overflow container로 감쌉니다. 키보드 사용자가 넓은 table을 이동해야 하면 `containerProps`로 해당 영역에 접근 가능한 이름과 `tabIndex={0}`을 제공하세요. `containerRef`는 같은 wrapper를 가리키고 일반 `ref`는 계속 `<table>`을 가리킵니다.
