## Contract

Navigation accepts recursive `group`, `page`, and `link` data. `renderLink` injects the active router link through the Base UI render contract.

## Examples

Nested groups open automatically when they contain the active or pending page.

## API

`currentPath` and `pendingPath` are plain values. The component does not import React Router or calculate routes.
