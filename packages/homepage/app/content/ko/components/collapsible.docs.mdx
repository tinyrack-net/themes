| 축 | 값 | 기본값 |
| --- | --- | --- |
| state | controlled `open` or uncontrolled `defaultOpen` | closed |
| availability | `disabled` | interactive |
| lifecycle | unmount, `keepMounted`, or `hiddenUntilFound` | unmount |

## API

Base UI는 button 의미, `aria-expanded`, panel 관계, focus, transition-state data attribute를 제공합니다. Tinyrack CSS는 `--tr-collapsible-duration`, `--tinyrack-duration-normal`, `--tinyrack-ease-standard`로 panel을 애니메이션하고 reduced motion을 선호하면 장식적 motion을 제거합니다.
