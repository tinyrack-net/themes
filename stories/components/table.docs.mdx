# Table

Native table structure with a compact Tinyrack visual contract.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `tr-table` | `Table` | Required | Keep native table semantics. |
| Part | `tr-table-container` | `TableContainer` | Required when columns may overflow | Wrap wide tables so the page itself never scrolls horizontally. |
| Size | `data-density` | `density` | `compact`, `normal` (default), `comfortable` | Use `normal` for most operational tables. |
| Variant | `data-striped="true"` | `striped` | `false` (default), `true` | Add striping only when it materially improves row scanning. |
| Native | Table and div attributes | Native table and div props | Pass-through | Use real `caption`, `thead`, `tbody`, `th`, and `td` elements. |

## Guidance

<ComponentGuidance
  whenToUse={[
    'Use a table when users need to compare values across consistent rows and columns.',
    'Use TableContainer whenever content can exceed the available viewport width.',
  ]}
  avoid={[
    'Do not use table markup only to align page layout; use grid or flex layout instead.',
    'Do not collapse operational columns into ambiguous abbreviations just to avoid horizontal scrolling.',
  ]}
  accessibility={[
    'Provide a concise caption when the surrounding heading does not fully name the data set.',
    'Use scope="col" for column headers and scope="row" when the first cell identifies its row.',
    'Keep status meaning in text rather than relying on color or striping alone.',
  ]}
/>

## API

| Prop | Values | Note |
| --- | --- | --- |
| `density` | `compact`, `normal`, `comfortable` | Default `normal`; sets `data-density`. |
| `striped` | `boolean` | Default `false`; adds `data-striped="true"` when enabled. |
| `className` | `string` | Merged with `tr-table` or `tr-table-container`. |
| native table props | `TableHTMLAttributes<HTMLTableElement>` | Passed through to `<table>`. |
| native div props | `HTMLAttributes<HTMLDivElement>` | Passed through to `TableContainer`. |
