# Link

Native anchor behavior with Tinyrack text-link styling.

## Contract

| Surface | API | Values | Default | Rule |
| --- | --- | --- | --- | --- |
| CSS class | `tr-link` | anchor element | required | Use on native anchor links. |
| Variant attribute | `data-variant` | `neutral`, `primary`, `danger` | `primary` | Use `primary` for normal navigation. |
| Underline attribute | `data-underline` | `none`, `hover`, `always` | `hover` | Preserve recognizable link affordance. |
| React prop | `href` | native anchor href | required for navigation | Use buttons for actions. |
| Native props | `AnchorHTMLAttributes<HTMLAnchorElement>` | anchor props | pass-through | Keep browser link behavior. |

## Install

| Surface | Install | Import |
| --- | --- | --- |
| CSS / HTML | `pnpm add @tinyrack/ui tailwindcss` | `@tinyrack/ui/core/core.css` and `@tinyrack/ui/components/link/link.css` |
| React | `pnpm add @tinyrack/ui react react-dom` | `@tinyrack/ui/components/link/react` |
| React MDX | `pnpm add @tinyrack/ui @mdx-js/react` | `@tinyrack/ui/mdx/react` maps markdown links. |
| Astro MDX | `pnpm add @tinyrack/ui astro @astrojs/mdx` | `@tinyrack/ui/mdx/astro` maps markdown links. |

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/link/link.css";
```

## API

| Prop | Values | Note |
| --- | --- | --- |
| `variant` | `neutral`, `primary`, `danger` | Default `primary`; sets `data-variant`. |
| `underline` | `none`, `hover`, `always` | Default `hover`; sets `data-underline`. |
| `href` | `string` | Native anchor href. |
| `className` | `string` | Merged with `tr-link`. |
| native anchor props | `AnchorHTMLAttributes<HTMLAnchorElement>` | Passed through to `<a>`. |
