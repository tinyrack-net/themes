# Modal

Focus-trapped dialog surfaces with four sizes and five placements.

## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `size` on `Popup` | `sm`, `md`, `lg`, `full` | `md` |
| `placement` on `Popup` | `middle`, `top`, `bottom`, `start`, `end` | `middle` |
| state | controlled `open` or uncontrolled `defaultOpen` | closed |
