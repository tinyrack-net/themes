# Badge

Compact status labels with semantic color and density axes.

## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `variant` | `neutral`, `info`, `success`, `warning`, `danger` | `neutral` |
| `size` | `sm`, `md`, `lg` | `md` |

## Install

<ComponentInstall surfaces={[{ label: 'React', install: 'pnpm add @tinyrack/ui react react-dom', imports: ["import '@tinyrack/ui/core.css';", "import '@tinyrack/ui/components/badge.css';", "import { Badge } from '@tinyrack/ui/components/badge';"] }]} />
