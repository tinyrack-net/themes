## 계약

| 축 | 동작 |
| --- | --- |
| value | `defaultValue`로 비제어하거나 `value`와 `onValueChange`로 제어합니다. |
| open | `defaultOpen`으로 비제어하거나 `open`과 `onOpenChange`로 제어합니다. |
| disclosure | `Trigger`가 focus를 소유하고 `Portal`, `Positioner`, `Popup`이 floating layer를 구성합니다. |
| collection | group, label, separator, disabled item, text, indicator가 menu anatomy를 설명합니다. |
| form | `name`, `form`, `required`, `inputRef`가 native 제출, validation, reset과 통합됩니다. |

trigger가 밀도 높은 컨트롤 표면에 속하면 `TRSelect.Trigger uiSize="sm"`을 사용하세요.

  sources={[{ label: 'React', language: 'tsx', code: String.raw`import { TRButton } from '@tinyrack/ui/components/button';
import { TRField } from '@tinyrack/ui/components/field';
import { TRForm } from '@tinyrack/ui/components/form';
import { TRSelect } from '@tinyrack/ui/components/select';
import { ChevronDown } from 'lucide-react';
import { useState } from 'react';

const racks = { alpha: 'Rack Alpha', beta: 'Rack Beta', gamma: 'Rack Gamma', staging: 'Staging rack' } as const;

function RackSelectForm() {
  const [attempted, setAttempted] = useState(false);

  const invalid = attempted && value === null;

        <TRSelect.Root items={racks} name="rack" onValueChange={setValue} required value={value}>

          <TRSelect.Portal><TRSelect.Positioner sideOffset={8}><TRSelect.Popup><TRSelect.Arrow /><TRSelect.ScrollUpArrow aria-label="Scroll up">↑</TRSelect.ScrollUpArrow><TRSelect.List>
            <TRSelect.Group><TRSelect.GroupLabel>Production</TRSelect.GroupLabel>
              <TRSelect.Item value="alpha"><TRSelect.ItemText>Rack Alpha</TRSelect.ItemText><TRSelect.ItemIndicator aria-hidden="true">✓</TRSelect.ItemIndicator></TRSelect.Item>
              <TRSelect.Item value="beta"><TRSelect.ItemText>Rack Beta</TRSelect.ItemText><TRSelect.ItemIndicator aria-hidden="true">✓</TRSelect.ItemIndicator></TRSelect.Item>
              <TRSelect.Item value="gamma"><TRSelect.ItemText>Rack Gamma</TRSelect.ItemText><TRSelect.ItemIndicator aria-hidden="true">✓</TRSelect.ItemIndicator></TRSelect.Item>
            </TRSelect.Group><TRSelect.Separator /><TRSelect.Group><TRSelect.GroupLabel>Non-production</TRSelect.GroupLabel>
              <TRSelect.Item value="staging"><TRSelect.ItemText>Staging rack</TRSelect.ItemText><TRSelect.ItemIndicator aria-hidden="true">✓</TRSelect.ItemIndicator></TRSelect.Item>
            </TRSelect.Group>
          </TRSelect.List><TRSelect.ScrollDownArrow aria-label="Scroll down">↓</TRSelect.ScrollDownArrow></TRSelect.Popup></TRSelect.Positioner></TRSelect.Portal>
        </TRSelect.Root>
        {invalid ? <TRField.Error match>Choose a deployment rack.</TRField.Error> : null}
      </TRField.Root>
      <TRButton type="submit">Deploy</TRButton>
      <output aria-live="polite">{attempted && value ? 'Ready to deploy to ' + racks[value as keyof typeof racks] + '.' : ''}</output>
    </TRForm>
  );
}` }]}
/>

## API

전체 anatomy는 `Root`, `Label`, `Trigger`, `Value`, `Icon`, `Portal`, `Backdrop`, `Positioner`, `Popup`, `List`, `Item`, `ItemIndicator`, `ItemText`, `Arrow`, `ScrollDownArrow`, `ScrollUpArrow`, `Group`, `GroupLabel`, `Separator`입니다. 모든 part는 `TRSelect.Part`와 named `TRSelectPart` export로 제공되며 대응하는 `TRSelectPartProps`, `TRSelectPartState` type을 export합니다.

`Trigger`에 접근 가능한 이름을 제공하세요. 키보드 사용자는 Arrow Down으로 열고, 사용 가능한 option 사이를 이동하고, typeahead를 사용하고, Enter로 확정하며, Escape로 닫은 뒤 trigger로 focus를 돌려받습니다. 필요하면 `value`와 `open`을 독립적으로 제어하세요. 사용할 수 없는 control에는 `disabled`, 변경할 수 없는 focusable value에는 `readOnly`를 사용합니다.
