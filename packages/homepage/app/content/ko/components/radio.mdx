## 계약

| 축 | 동작 |
| --- | --- |
| ownership | selection is owned by the nearest `TRRadioGroup` |
| identity | every item needs a stable `value`; labels should target a unique `id` |
| state | selected, unselected, read-only, and disabled |

밀도 높은 컨트롤 표면에서 작은 radio가 필요하면 `uiSize="sm"`을 사용하세요.

## API

`Root`, `Indicator`와 prop/state type은 `@tinyrack/ui/components/radio`에서 export됩니다. `TRRadioGroup` 안에서만 사용하세요. 전체 label row를 최소 24px 높이로 유지해 보이는 텍스트가 radio glyph보다 큰 pointer target을 만들게 하세요.
