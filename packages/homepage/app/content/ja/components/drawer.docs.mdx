## コントラクト

この Tinyrack wrapper は Base UI の動作、React 19 refs、ネイティブ props、events、`render` contract を保ちます。方向の遷移は `swipeDirection` に従い、swipe gesture と同期し、reduced-motion 設定を尊重します。Playground control は `label`、`open`、`modal`、`swipeDirection`、制御付き `activeSnapPoint` mapping を扱います。

## API

すべての compound part と prop/state type は `@tinyrack/ui/components/drawer` から export されます。style は対応する CSS subpath による opt-in のままです。
