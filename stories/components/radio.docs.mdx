# Radio

The single option primitive for a `RadioGroup`; its indicator reflects the group's selected value.

## Contract

| Axis | Behavior |
| --- | --- |
| ownership | selection is owned by the nearest `RadioGroup` |
| identity | every item needs a stable `value`; labels should target a unique `id` |
| state | selected, unselected, read-only, and disabled |

## API

`Root`, `Indicator`, and their prop and state types are exported from `@tinyrack/ui/components/radio`. Use them only inside `RadioGroup`.
