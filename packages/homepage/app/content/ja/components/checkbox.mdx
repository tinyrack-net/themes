## コントラクト

| Axis | Values |
| --- | --- |
| state | unchecked, checked, `indeterminate` |
| availability | enabled, `readOnly`, `disabled` |
| form | `name`, `value`, `required`, hidden input validity |

密度の高いコントロールサーフェスで小さな checkbox が必要な場合は `uiSize="sm"` を使います。

  const [attempted, setAttempted] = useState(false);
  const [checked, setChecked] = useState(false);
  const invalid = attempted && !checked;

  return (
    <TRForm noValidate onSubmit={(event) => { event.preventDefault(); setAttempted(true); }}>
      <TRField.Root invalid={invalid}>
        <TRField.Label>
          <TRCheckbox.Root checked={checked} name="terms" onCheckedChange={setChecked} required>
            <TRCheckbox.Indicator aria-hidden="true">✓</TRCheckbox.Indicator>
          </TRCheckbox.Root>
          I accept the maintenance window.
        </TRField.Label>
        {invalid ? <TRField.Error match>Accept the maintenance window to continue.</TRField.Error> : null}
      </TRField.Root>
      <TRButton type="submit">Continue</TRButton>
    </TRForm>
  );
}` }]}

<ComponentExampleTabs

  title="外部 form と明示的な off 値"
  description="checkbox は所有 form の外に描画されます。`value` と `uncheckedValue` により FormData で on/off の状態を明示します。"

  sources={[{ label: 'React', language: 'tsx', code: String.raw`import { useState } from 'react';
import { TRButton } from '@tinyrack/ui/components/button';
import { TRCheckbox } from '@tinyrack/ui/components/checkbox';
import { TRForm } from '@tinyrack/ui/components/form';

export function MonitoringSetting() {
  const [result, setResult] = useState('');

  return <>
    <TRForm id="monitoring-form" onSubmit={(event) => {
      event.preventDefault();
      setResult(String(new FormData(event.currentTarget).get('monitoring')));
    }}>
      <TRButton type="submit">Read form value</TRButton>
    </TRForm>
    <label>
      <TRCheckbox.Root
        form="monitoring-form"
        name="monitoring"
        uncheckedValue="disabled"
        value="enabled"
      >
        <TRCheckbox.Indicator aria-hidden="true">✓</TRCheckbox.Indicator>
      </TRCheckbox.Root>
      Monitoring
    </label>
    <output aria-live="polite">{result}</output>
  </>;
}` }]}
/>

## API

`Root`、`Indicator`、それらの prop type、Base UI state type は `@tinyrack/ui/components/checkbox` から export されます。CSS は opt-in のままです。
