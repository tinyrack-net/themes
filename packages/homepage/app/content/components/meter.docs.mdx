# Meter

Displays a scalar measurement within a known range.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Playground controls cover `label`, `min`, `max`, `value`, and number formatting.

## API

Keep `value` inside the current `min`/`max` range and use `format` for visible and announced units. Tinyrack intentionally adds no separate low/high threshold API.
