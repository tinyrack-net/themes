# Tinyrack UI

Install the CSS-first Tinyrack package, import the core token CSS once, then add component CSS only for the components you render.

## Installation

Install the package.

```sh
pnpm add @tinyrack/ui
```

Install the peers required by the surface you use.

| Surface | Peer install | Use |
| --- | --- | --- |
| CSS tokens and utilities | `pnpm add tailwindcss` | Enables Tinyrack prefixed utility classes from `core.css`. |
| React components | `pnpm add react react-dom` | Enables the React wrapper exports. |
| CodeBlock syntax highlight | `pnpm add shiki` | Enables the optional Shiki client highlighter. |

## Usage

### CSS tokens

Import Tailwind first, then Tinyrack core CSS.

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
```

```html
<section data-theme="tinyrack-dark" class="bg-tinyrack-surface text-tinyrack-text">
  <h1 class="font-tinyrack-heading text-tinyrack-primary">Rack A</h1>
  <p class="text-tinyrack-muted">Online and ready for deploy.</p>
</section>
```

### Button CSS

Add Button CSS only where the Button contract is used.

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/button/button.css";
```

```html
<button class="tr-btn" data-size="md" data-variant="primary" data-appearance="solid">
  Deploy
</button>
```

### React Button

```tsx
import '@tinyrack/ui/core/core.css';
import '@tinyrack/ui/components/button/button.css';
import { Button, IconButton } from '@tinyrack/ui/components/button/react';

export function DeployAction() {
  return (
    <Button size="md" variant="primary" appearance="solid">
      Deploy
    </Button>
  );
}
```

### Primitive CSS

Import only the primitive groups used by the rendered page.

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/link/link.css";
@import "@tinyrack/ui/components/form/form.css";
```

Use Tailwind utilities directly for layout and spacing.

```html
<section class="mx-auto grid max-w-5xl gap-6 px-4">
  <div class="flex flex-wrap items-center gap-2">...</div>
</section>
```

### React Form

```tsx
import '@tinyrack/ui/core/core.css';
import '@tinyrack/ui/components/form/form.css';
import { Field, FormMessage, Input, Label } from '@tinyrack/ui/components/form/react';

export function RackField() {
  return (
    <Field>
      <Label htmlFor="rack-name">Rack name</Label>
      <Input id="rack-name" placeholder="rack-a-01" />
      <FormMessage>Use a stable rack identifier.</FormMessage>
    </Field>
  );
}
```

### Table CSS

Add Table CSS only where the Table contract is used.

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/table/table.css";
```

```html
<div class="tr-table-container">
  <table class="tr-table" data-density="normal">
    <thead>
      <tr>
        <th scope="col">Node</th>
        <th scope="col">Load</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>rack-a-01</td>
        <td>41%</td>
      </tr>
    </tbody>
  </table>
</div>
```

### React Table

```tsx
import '@tinyrack/ui/core/core.css';
import '@tinyrack/ui/components/table/table.css';
import { Table, TableContainer } from '@tinyrack/ui/components/table/react';

export function RackTable() {
  return (
    <TableContainer>
      <Table density="normal">
        <thead>
          <tr>
            <th scope="col">Node</th>
            <th scope="col">Load</th>
          </tr>
        </thead>
        <tbody>
          <tr>
            <td>rack-a-01</td>
            <td>41%</td>
          </tr>
        </tbody>
      </Table>
    </TableContainer>
  );
}
```

### React Tabs

```tsx
import '@tinyrack/ui/core/core.css';
import '@tinyrack/ui/components/tabs/tabs.css';
import { Tabs, TabsList, TabsPanel, TabsTrigger } from '@tinyrack/ui/components/tabs/react';

export function RackSections() {
  return (
    <Tabs defaultValue="overview">
      <TabsList aria-label="Rack sections">
        <TabsTrigger value="overview">Overview</TabsTrigger>
        <TabsTrigger value="logs">Logs</TabsTrigger>
      </TabsList>
      <TabsPanel value="overview">Rack A is healthy.</TabsPanel>
      <TabsPanel value="logs">03:18 backup completed.</TabsPanel>
    </Tabs>
  );
}
```

## Use boundary

| Use this when | Do not use this for |
| --- | --- |
| Product surfaces need compact color, type, spacing, and radius decisions. | Another component system wrapper or adapter gallery. |
| Apps can compose CSS explicitly from stable package paths. | Root-level React exports from `@tinyrack/ui`. |
| Owned component workflows still care about native markup. | Visual primitives outside the public core export. |

## Public paths

| Path | Entry | Use |
| --- | --- | --- |
| Core metadata | `@tinyrack/ui/core` | Token values for non-CSS consumers. |
| CSS tokens | `@tinyrack/ui/core/core.css` | Theme variables and prefixed utility classes. |
| React Badge | `@tinyrack/ui/components/badge/react` | Owned React wrapper for compact status metadata. |
| Badge CSS | `@tinyrack/ui/components/badge/badge.css` | Standalone badge contract CSS. |
| React Button | `@tinyrack/ui/components/button/react` | Owned React wrapper for the Button contract. |
| Button CSS | `@tinyrack/ui/components/button/button.css` | Standalone button contract CSS. |
| React Code | `@tinyrack/ui/components/code/react` | Inline code display wrapper. |
| Code CSS | `@tinyrack/ui/components/code/code.css` | Standalone inline code CSS. |
| React CodeBlock | `@tinyrack/ui/components/code-block/react` | SSR-safe plain block code wrapper. |
| Shiki CodeBlock | `@tinyrack/ui/components/code-block/shiki-react` | Optional client syntax highlight enhancer. |
| CodeBlock CSS | `@tinyrack/ui/components/code-block/code-block.css` | Standalone block code CSS. |
| React Link | `@tinyrack/ui/components/link/react` | Native anchor wrapper for the Link contract. |
| Link CSS | `@tinyrack/ui/components/link/link.css` | Standalone link contract CSS. |
| React Form | `@tinyrack/ui/components/form/react` | Native form primitive wrappers. |
| Form CSS | `@tinyrack/ui/components/form/form.css` | Standalone form control CSS. |
| React Table | `@tinyrack/ui/components/table/react` | Owned React wrapper for the Table contract. |
| Table CSS | `@tinyrack/ui/components/table/table.css` | Standalone table contract CSS. |
| React Tabs | `@tinyrack/ui/components/tabs/react` | Owned React wrapper for the Tabs contract. |
| Tabs CSS | `@tinyrack/ui/components/tabs/tabs.css` | Standalone tabs contract CSS. |
