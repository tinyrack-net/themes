| availability | enabled or `disabled` | enabled |
| label | visible text | required |

  return (
    <div className="grid justify-items-start gap-3">
      <TRToggle onPressedChange={setPressed} pressed={pressed}>

      </TRToggle>

      </output>
    </div>
  );

## API

TRToggle は Base UI button の動作、ref、ネイティブ props、event、`render` を保ちます。`onPressedChange` と制御付き `pressed`、または非制御の `defaultPressed` を使います。ネイティブの Space/Enter activation と `disabled` の意味は button の動作に従います。各 toggle には表示される label を提供し、TRToggleGroup item には安定した `value` も必要です。polymorphic render は button の keyboard semantics を保つ必要があります。フォーム送信や required 検証に値を参加させる場合は TRCheckbox または TRSwitch を使います。
