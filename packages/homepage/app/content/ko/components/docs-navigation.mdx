## 계약

`TRDocsNavigation`은 이름이 있는 탐색 landmark 안에 재귀적인 `group`, `page`, `link` 항목을 렌더링합니다. 현재 또는 대기 경로가 포함된 그룹은 라우트 상태가 바뀐 뒤에도 자동으로 열립니다. 사용자는 포인터나 키보드로 다른 그룹을 열고 닫을 수 있습니다.

## 예시

### React Router

애플리케이션에 `react-router`를 설치한 뒤 현재 경로와 대기 경로를 읽습니다. `renderLink`는 Base UI render contract를 사용하므로 생성된 콘텐츠, 클래스, 접근성 상태, ref, 이벤트 핸들러가 라우터 링크에 병합됩니다.

## 접근성

루트는 `label`로 이름을 지정한 `<nav>`입니다. 현재 페이지에는 `aria-current="page"`가 적용되고 대기 링크에는 `data-pending`과 장식용 spinner가 표시됩니다. 그룹 trigger는 Collapsible가 제공하는 기본 button이며 `Enter`, `Space`, 일반 tab 탐색을 지원합니다. 항목 label은 명확하게 작성하고 기본 `Documentation` 이름이 적합하지 않으면 `label`을 현지화하세요.

## API

### `TRDocsNavigation`

| Prop | Type | 기본값 | 용도 |
| --- | --- | --- | --- |
| `items` | `readonly TRDocsNavigationItem[]` | 필수 | 재귀적 탐색 데이터입니다. |
| `currentPath` | `string` | 필수 | 현재 페이지로 표시할 정확한 경로입니다. |
| `pendingPath` | `string` | `undefined` | 대기 상태로 표시할 정확한 목적지 경로입니다. |
| `defaultGroupsOpen` | `boolean` | `false` | 현재 또는 대기 경로가 없는 그룹도 처음에 엽니다. |
| `label` | `string` | `'Documentation'` | 탐색 landmark의 접근 가능한 이름입니다. |
| `renderLink` | `(item, state) => ReactElement` | 기본 anchor | 프레임워크 라우터 링크를 주입합니다. `state`에는 `active`와 `pending`이 있습니다. |
| `onNavigate` | `(item) => void` | `undefined` | 페이지나 링크가 활성화된 뒤 실행됩니다. 모바일 drawer를 닫을 때 유용합니다. |

`TRDocsNavigation`은 기본 `nav` prop, `className`, `style`, 이벤트, data attribute, React 19 `ref`도 전달합니다.

### `TRDocsNavigationItem`

`page` 항목은 `label`, `path`, `type: 'page'`를 가집니다. `link` 항목에는 선택적 `external`이 추가됩니다. 외부 목적지에 target이나 별도 표시가 필요하면 `renderLink`에서 이 메타데이터를 사용하세요. `group` 항목은 `label`, `type: 'group'`, 재귀적인 `children`을 가집니다.

내보내는 데이터 type은 `TRDocsNavigationItem`, `TRDocsNavigationGroup`, `TRDocsNavigationPage`, `TRDocsNavigationLink`입니다. `TRDocsNavigationLinkState`는 `renderLink`에 전달되는 `active`, `pending` boolean을 포함하며 `TRDocsNavigationProps`는 전체 컴포넌트 prop type입니다.

## 스타일

core token 다음에 컴포넌트 stylesheet를 import하세요. `--tr-docs-navigation-gap`, `--tr-docs-navigation-item-gap`, `--tr-docs-navigation-group-color`, `--tr-docs-navigation-link-color`, `--tr-docs-navigation-link-hover-background`, `--tr-docs-navigation-link-hover-color`, `--tr-docs-navigation-link-active-border-color`를 재정의할 수 있습니다. 긴 label은 좁은 sidebar와 모바일 drawer 안에서 줄바꿈됩니다.

## SSR

서버 출력에는 현재 경로와 처음 열린 상위 그룹이 포함됩니다. 동일한 `items`, `currentPath`, `pendingPath`로 hydrate하세요. hydration 뒤 라우트가 바뀌면 탐색 트리를 교체하지 않고 관련 그룹을 엽니다.
