# Card

Structured content surfaces with semantic sections, elevation, and padding density.

## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `variant` | `default`, `outlined`, `elevated` | `default` |
| `padding` | `none`, `sm`, `md`, `lg` | `md` |
| parts | `Root`, `Header`, `Title`, `Description`, `Content`, `Footer` | compound API |

<ComponentInstall surfaces={[{ label: 'React', install: 'pnpm add @tinyrack/ui react react-dom', imports: ["import '@tinyrack/ui/core.css';", "import '@tinyrack/ui/components/card.css';", "import { Card } from '@tinyrack/ui/components/card';"] }]} />

## API

`Card.Root` and `Card.Title` support Base UI's additive `render` contract. Every part forwards native props and `ref`; keep headings in document order when composing cards.
