## API

`TRAlert.Root` はネイティブ `div` props と React 19 `ref` を転送します。通知の緊急度はアプリケーションの文脈で決まるため、live-region role は自動付与しません。既定の `div` や `p` が周囲の意味に合わない場合、`TRAlert.Title` と `TRAlert.Description` は Base UI の `render` contract に対応します。
