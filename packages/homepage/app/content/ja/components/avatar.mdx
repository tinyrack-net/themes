## API

- `TRAvatar.Root` は Base UI root props、`className`、`style`、`render`、React 19 `ref` を転送します。`uiSize` の既定値は `md`、`shape` の既定値は `circle` です。
- `TRAvatar.Image` は `render`、`ref`、`onLoadingStatusChange` を含む native image と Base UI props を転送します。ID 画像には意味のある `alt` テキストを使います。
- `TRAvatar.Fallback` は Image がない場合または失敗した場合に表示されます。内容は短く認識しやすく保ち、Image の読み込み中のちらつきを避けるには `delay` をミリ秒単位で指定します。
- コンポーネント単位のスタイルには `--tr-avatar-size`、`--tr-avatar-radius`、`--tr-avatar-background`、`--tr-avatar-border`、`--tr-avatar-color`、`--tr-avatar-font-size`、`--tr-avatar-font-weight` を上書きします。
