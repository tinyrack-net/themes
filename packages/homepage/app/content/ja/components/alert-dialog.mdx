## コントラクト

この Tinyrack wrapper は Base UI の動作、React 19 refs、ネイティブ props、events、`render` contract を保ちます。Playground の control は `label`、`open`、`disabled`、共有 `handleMode` の関連付けを扱います。

## API

すべての compound part と prop/state type は `@tinyrack/ui/components/alert-dialog` から export されます。style は対応する CSS subpath で opt-in のままです。確認コントロールを `tr-alert-dialog-actions` にまとめると、狭い viewport で読みやすいアクションへ折り返せます。Cancel には secondary TRButton、破壊的な確認には danger TRButton を使います。
