## 계약

`Root`, `Directory`, `File`은 MDX children과 native list 의미를 보존합니다. 디렉터리는 플랫폼 `details` disclosure를 사용합니다.

## 예시

유효한 list 구조를 보존하려면 디렉터리 안에 다른 `Root`를 중첩하세요.

## API

모든 part는 native props와 React 19 refs를 전달합니다. `defaultOpen`은 디렉터리 disclosure의 초기 상태를 제어합니다.
