React 19アプリケーションにTailwind CSS 4.3以降とTinyrack UIを設定し、テーマを
適用した主要アクションを表示します。既存のVite Reactアプリケーションと
Node.js 24以降が必要です。

`@tailwindcss/vite`を開発依存関係としてインストールし、TailwindのViteプラグインを
設定に追加してください。

## 3. CSSの読み込み順を設定する

グローバルエントリーポイントで`core.css`を一度だけ読み込み、すべての
コンポーネントスタイルより前に配置してください。コンポーネントのJavaScriptは
CSSを自動で読み込みません。

## 4. テーマを選ぶ

ドキュメントルートに`data-theme`を設定してください。`rack-blue`は組み込みの
製品テーマです。

## 5. 主要アクションを表示する

公開サブパスからコンポーネントを読み込み、`intent`でアクションの意味を
指定してください。

<GettingStartedCode label="主要ボタンの例" snippet="button" />

## 結果を確認する

Vite開発サーバーを起動してください。ボタンに青いテーマと主要アクションの
スタイルが適用され、キーボードフォーカスが表示されます。スタイルが適用されない
場合は、`core.css`が`button.css`より前にあることを確認してください。

[基礎](/ja/foundations/)でシステムのルールを理解するか、
[コンポーネント](/ja/components/)からインターフェースを構築してください。
