## コントラクト

| 軸 | 値 | 既定値 |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | `md` |
| `shape` | `circle`, `square` | `circle` |
| parts | `Root`, `Image`, `Fallback` | compound API |
