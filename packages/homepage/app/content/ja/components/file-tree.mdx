## コントラクト

`TRFileTree` は、記述された Markdown の unordered list を意味のあるファイル一覧に変換します。ネストした list はディレクトリになり、`/` で終わる項目は装飾的な placeholder を持つ空のディレクトリになります。ディレクトリはプラットフォームの `details` disclosure を使い、デフォルトで開き、pointer、Enter、Space のネイティブ操作を維持します。

## API

`TRFileTree` に component 固有の scalar props はありません。`children` には記述した unordered list を渡し、`aria-label`、`className`、`style`、event handler、React 19 の `ref` を含む native `ul` props を転送します。周囲の content で tree がラベル付けされていない場合は、`aria-label` で accessible name を指定してください。

表示は `--tr-file-tree-background`、`--tr-file-tree-border`、`--tr-file-tree-border-width`、`--tr-file-tree-color`、`--tr-file-tree-font-family`、`--tr-file-tree-font-size`、`--tr-file-tree-gap`、`--tr-file-tree-indent`、`--tr-file-tree-margin`、`--tr-file-tree-marker`、`--tr-file-tree-padding`、`--tr-file-tree-placeholder-color`、`--tr-file-tree-radius` で調整できます。各 token は対応する Tinyrack semantic token に fallback します。
