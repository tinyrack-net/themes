## コントラクト

この Tinyrack wrapper は Base UI の動作、React 19 の ref、native prop、event、`render` contract を維持します。Playground は orientation、Root の disabled、controlled toggle state を扱い、下の example で focus loop と個別の disabled 動作を示します。Bold を操作すると controlled preview arg が更新され、Reset は Bold の pressed state とすべての control をデフォルトに戻します。

## API

### Anatomy

```tsx
<TRToolbar.Root aria-label="Editor controls">
  <TRToolbar.Group aria-label="Formatting"><TRToolbar.Button>Bold</TRToolbar.Button></TRToolbar.Group>
  <TRToolbar.Separator />
  <TRToolbar.Link href="#help">Help</TRToolbar.Link>
  <TRToolbar.Input aria-label="Document title" />
</TRToolbar.Root>
```

完全な anatomy は `Root`, `Group`, `Button`, `Link`, `Input`, and `Separator` です。各 part は `TRToolbar.Part` と named `TRToolbarPart` export の両方で提供され、対応する props/state type も export されます。

### Root props

| Prop | Type | デフォルト | 用途 |
| --- | --- | --- | --- |
| `orientation` | `horizontal \| vertical` | `horizontal` | Left/Right または Up/Down の roving navigation を選択します。 |
| `loopFocus` | `boolean` | `true` | 両端で keyboard focus を循環します。 |
| `disabled` | `boolean` | `false` | toolbar 内の button と input を無効にします。 |
| `render` | `ReactElement \| function` | `<div>` | 動作と ref を保ちながら Root element を置換します。 |

`Group` は名前付き `group` を描画し、内部の button と input を無効にできます。`Button` はデフォルトで `type="button"` となり、`focusableWhenDisabled` をサポートします。`false` にすると disabled command を roving focus から外します。`Input` も同じ disabled-focus policy と通常の text-editing key を維持します。`Link` は意図的に常に navigation 可能で、disabled Root/Group 内でも有効です。`Separator` は Root と反対の orientation がデフォルトです。horizontal toolbar では Input を最大 1 つ、最後に配置してください。

すべての part は `className`、`style`、native attribute、event handler、React 19 ref、Base UI の state-driven `render` contract を保持します。すべての Root と Group に accessible name を付けてください。

### Appearance

| Token | Fallback | 用途 |
| --- | --- | --- |
| `--tr-toolbar-background` | `--tinyrack-surface` | Toolbar surface. |
| `--tr-toolbar-border` | `--tinyrack-border` | Toolbar border. |
| `--tr-toolbar-color` | `--tinyrack-text` | Toolbar text. |
| `--tr-toolbar-control-background` | `transparent` | Idle button surface. |
| `--tr-toolbar-control-background-hover` | `--tinyrack-surface-hover` | Enabled button/link hover. |
| `--tr-toolbar-control-background-pressed` | `--tinyrack-surface-pressed` | Active/pressed control. |
| `--tr-toolbar-input-background` | `--tinyrack-surface` | Input surface. |
| `--tr-toolbar-input-border` | `--tinyrack-control-border` | Input border. |
| `--tr-toolbar-separator-color` | `--tinyrack-border` | Separator color. |

`@tinyrack/ui/components/toolbar.css` を明示的に import してください。foundation token が light/dark color、geometry、focus indicator、disabled opacity を提供します。
