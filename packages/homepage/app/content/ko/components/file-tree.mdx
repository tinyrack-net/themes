## 계약

`TRFileTree`는 Markdown unordered list를 children으로 받습니다. 중첩된 list는 디렉터리가 되고, `/`로 끝나는 list item은 비어 있는 디렉터리가 됩니다. 디렉터리는 플랫폼 `details` disclosure를 사용합니다.

## 예시

다른 Markdown list를 중첩해 디렉터리를 만드세요. `...` 또는 `…`은 placeholder로 사용할 수 있으며, `**bold**`나 `` `code` `` 같은 inline Markdown도 보존됩니다.

## API

`TRFileTree`는 native `ul` props와 React 19 refs를 전달합니다. 디렉터리는 기본으로 펼쳐지며 개별적으로 토글할 수 있습니다.
