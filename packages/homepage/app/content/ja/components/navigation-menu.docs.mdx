## コントラクト

Use TRNavigationMenu for a site header with a brand, a small number of direct destinations, and optional Product or Resources flyouts. The desktop composition keeps Base UI roving keyboard navigation; the mobile composition exposes the same destinations in a TRDrawer.

## API

すべての compound part と prop/state type は `@tinyrack/ui/components/navigation-menu` から export されます。style は対応する CSS subpath による opt-in のままです。

TRNavigationMenu は十分な横幅で pointer と keyboard によるサイトナビゲーションを行うためのものです。flyout group は task 指向に保ち、preview が不要な destination には direct link を残します。狭い breakpoint では horizontal keyboard model を圧縮せず、同じ destination set を持つ TRDrawer 構成に切り替えます。
