# Button

Native button behavior with Tinyrack size, variant, and appearance axes.

## Contract

| Surface | API | Values | Default | Rule |
| --- | --- | --- | --- | --- |
| CSS class | `tr-btn` | button element | required | Use for text buttons with native button semantics. |
| CSS class | `tr-icon-btn` | button element | required for icon-only | Pair with an accessible label. |
| Size attribute | `data-size` | `sm`, `md`, `lg` | `md` | Use `md` for normal actions. |
| Variant attribute | `data-variant` | `neutral`, `primary`, `danger` | `neutral` | Use `primary` once per action group. |
| Appearance attribute | `data-appearance` | `solid`, `outline`, `ghost` | `solid` | Choose emphasis without changing semantics. |
| React prop | `label` | `string` | required for `IconButton` | Names icon-only actions. |
| Native props | `ButtonHTMLAttributes<HTMLButtonElement>` | button props | pass-through | `type` defaults to `button`. |

## Install

| Surface | Install | Import |
| --- | --- | --- |
| CSS / HTML | `pnpm add @tinyrack/ui tailwindcss` | `@tinyrack/ui/core/core.css` and `@tinyrack/ui/components/button/button.css` |
| React | `pnpm add @tinyrack/ui react react-dom` | `@tinyrack/ui/components/button/react` |
| Icons | `pnpm add lucide-react` | Optional for `IconButton` examples. |

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/button/button.css";
```

## API

| Prop | Values | Note |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | Default `md`; sets `data-size`. |
| `variant` | `neutral`, `primary`, `danger` | Default `neutral`; sets `data-variant`. |
| `appearance` | `solid`, `outline`, `ghost` | Default `solid`; sets `data-appearance`. |
| `label` | `string` | Required for `IconButton` accessible names. |
| Icon child | `ReactNode` | Use `aria-hidden="true"` when `label` names the action. |
| `disabled` | `boolean` | Native disabled state. |
| `type` | `button`, `submit`, `reset` | Default `button`. |
| native button props | `ButtonHTMLAttributes<HTMLButtonElement>` | Passed through to `<button>`. |
