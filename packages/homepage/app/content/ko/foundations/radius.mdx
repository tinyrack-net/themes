## 구현

Radius 값은 public TypeScript metadata와 runtime CSS token입니다. 컴포넌트 CSS는 semantic radius role을 사용하고 애플리케이션 layout은 접두사 Tailwind utility를 사용할 수 있습니다.

```tsx
<TRCard.Root>
  <TRButton>배포</TRButton>
  <TRBadge>준비됨</TRBadge>
</TRCard.Root>
```

```ts
import { tinyrackRadii } from '@tinyrack/ui/core';

const panelRadius = tinyrackRadii.lg;
const statusRadius = tinyrackRadii.full;
```

CSS에서는 `--tinyrack-radius-*`를, Tailwind markup에서는 `rounded-tinyrack-*` utility를 사용하세요.
