## 계약

`Root`는 native fieldset grouping과 disabled 전파를 제공합니다. `Legend`가 group의 이름을 지정하며 각 child control에는 여전히 자체 visible label이 필요합니다.

## API

`Root`와 `Legend`는 개별적으로도 `TRFieldset`을 통해서도 export됩니다. 중첩된 form control은 native 제출 동작을 유지합니다.
