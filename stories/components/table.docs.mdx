# Table

Native table structure with a compact Tinyrack visual contract.

## Contract

| Surface | API | Values | Default | Rule |
| --- | --- | --- | --- | --- |
| Container class | `tr-table-container` | div element | required for overflow | Wrap wide tables for horizontal scrolling. |
| Table class | `tr-table` | table element | required | Keep native table semantics. |
| Density attribute | `data-density` | `compact`, `normal`, `comfortable` | `normal` | Use `normal` for operational tables. |
| Striped attribute | `data-striped` | `true` | unset | Opt in for dense row scanning. |
| React prop | `striped` | `boolean` | false | Adds `data-striped="true"`. |
| Native props | table and div attributes | native values | pass-through | Use real `thead`, `tbody`, `th`, and `td`. |

## Install

| Surface | Install | Import |
| --- | --- | --- |
| CSS / HTML | `pnpm add @tinyrack/ui tailwindcss` | `@tinyrack/ui/core/core.css` and `@tinyrack/ui/components/table/table.css` |
| React | `pnpm add @tinyrack/ui react react-dom` | `@tinyrack/ui/components/table/react` |
| React MDX | `pnpm add @tinyrack/ui @mdx-js/react` | `@tinyrack/ui/mdx/react` maps markdown tables. |
| Astro MDX | `pnpm add @tinyrack/ui astro @astrojs/mdx` | `@tinyrack/ui/mdx/astro` maps markdown tables. |

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/table/table.css";
```

## API

| Prop | Values | Note |
| --- | --- | --- |
| `density` | `compact`, `normal`, `comfortable` | Default `normal`; sets `data-density`. |
| `striped` | `boolean` | Default `false`; adds `data-striped="true"` when enabled. |
| `className` | `string` | Merged with `tr-table` or `tr-table-container`. |
| native table props | `TableHTMLAttributes<HTMLTableElement>` | Passed through to `<table>`. |
| native div props | `HTMLAttributes<HTMLDivElement>` | Passed through to `TableContainer`. |
