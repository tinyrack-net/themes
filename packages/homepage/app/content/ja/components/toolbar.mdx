## コントラクト

この Tinyrack wrapper は Base UI の動作、React 19 の ref、native prop、event、`render` contract を維持します。Playground control は視覚的な状態である `disabled` と `orientation` を扱い、accessible label は semantic のまま、focus loop はデフォルト動作を使用します。

## API

すべての compound part と prop/state type は `@tinyrack/ui/components/toolbar` から export されます。style は対応する CSS subpath による opt-in のままです。`TRToolbar.Root` は loop 有効の horizontal arrow-key focus が既定で、vertical toolbar は Up/Down を使います。`TRToolbar.Group` は関連 control に名前を付け、`TRToolbar.Input` は通常の text-editing key を保ち、disabled group/item は activation を抑制します。horizontal toolbar では `TRToolbar.Input` を最大 1 つだけ使って最後に置き、text-editing key が後続の toolbar item を妨げないようにします。focus 可能な disabled `TRToolbar.Button` は利用不可であることが見えるようにし、すべての `TRToolbar` と nested `Group` にアクセシブルな名前を付けます。
