# ToggleGroup

Coordinates mutually exclusive or multiple toggle buttons.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Storybook controls cover `disabled`, `multiple`, `orientation`, `value`.
