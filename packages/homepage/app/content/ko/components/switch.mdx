## 계약

| 축 | 동작 |
| --- | --- |
| 상태 | `defaultChecked`를 사용하는 비제어 방식 또는 `checked`와 `onCheckedChange`를 사용하는 제어 방식 |
| 의미 | switch를 보이는 label과 연결합니다. 여러 선택 항목을 한꺼번에 제출할 때는 checkbox를 사용합니다. |
| form | hidden native input을 통해 `form`, `name`, `value`, `uncheckedValue`, `required`를 지원합니다. |

## API

`TRSwitch`, `TRSwitchRoot`, `TRSwitchThumb`, `TRSwitchRootProps`, `TRSwitchThumbProps`, `TRSwitchRootState`, `TRSwitchThumbState`는 `@tinyrack/ui/components/switch`에서 export됩니다. CSS는 opt-in입니다. 보이는 label text를 `Root`와 연결하고 Space로 전환하세요. 변경할 수 없는 focusable value에는 `readOnly`, 사용할 수 없는 control에는 `disabled`를 사용합니다. hidden native input이 `form`, `name`, `value`, `uncheckedValue`, `required`를 전달하여 form 제출, reset, 검증을 지원합니다.
