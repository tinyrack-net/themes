## 계약

`shape`은 `text`, `rectangle`, `circle`을 받으며 기본값은 `text`입니다. Skeleton은 기본적으로 shimmer하며 정적인 placeholder에는 `animate={false}`를 설정하세요. 크기는 layout이나 `style`에서 결정됩니다.

## API

An unlabeled Skeleton has `aria-hidden="true"`; adding `aria-label`, `aria-labelledby`, or an explicit status role opts into one live region. The shimmer is disabled automatically when the user prefers reduced motion.

`--tinyrack-skeleton-fill`과 `--tinyrack-skeleton-highlight`는 theme-aware semantic token입니다. loading 표면에 컴포넌트 범위의 contrast 조정이 필요하면 `--tr-skeleton-fill` 또는 `--tr-skeleton-highlight`를 local override하세요.
