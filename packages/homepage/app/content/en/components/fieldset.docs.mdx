## Contract

`Root` supplies native fieldset grouping and disabled propagation. `Legend` names the group; each child control still needs its own visible label.

## Install

<ComponentInstall surfaces={[{ label: 'React', install: 'pnpm add @tinyrack/ui react react-dom', imports: ["import '@tinyrack/ui/core.css';", "import '@tinyrack/ui/components/fieldset.css';", "import '@tinyrack/ui/components/checkbox.css';", "import { TRFieldset } from '@tinyrack/ui/components/fieldset';", "import { TRCheckbox } from '@tinyrack/ui/components/checkbox';"] }]} />

## API

`Root` and `Legend` are exported individually and through `TRFieldset`; nested form controls retain their native submission behavior.
