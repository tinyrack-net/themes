## コントラクト

Navigation は再帰的な `group`、`page`、`link` データを受け取ります。`renderLink` は Base UI の render contract を通じてアクティブなルーターリンクを注入します。

## 例

アクティブまたは保留中のページを含むネストしたグループは自動的に開きます。

## API

`currentPath` と `pendingPath` は単純な値です。このコンポーネントは React Router を import したり、ルートを計算したりしません。
