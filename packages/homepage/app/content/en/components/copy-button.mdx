## Contract

The control reports `idle`, `copied`, or `unavailable` through `data-copy-status` and resets after two seconds by default.

## API

`value` is copied with Clipboard API and an offscreen selection fallback. Labels, `resetDelay`, status changes, and all normal TRButton appearance, variant, uiSize, loading, and event props are supported.
