# Tooltip

Brief, non-interactive descriptions for hoverable and focusable triggers.

## Contract

`Provider` manages delay groups; `Positioner` accepts side, alignment, offset, and collision props. Keep popup content concise.

## Install

<ComponentInstall surfaces={[{ label: 'React', install: 'pnpm add @tinyrack/ui react react-dom', imports: ["import '@tinyrack/ui/core.css';", "import '@tinyrack/ui/components/tooltip.css';", "import { Tooltip } from '@tinyrack/ui/components/tooltip';"] }]} />
