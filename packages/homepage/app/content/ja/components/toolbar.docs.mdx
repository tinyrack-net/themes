## コントラクト

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Playground controls cover `label`, `disabled`, `orientation`, and `loopFocus`.

## API

すべての compound part と prop/state type は `@tinyrack/ui/components/toolbar` から export されます。style は対応する CSS subpath による opt-in のままです。`Root` は loop 有効の horizontal arrow-key focus が既定で、vertical toolbar は Up/Down を使います。`Group` は関連 control に名前を付け、`Input` は通常の text-editing key を保ち、disabled group/item は activation を抑制します。horizontal toolbar では Input を最大 1 つだけ使って最後に置き、text-editing key が後続の toolbar item を妨げないようにします。focus 可能な disabled Button は利用不可であることが見えるようにし、すべての Toolbar と nested Group にアクセシブルな名前を付けます。
