## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Playground controls normalize invalid ranges and cover `label`, `min`, `max`, `value`, number formatting, and every explicit semantic variant.

## API

Keep `value` inside the current `min`/`max` range and use `format` for visible units plus `getAriaValueText` for contextual announcements. `neutral`, `info`, `success`, `warning`, and `danger` are explicit; Tinyrack intentionally adds no implicit low/high threshold API.
