## м„Өм№ҳ

```bash
pnpm add @tinyrack/ui @mdx-js/react react@^19 react-dom@^19 tailwindcss@^4
pnpm add -D @mdx-js/rollup remark-gfm
```

React provider importмҷҖ GFM pluginмқ„ мӮ¬мҡ©н•ҙ `.mdx` нҢҢмқјмқ„ м»ҙнҢҢмқјн•ҳм„ёмҡ”. Viteм—җм„ңлҠ” лӢӨмқҢкіј к°ҷмқҙ м„Өм •н•©лӢҲлӢӨ.

```ts
import mdx from '@mdx-js/rollup';
import { defineConfig } from 'vite';
import remarkGfm from 'remark-gfm';

export default defineConfig({
  plugins: [
    {
      enforce: 'pre',
      ...mdx({
        providerImportSource: '@mdx-js/react',
        remarkPlugins: [remarkGfm],
      }),
    },
  ],
});
```

## мӮ¬мҡ©лІ•

```tsx

import Content from './content.mdx';
import '@tinyrack/ui/core.css';
import '@tinyrack/ui/mdx.css';
import '@tinyrack/ui/components/code.css';
import '@tinyrack/ui/components/code-block.css';
import '@tinyrack/ui/components/link.css';
import '@tinyrack/ui/components/table.css';
import { tinyrackMdxComponents } from '@tinyrack/ui/mdx';

export function MdxArticle() {
  return (
    <MDXProvider components={tinyrackMdxComponents}>
      <Content />
    </MDXProvider>
  );
}
```

`Content`лҠ” `content.mdx`мқҳ м»ҙнҢҢмқјлҗң default exportмқҙл©° rendererм—җ м „лӢ¬н•ҳлҠ” HTML

`@mdx-js/react`мҷҖ TinyrackмқҖ runtime dependencyлЎң мң м§Җн•ҳм„ёмҡ”.

кіөмң  mapмқ„ ліҖкІҪн•ҳм§Җ м•Ҡкі  custom elementлЎң кё°ліёк°’мқ„ overrideн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

```tsx

import { createTinyrackMdxComponents } from '@tinyrack/ui/mdx';

function CustomHeading({ children, ...props }: ComponentPropsWithoutRef<'h1'>) {
  return <h1 {...props} className="product-heading">{children}</h1>;
}

const components = createTinyrackMdxComponents({
  components: { h1: CustomHeading },
});
```
