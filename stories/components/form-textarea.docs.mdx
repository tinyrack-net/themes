# Textarea

Native multiline input with the shared Tinyrack text control contract.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `textarea.tr-textarea` | `Textarea` | native textarea element | Use for multiline free-form input. |
| Size | `data-size` | `size` | `sm`, `md`, `lg`; default `md` | Shares the control scale with Input and Select. |
| State | `data-invalid`, `aria-invalid` | `invalid` | `true` or unset; default unset | Pair invalid state with an associated error message. |
| Native | `rows` | `rows` | positive number; browser default | Set an initial height without removing native resize behavior. |
| Native | `required`, `disabled`, `aria-describedby` | native textarea props | pass-through | Put semantics on the native textarea. |

## Usage

<ComponentExampleTabs
  id="form-textarea-basic"
  title="Basic textarea field"
  description="Compose multiline input with a visible label and associated format guidance."
  previewLayout="stretch"
  preview={(
    <Field>
      <Label htmlFor="maintenance-note">Maintenance note</Label>
      <Textarea
        aria-describedby="maintenance-note-help"
        id="maintenance-note"
        rows={4}
        placeholder="Window, owner, or alert routing."
      />
      <FormMessage id="maintenance-note-help">Include the window, owner, and rollback route.</FormMessage>
    </Field>
  )}
  sources={[
    {
      label: 'React',
      language: 'tsx',
      code: String.raw`import '@tinyrack/ui/core/core.css';
import '@tinyrack/ui/components/form/form.css';
import { Field, FormMessage, Label, Textarea } from '@tinyrack/ui/components/form/react';

<Field>
  <Label htmlFor="maintenance-note">Maintenance note</Label>
  <Textarea
    aria-describedby="maintenance-note-help"
    id="maintenance-note"
    rows={4}
    placeholder="Window, owner, or alert routing."
  />
  <FormMessage id="maintenance-note-help">Include the window, owner, and rollback route.</FormMessage>
</Field>;`,
    },
    {
      label: 'HTML',
      language: 'html',
      code: String.raw`<div class="tr-field" data-size="md">
  <label class="tr-label" for="maintenance-note">Maintenance note</label>
  <textarea
    aria-describedby="maintenance-note-help"
    class="tr-textarea"
    data-size="md"
    id="maintenance-note"
    rows="4"
    placeholder="Window, owner, or alert routing."
  ></textarea>
  <p class="tr-form-message" id="maintenance-note-help">
    Include the window, owner, and rollback route.
  </p>
</div>`,
    },
  ]}
/>

## Guidance

<ComponentGuidance
  whenToUse={[
    'Use Textarea for free-form multiline content such as notes, descriptions, and operational instructions.',
    'Choose rows for a useful initial height while allowing content and responsive layouts to grow.',
  ]}
  avoid={[
    'Do not use Textarea for a short single-line value or make placeholder text the only label.',
    'Do not signal an invalid value only with border color or remove native scrolling and resizing without a task-specific reason.',
  ]}
  accessibility={[
    'Associate a visible label with htmlFor/id and help or error text with aria-describedby.',
    'Set required, disabled, and invalid/aria-invalid on the native textarea so assistive technology receives the state.',
    'Preserve native focus, selection, scrolling, and keyboard editing behavior.',
  ]}
/>

## API

| Prop | Values | Note |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | Default `md`; sets `data-size`. |
| `invalid` | `boolean` | Sets `aria-invalid` and `data-invalid`. |
| `rows` | native number | Passed through to `<textarea>`. |
| native textarea props | `TextareaHTMLAttributes<HTMLTextAreaElement>` except native `size` | Passed through to `<textarea>`. |
