## コントラクト

`TRDrawer` は Base UI の controlled/uncontrolled state、swipe と snap-point gesture、focus 管理、keyboard dismissal、portal、React 19 ref と `render` prop を保ちます。motion は `swipeDirection` と reduced-motion 設定に従います。Playground は drawer event で `open` と `snapPoint` を同期し、Reset は閉じた full-height downward の既定値を復元します。

## 構造

`Root` は state を所有し、`Trigger` は drawer を開きます。`Portal` は overlay を body または指定 container に mount します。`Backdrop`、`Viewport`、`Popup`、scroll 可能な `Content` が modal surface を構成します。`Title` と `Description` は dialog label を提供し、`Close` は閉じます。`SwipeArea` は edge opening を提供します。`Provider`、`Indent`、`IndentBackground` は nested page effect を調整します。`VirtualKeyboardProvider` は mobile keyboard viewport change を処理します。`createHandle` は typed payload を使う detached trigger を接続します。

## 動作

- controlled visibility には `open` と `onOpenChange`、uncontrolled visibility には `defaultOpen` を使います。
- controlled position には `snapPoint` と `onSnapPointChange`、uncontrolled position には `defaultSnapPoint` を使い、許可値を `snapPoints` に指定します。
- `swipeDirection` は edge と dismissal 方向を決めます。`snapToSequentialPoints` は velocity による point skipping を防ぎます。
- Escape、`Close`、backdrop press、non-modal focus-out、swipe が既定の dismissal です。outside pointer dismissal を防ぐには `disablePointerDismissal` を使います。
- `modal` の既定値は `true` です。outside interaction には `false`、scroll lock なしで focus のみを閉じ込めるには `trap-focus` を使います。
- `Popup` の `initialFocus` と `finalFocus` で focus 移動を指定します。既定では focus は drawer に入り trigger に戻ります。
- `actionsRef` は imperative `close` と `unmount` を提供します。通常の flow では declarative state を推奨します。
- `Portal` は client で mount され、閉じた SSR markup は hydration-safe です。`Viewport` は visual viewport と safe area 内に surface を保ちます。

## API

すべての compound part と prop/state type は `@tinyrack/ui/components/drawer` から export されます。CSS は `@tinyrack/ui/components/drawer.css` で opt-in し、`--tr-drawer-popup-max-width`、`--tr-drawer-popup-background`、`--tr-drawer-backdrop`、`--tr-drawer-control-background`、`--tr-drawer-motion-duration`、`--tr-drawer-motion-easing` で appearance と motion を override できます。
