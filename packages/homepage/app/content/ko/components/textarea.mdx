## 핵심 속성

TRTextarea는 네이티브 ref, 이름, FormData, 초기화, 비활성, 읽기 전용, 필수, 잘못된 값 상태를 그대로 보존해요.

밀도 높은 화면에는 `uiSize="sm"`을, 읽고 편집할 공간이 더 필요하면 `uiSize="lg"`를 사용하세요. 내용이 표시 영역을 넘으면 네이티브 스크롤이 동작하고, 가로로 레이아웃이 넘치지 않도록 크기는 세로로만 조절할 수 있어요.

import { TRField } from '@tinyrack/ui/components/field';
import { TRForm } from '@tinyrack/ui/components/form';
import { useState } from 'react';

function ChangeReasonForm() {
  const [attempted, setAttempted] = useState(false);
  const [submitted, setSubmitted] = useState(false);
  const [value, setValue] = useState('');
  const invalid = attempted && value.trim().length === 0;

  return (
    <TRForm noValidate onSubmit={(event) => {
      event.preventDefault();
      setAttempted(true);
      setSubmitted(value.trim().length > 0);
      if (value.trim().length === 0) {
        const reason = event.currentTarget.elements.namedItem('reason');
        if (reason instanceof HTMLTextAreaElement) reason.focus();
      }
    }}>
      <TRField.Root invalid={invalid}>
        <TRField.Label htmlFor="reason">변경 사유</TRField.Label>
        <TRTextarea aria-invalid={invalid} id="reason" name="reason" onChange={(event) => {
          setValue(event.currentTarget.value);
          setSubmitted(false);
        }} required value={value} />
        <TRField.Error match>{invalid ? '제출하기 전에 사유를 입력하세요.' : null}</TRField.Error>
      </TRField.Root>
      <TRButton type="submit">변경 사항 제출</TRButton>
      <output aria-live="polite">{submitted ? '변경 사항을 제출했어요.' : ''}</output>
    </TRForm>
  );
}` ,
      styles: { code: "@import '@tinyrack/ui/components/button.css';\n@import '@tinyrack/ui/components/field.css';\n@import '@tinyrack/ui/components/form.css';" },

## API

`TRTextarea`는 네이티브 텍스트 영역 속성, ref, 이벤트, 클래스, 스타일을 모두 전달해요. 크기 조절은 기본적으로 세로 방향이고 네이티브 유효성 검사를 그대로 따라요.

| 속성 | 타입 | 기본값 | 설명 |
| --- | --- | --- | --- |
| `uiSize` | `'sm' \| 'md' \| 'lg'` | `'md'` | 해당 컨트롤의 타이포그래피, 안쪽 여백, 최소 블록 크기를 적용해요. |
| 네이티브 속성 | `ComponentPropsWithRef<'textarea'>` | 해당 없음 | `value`, `defaultValue`, `rows`, `name`, 유효성 검사 속성, 이벤트, 스타일, `ref`를 포함해요. |
