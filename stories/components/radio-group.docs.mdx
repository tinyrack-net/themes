# Radio Group

Provides one-of-many selection, arrow-key navigation, and a single native form value.

## Contract

| Axis | Behavior |
| --- | --- |
| value | uncontrolled with `defaultValue`, controlled with `value` and `onValueChange` |
| keyboard | arrow keys move selection; Tab enters and leaves the group once |
| form | `name` and `required` apply to the group; individual radios provide values |

  return (
    <Form noValidate onSubmit={(event) => {
      event.preventDefault();
      event.currentTarget.checkValidity();
      setAttempted(true);
    }}>
      <Field.Root invalid={invalid}>
        <Field.Label>Deployment cadence</Field.Label>
        <RadioGroup name="cadence" onValueChange={setValue} required value={value}>
          <label><Radio.Root value="daily"><Radio.Indicator /></Radio.Root> Daily</label>
          <label><Radio.Root value="weekly"><Radio.Indicator /></Radio.Root> Weekly</label>
        </RadioGroup>
        <Field.Error match>Choose a cadence to continue.</Field.Error>
      </Field.Root>

    </Form>
  );
}` }]}

## API

`RadioGroup` and its prop and state types are exported from `@tinyrack/ui/components/radio-group`; compose it with `Radio.Root` and `Radio.Indicator`.
