## コントラクト

| 軸 | 値 | 既定値 |
| --- | --- | --- |
| `variant` | `default`, `muted`, `danger` | `default` |
| `underline` | `always`, `hover`, `none` | `hover` |
| `disabled` | blocks pointer and keyboard navigation | `false` |

## API

ナビゲーションには `Link`、コマンドには `Button` を使います。`target`、`rel`、`download`、ネイティブ anchor props、`ref` は変更せず転送されます。Base UI の `render` contract は nested anchor なしで React Router と組み合わせられます: `<Link render={<RouterLink to="/racks" />}>Racks</Link>`.
