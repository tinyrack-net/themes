## 계약

표면에는 `Root/Trigger/Portal/Positioner/Popup`을 사용하고 내부에 `Item` 또는 `Separator`를 조합하세요. nested menu는 자체 `Portal/Positioner/Popup` 표면을 `SubmenuRoot`와 `SubmenuTrigger`로 감쌉니다.

## API

Base UI가 roving focus, typeahead, keyboard activation, portal 위치 지정, outside/Escape 닫기를 담당합니다. 하나의 popup이 여러 trigger 사이에서 콘텐츠를 전환할 때만 `Viewport`가 필요하며, 여기서는 전체 public anatomy를 복사할 수 있도록 보여 줍니다.
