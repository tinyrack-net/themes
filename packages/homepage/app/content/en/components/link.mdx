## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `variant` | `default`, `muted`, `danger` | `default` |
| `underline` | `always`, `hover`, `none` | `hover` |
| `disabled` | blocks pointer and keyboard navigation | `false` |

## API

Use `TRLink` for navigation and `TRButton` for commands; `target`, `rel`, `download`, native anchor props, and `ref` are forwarded unchanged. Its Base UI `render` contract composes React Router without nested anchors: `<TRLink render={<RouterLink to="/racks" />}>Racks</TRLink>`.
