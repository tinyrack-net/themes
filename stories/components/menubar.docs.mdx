# Menubar

Coordinates application menus with desktop-style arrow-key navigation.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Storybook controls cover `disabled`, `orientation`, `loopFocus`, and the accessible `label`.

<ComponentExampleTabs

  title="Three-menu application bar"
  description="File, Edit, and View are separate top-level menus. Arrow keys move between them; Enter or Space opens the focused menu; Escape closes it and restores focus."

    ['View', ['Show status', 'Keyboard shortcuts']],
  ].map(([label, items]) => (
    <Menu.Root key={label}>
      <Menu.Trigger>{label}</Menu.Trigger>
      <Menu.Portal>
        <Menu.Positioner>
          <Menu.Popup>
            {items.map((item) => <Menu.Item key={item}>{item}</Menu.Item>)}
          </Menu.Popup>
        </Menu.Positioner>
      </Menu.Portal>
    </Menu.Root>
  ))}
</Menubar>` }]}
/>
