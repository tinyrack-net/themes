## API

`TRBadge`는 `span`을 렌더링하고 네이티브 속성, 이벤트, 스타일, React 19 `ref`를 전달합니다. `TRBadgeProps`, `TRBadgeUiSize`, `TRBadgeVariant` 타입도 내보냅니다. 상태 텍스트를 표시하고, 아이콘만 사용할 때는 애플리케이션이 접근 가능한 레이블을 제공해야 합니다.
