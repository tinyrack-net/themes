## 계약

Use TRNavigationMenu for a site header with a brand, a small number of direct destinations, and optional Product or Resources flyouts. The desktop composition keeps Base UI roving keyboard navigation; the mobile composition exposes the same destinations in a TRDrawer.

## API

모든 compound part와 prop/state type은 `@tinyrack/ui/components/navigation-menu`에서 export됩니다. style은 대응 CSS subpath를 통한 opt-in으로 유지됩니다.

TRNavigationMenu는 충분한 가로 공간이 있는 pointer·keyboard 사이트 탐색을 위한 것입니다. flyout group을 task 중심으로 유지하고 preview가 필요 없는 목적지에는 direct link를 보존하세요. 좁은 breakpoint에서는 horizontal keyboard model을 압축하지 말고 같은 destination set을 가진 TRDrawer 조합으로 바꾸세요.
