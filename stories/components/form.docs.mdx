# Form

Accessible field composition for labels, controls, descriptions, and validation errors.

## Contract

| Part | Responsibility |
| --- | --- |
| `Field` | disabled, invalid, and validation state |
| `Label` | accessible control name |
| `Control` | native input behavior through Base UI render |
| `Description` | persistent supporting text |
| `Error` | validation-specific feedback |
