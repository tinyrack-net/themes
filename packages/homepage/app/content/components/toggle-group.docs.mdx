# ToggleGroup

Coordinates mutually exclusive or multiple toggle buttons. Like Toggle, it does not submit a form value by itself.

## Contract

| Axis | Values | Default |
| --- | --- | --- |
| selection | single or `multiple` | single |
| state | controlled `value` array or uncontrolled `defaultValue` | `[]` |
| orientation | horizontal or vertical | horizontal |
| keyboard | looping or bounded arrow focus | looping |
| availability | group or item disabled | enabled |

## API

ToggleGroup preserves Base UI state, arrow-key focus, orientation, refs, native props, events, and `render`. Supply stable item `value`s, keep controlled and uncontrolled selection as string arrays, and use Left/Right arrows horizontally or Up/Down vertically; `loopFocus={false}` stops at the group edges. Group `disabled` applies to every item, while an individual Toggle can be disabled independently. Give the group an accessible name. Use CheckboxGroup or RadioGroup when the selection must be a named form field.
