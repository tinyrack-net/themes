## 계약

`Root`, `Header`, `Brand`, `Actions`, `Sidebar`, `Main`, `Outline`이 반응형 shell을 구성합니다. `docs`와 `splash` layout에서 `Header`는 sidebar와 content 위의 viewport 전체를 차지합니다. router 상태는 props로만 전달됩니다.

## 예시

전역 header를 유지하는 제품 landing page에는 `splash`를, navigation chrome 없는 내장 API reference에는 `standalone`을 사용하세요.

## API

`layout`은 `docs`, `splash`, `standalone`을 지원하며 모바일 disclosure와 scroll restoration은 UI 내부에 남습니다.
