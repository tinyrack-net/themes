## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Playground controls cover the visually meaningful `label` and `disabled` props; open state remains interactive in the preview.

## API

Every compound part and prop/state type is exported from `@tinyrack/ui/components/alert-dialog`; styles remain opt-in through the matching CSS subpath. Group confirmation controls in `tr-alert-dialog-actions`; they wrap into full readable actions on narrow viewports. Use a secondary TRButton for Cancel and a danger TRButton for the destructive confirmation.
