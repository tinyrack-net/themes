# Checkbox

Boolean native input with a custom visual control and preserved input semantics.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `label.tr-checkbox` | `Checkbox` | native checkbox input plus optional label | Keep the whole visible label clickable. |
| Part | `tr-checkbox-input` | native input props | `type="checkbox"`; required part | The native input remains focusable and operable. |
| Part | `tr-checkbox-control` | generated control | decorative span; required part | Mark custom visual control `aria-hidden="true"`. |
| Part | `tr-checkbox-label` | `children` | `ReactNode`; omitted by default | Use a clear affirmative label. |
| Size | `data-size` | `size` | `sm`, `md`, `lg`; default `md` | Shares the choice-control scale with Radio and Switch. |
| State | `data-invalid`, `aria-invalid` | `invalid` | `true` or unset; default unset | Pair invalid state with an associated error message. |
| Native | `checked`, `required`, `disabled` | native checkbox props | pass-through | Preserve native state and Space-key behavior. |

## Guidance

<ComponentGuidance
  whenToUse={[
    'Use Checkbox for independent yes/no selections or for choosing any number of options from a group.',
    'Use an affirmative visible label that describes what the checked state means.',
  ]}
  avoid={[
    'Do not use Checkbox for mutually exclusive choices; use a radio group instead.',
    'Do not make the decorative control focusable or communicate invalid state only with color.',
  ]}
  accessibility={[
    'Keep the native checkbox in the tab order and preserve Space-key toggling; the visual control remains aria-hidden.',
    'Set required, disabled, and aria-invalid on the native input and associate any help or error text with aria-describedby.',
    'Wrap related checkbox sets in a fieldset with a legend so the group has an accessible name.',
  ]}
/>

## API

| Prop | Values | Note |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | Default `md`; sets `data-size`. |
| `invalid` | `boolean` | Sets `aria-invalid` on input and `data-invalid` on wrapper. |
| `children` | `ReactNode` | Renders `tr-checkbox-label` when present. |
| native checkbox props | `InputHTMLAttributes<HTMLInputElement>` except `children`, `size`, `type` | Passed through to the native checkbox input. |
