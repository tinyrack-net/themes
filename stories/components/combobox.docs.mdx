# Combobox

Filters a collection through a text input and commits one semantic item value from the popup.

## Contract

| Axis | Behavior |
| --- | --- |
| state | input text, selected value, and open state are separate controllable axes |
| input anatomy | Label → InputGroup → Input + Clear + Trigger + Icon |
| popup anatomy | Portal → Positioner → Popup → Status + List; groups, indicators, separators, and Empty cover collection feedback |

  const [attempted, setAttempted] = useState(false);
  const [query, setQuery] = useState('');
  const [selected, setSelected] = useState<string | null>(null);
  const invalid = attempted && selected === null;

  return (
    <Form noValidate onSubmit={(event) => { event.preventDefault(); event.currentTarget.checkValidity(); setAttempted(true); }}>
      <Field.Root invalid={invalid}>
        <Combobox.Root inputValue={query} items={['Rack A', 'Rack B']} name="rack" onInputValueChange={setQuery} onValueChange={setSelected} required value={selected}>
          <Combobox.Label render={<label htmlFor="required-rack">Deployment rack</label>} />
          <Combobox.InputGroup><Combobox.Input id="required-rack" placeholder="Choose a rack" /><Combobox.Clear aria-label="Clear">×</Combobox.Clear><Combobox.Trigger aria-label="Show racks"><Combobox.Icon>⌄</Combobox.Icon></Combobox.Trigger></Combobox.InputGroup>
          <Combobox.Portal><Combobox.Positioner><Combobox.Popup><Combobox.Status /><Combobox.List>
            <Combobox.Item value="Rack A">Rack A<Combobox.ItemIndicator>✓</Combobox.ItemIndicator></Combobox.Item>
            <Combobox.Item value="Rack B">Rack B<Combobox.ItemIndicator>✓</Combobox.ItemIndicator></Combobox.Item>
            <Combobox.Empty>No matching racks</Combobox.Empty>
          </Combobox.List></Combobox.Popup></Combobox.Positioner></Combobox.Portal>
        </Combobox.Root>
        <Field.Error match>Choose a rack from the list.</Field.Error>
      </Field.Root>
      <Button type="submit">Deploy</Button>
    </Form>
  );
}` }]}

## API

The complete label, input, popup, collection, group, item-indicator, status, empty-state, and arrow anatomy is exported from `@tinyrack/ui/components/combobox` with matching prop and state types.
