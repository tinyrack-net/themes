# Tabs

Keyboard-navigable views with three densities and horizontal or vertical orientation.

## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | `md` |
| `orientation` | `horizontal`, `vertical` | `horizontal` |
| state | controlled `value` or uncontrolled `defaultValue` | required selection |
