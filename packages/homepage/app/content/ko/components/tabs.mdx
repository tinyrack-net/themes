| 축 | 값 | 기본값 |
| --- | --- | --- |
| `uiSize` | `sm`, `md`, `lg` | `md` |
| `orientation` | `horizontal`, `vertical` | `horizontal` |
| disabled tab | `TRTabs.Tab disabled` | `false` |
| activation | manual or `activateOnFocus` | manual |
| focus | looping or bounded | looping |
| state | controlled `value`, including `null`, or uncontrolled `defaultValue` | first enabled tab |

## 구조

- `Root`는 제어형 또는 비제어형 선택 상태, 방향, `uiSize`를 관리합니다.
- `List`는 접근 가능한 이름이 있는 tablist이며 활성화 방식과 focus 순환을 설정합니다.
- `Tab`은 `value`로 view를 식별합니다. disabled tab은 focus할 수 있지만 활성화할 수 없습니다.
- `Indicator`는 선택 사항이며 overflow strip에서도 active tab을 따라갑니다.
- `Panel`은 같은 `value`를 가진 tab과 접근 가능한 콘텐츠를 연결합니다.

## API

| 파트 | 주요 props | 설명 |
| --- | --- | --- |
| `Root` | `value`, `defaultValue`, `onValueChange`, `orientation`, `uiSize` | active tab이 없게 하려면 `value={null}`을 사용합니다. 값을 지정하지 않으면 첫 enabled tab이 선택됩니다. |
| `List` | `activateOnFocus`, `loopFocus` | 수동 활성화와 focus 순환이 기본값입니다. |
| `Tab` | `value`, `disabled`, `render` | 화살표 키는 focus를 이동하고 수동 모드에서는 Enter 또는 Space로 활성화합니다. |
| `Indicator` | `renderBeforeHydration`, `render` | Base UI geometry 변수로 움직이는 선택적 시각 marker입니다. |
| `Panel` | `value`, `keepMounted`, `hiddenUntilFound`, `render` | tab value와 일치하며 ARIA 관계를 유지합니다. |

Base UI가 role, focus 이동, selection, hydration-safe 상태를 제공합니다. Tinyrack foundation style 뒤에 `tabs.css`를 import하세요. `--tr-tabs-indicator-background`, `--tr-tabs-indicator-size`, `--tr-tabs-radius`, `--tr-tabs-tab-height`, `--tr-tabs-panel-padding`으로 스타일을 조정할 수 있습니다.
