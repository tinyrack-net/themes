## Contract

`TRFileTree` accepts a Markdown unordered list as its children. Nested lists become directories, while list items ending in `/` become empty directories. Directories use the platform `details` disclosure.

## Examples

Nest another Markdown list to create a directory. Use `...` or `…` for a placeholder, and use inline Markdown such as `**bold**` or `` `code` `` for highlighted entries.

## API

`TRFileTree` forwards native `ul` props and React 19 refs. Directories are open by default and can be toggled individually.
