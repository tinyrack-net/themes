## 계약

| 축 | 값 | 기본값 |
| --- | --- | --- |
| `variant` | `default`, `muted`, `danger` | `default` |
| `underline` | `always`, `hover`, `none` | `hover` |
| `disabled` | blocks pointer and keyboard navigation | `false` |

## API

탐색에는 `TRLink`, 명령에는 `TRButton`을 사용하세요. `target`, `rel`, `download`, native anchor props, `ref`는 변경 없이 전달됩니다. Base UI `render` contract는 nested anchor 없이 React Router를 조합합니다: `<TRLink render={<RouterLink to="/racks" />}>Racks</TRLink>`.
