## コントラクト

| 軸 | 値 | 既定値 |
| --- | --- | --- |
| `variant` | `neutral`, `info`, `success`, `warning`, `danger` | `neutral` |
| `size` | `sm`, `md`, `lg` | `md` |

## API

`Badge` はネイティブ `span` props、events、styles、React 19 `ref` を転送します。ステータステキストを表示し、アイコンだけを使う場合はアプリケーションが提供するアクセシブルなラベルが必要です。
