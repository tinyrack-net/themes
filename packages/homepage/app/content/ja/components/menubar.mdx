## コントラクト

`TRMenubar` は単一の menubar root です。`TRMenu.Root` の子を構成し、Base UI の keyboard 動作、React 19 ref、ネイティブ prop と event、`render` contract を保ちます。`orientation` は水平の Left/Right または垂直の Up/Down roving focus を選択し、`loopFocus` は末端での循環を制御し、`disabled` はすべての子 menu を無効にします。

## API

`@tinyrack/ui/components/menubar` は `TRMenubar` root と `TRMenubarProps`、`TRMenubarState` を export します。`TRMenu.Root`、`TRMenu.Trigger`、`TRMenu.Portal`、`TRMenu.Positioner`、`TRMenu.Popup` などの menu anatomy は `@tinyrack/ui/components/menu` から別に export されます。menubar CSS subpath には、組み合わせた menu part に必要な style が含まれます。
