## Contract

`Root`, `Header`, `Brand`, `Actions`, `Sidebar`, `Main`, `Outline`이 반응형 shell을 구성합니다. `docs`는 desktop navigation과 focus가 관리되는 mobile drawer를 제공하고, `splash`는 global header만 유지하며, `standalone`은 두 navigation 영역을 모두 제거합니다. Router 상태는 props로만 전달됩니다.

## API

`Root`에는 `currentPath`와 `locationKey`가 필요합니다. Shell 내부 heading scroll에는 `hash`, scroll 복원에는 `POP`, `PUSH`, `REPLACE` 중 하나인 `navigationKind`, pending progress와 `aria-busy` 표시에는 `pendingPath`를 전달합니다. `layout`은 `docs`, `splash`, `standalone` 중에서 선택합니다.

`Header`, `Brand`, `Actions`, `Sidebar`, `Outline`은 native props, class name, style, ref를 보존합니다. `Main`은 inner content용 `contentClassName`과 scroll region의 accessible name용 `viewportLabel`도 받습니다.

제한된 preview나 application 영역에 shell을 넣을 때는 `--tr-docs-shell-block-size`를 설정하세요. 기본값은 `100dvh`입니다.
