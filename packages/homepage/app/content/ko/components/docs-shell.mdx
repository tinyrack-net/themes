## 핵심 속성

`Root`, `Header`, `Brand`, `Actions`, `Sidebar`, `Main`, `Outline`이 반응형 셸을 구성해요. `docs`는 데스크톱 탐색과 포커스를 관리하는 모바일 드로어를 제공하고, `splash`는 전역 헤더만 남기며, `standalone`은 두 탐색 영역을 모두 없애요. 라우터 상태는 props로만 전달돼요.

## API

`Root`에는 `currentPath`와 `locationKey`가 필요해요. 셸 안쪽 제목으로 스크롤할 때는 `hash`, 스크롤을 복원할 때는 `POP`, `PUSH`, `REPLACE` 중 하나인 `navigationKind`, 대기 진행 표시와 `aria-busy`에는 `pendingPath`를 전달해요. `layout`은 `docs`, `splash`, `standalone` 중에서 골라요.

`Header`, `Brand`, `Actions`, `Sidebar`, `Outline`은 네이티브 props, 클래스 이름, 스타일, ref를 보존해요. `Main`은 안쪽 콘텐츠용 `contentClassName`과 스크롤 영역의 접근 가능한 이름용 `viewportLabel`도 받아요.

좁은 미리 보기나 애플리케이션 영역에 셸을 넣을 때는 `--tr-docs-shell-block-size`를 설정해요. 기본값은 `100dvh`예요.
