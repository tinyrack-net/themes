## 계약

이 컴포넌트는 storage를 읽거나 document를 변경하지 않습니다. 애플리케이션 어댑터에서 `value`를 전달하고 `onValueChange`를 저장하세요.

## 예시

선택한 scheme을 애플리케이션 어댑터에 저장하고 `value`를 통해 다시 전달하세요.

## API

`value`는 `light | dark`입니다. label, size, disabled 상태, class, button ref를 설정할 수 있습니다.
