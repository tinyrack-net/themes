## 계약

| 축 | 값 | 기본값 |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | `md` |
| `orientation` | `horizontal`, `vertical` | `horizontal` |
| activation | manual or `activateOnFocus` | manual |
| focus | looping or bounded | looping |
| state | controlled `value`, including `null`, or uncontrolled `defaultValue` | first enabled tab |

## API

Base UI는 role, focus 이동, selection, 제어형 상태, activation 동작을 제공합니다. 기본은 수동 activation이며 panel 변경이 즉각적이고 비용이 낮을 때만 `activateOnFocus`를 사용하세요. 기본 문서 조합은 선택된 tab border만 사용합니다. 별도 active marker가 필요한 명시적 브랜드 조합을 위해 `Indicator`는 계속 export됩니다.
