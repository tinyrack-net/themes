## コントラクト

| 軸 | 値 | 既定値 |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | `md` |
| `orientation` | `horizontal`, `vertical` | `horizontal` |
| activation | manual or `activateOnFocus` | manual |
| focus | looping or bounded | looping |
| state | controlled `value`, including `null`, or uncontrolled `defaultValue` | first enabled tab |

## API

Base UI は role、focus の移動、selection、制御状態、activation 動作を提供します。既定は手動 activation で、panel の変更が即時かつ低コストの場合だけ `activateOnFocus` を使います。既定のドキュメント構成は選択した tab の border だけを使います。別の active marker が必要な明示的なブランド構成向けに `Indicator` は引き続き export されます。
