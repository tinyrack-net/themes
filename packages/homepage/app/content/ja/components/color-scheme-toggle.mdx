## 主なプロパティ

`TRColorSchemeToggle` は制御付きの 2 状態ボタンです。アイコンとアクセシブル名は次の操作を示し、`light` では `dark`、`dark` では `light` を提示します。コンポーネントは `localStorage` を読み取らず、`system` 設定を照会せず、`document` を変更しません。これらはアプリケーションアダプターで解決して保存します。

## API

| プロパティ | 型 | 既定値 | 説明 |
| --- | --- | --- | --- |
| `value` | `"light" \| "dark"` | 必須 | 現在解決済みの配色です。 |
| `onValueChange` | `(value: TRColorScheme) => void` | 必須 | キャンセルされていない操作後に反対の配色を通知します。 |
| `darkLabel` | `string` | `"Use dark color scheme"` | 次の値が `dark` のときのアクセシブルな操作名です。 |
| `lightLabel` | `string` | `"Use light color scheme"` | 次の値が `light` のときのアクセシブルな操作名です。 |
| `aria-labelledby` | `string` | — | 生成されるラベルを外部のアクセシブル名で置き換えます。 |
| `disabled` | `boolean` | `false` | フォーカスと操作を無効にします。 |
| `uiSize` | `"sm" \| "md" \| "lg"` | `"md"` | 継承された IconButton のサイズです。 |
| `ref` | `Ref<HTMLButtonElement>` | — | ボタンに転送される `ref` です。 |
| `onClick` | `MouseEventHandler<HTMLButtonElement>` | — | `onValueChange` より先に実行され、`preventDefault()` で値変更をキャンセルします。 |

その他の安全な `TRIconButton` のボタン属性、ネイティブイベント、`className`、`style`、カスタマイズ用トークンを継承します。見た目の `appearance` は `ghost` で、コントロールはネイティブボタンのままです。
