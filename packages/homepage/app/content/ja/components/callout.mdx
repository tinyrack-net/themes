## コントラクト

`note`、`tip`、`caution`、`danger` は、記述した MDX をアプリケーションフレームワークに結合せず、対応する TRAlert の意味にマッピングします。説明には段落、リスト、コードブロックなどの MDX ブロックコンテンツを使用でき、不正な HTML の入れ子も発生しません。

## 例

既定の英語見出しが適切でない場合は、ローカライズした `title` を使用します。

## API

`TRCallout` はネイティブの `div` props と React 19 の `ref` を転送します。`title` で既定の見出しをローカライズまたは置き換えできます。記述した children にはインラインの React コンテンツまたはブロックレベルの MDX を指定できます。
