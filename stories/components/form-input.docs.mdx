# Input

Native input wrapper for text-like, numeric, date, and time controls.

## Contract

| Surface | API | Values | Default | Rule |
| --- | --- | --- | --- | --- |
| CSS class | `tr-input` | input element | required | Use on native input elements. |
| Size attribute | `data-size` | `sm`, `md`, `lg` | `md` | Shared with `Textarea` and `Select`. |
| Invalid attribute | `data-invalid` | `true` | unset | Mirrors `aria-invalid` when `invalid` is set. |
| React prop | `type` | native input type | browser default | Use native browser behavior. |
| React prop | `invalid` | `boolean` | false | Sets `aria-invalid` and `data-invalid`. |

## Install

| Surface | Install | Import |
| --- | --- | --- |
| CSS / HTML | `pnpm add @tinyrack/ui tailwindcss` | `@tinyrack/ui/core/core.css` and `@tinyrack/ui/components/form/form.css` |
| React | `pnpm add @tinyrack/ui react react-dom` | `@tinyrack/ui/components/form/react` |

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/form/form.css";
```

## API

| Prop | Values | Note |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | Default `md`; sets `data-size`. |
| `invalid` | `boolean` | Sets `aria-invalid` and `data-invalid`. |
| `type` | native input type | Passed through to `<input>`. |
| native input props | `InputHTMLAttributes<HTMLInputElement>` except native `size` | Passed through to `<input>`. |
