# Skeleton

Layout-preserving loading placeholders with text, rectangle, and circle shapes.

## Contract

`shape` accepts `text`, `rectangle`, or `circle` and defaults to `text`. Dimensions come from layout or `style`.

## Install

<ComponentInstall surfaces={[{ label: 'React', install: 'pnpm add @tinyrack/ui react react-dom', imports: ["import '@tinyrack/ui/core.css';", "import '@tinyrack/ui/components/skeleton.css';", "import { Skeleton } from '@tinyrack/ui/components/skeleton';"] }]} />

## API

Provide `aria-label` only when the loading region needs an announced status; otherwise the visual placeholder remains unobtrusive.
