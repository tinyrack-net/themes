## Contract

`Root`、`Header`、`Brand`、`Actions`、`Sidebar`、`Main`、`Outline` でレスポンシブな shell を構成します。`docs` は desktop navigation と focus が管理された mobile drawer を提供し、`splash` は global header だけを残し、`standalone` は両方の navigation 領域を取り除きます。Router 状態は props だけで渡されます。

## API

`Root` には `currentPath` と `locationKey` が必要です。Shell 内の heading scroll には `hash`、scroll 復元には `POP`、`PUSH`、`REPLACE` のいずれかの `navigationKind`、pending progress と `aria-busy` の表示には `pendingPath` を渡します。`layout` は `docs`、`splash`、`standalone` から選びます。

`Header`、`Brand`、`Actions`、`Sidebar`、`Outline` は native props、class name、style、ref を保持します。`Main` は inner content 用の `contentClassName` と scroll region の accessible name 用の `viewportLabel` も受け取ります。

制限された preview や application 領域に shell を埋め込む場合は `--tr-docs-shell-block-size` を設定してください。既定値は `100dvh` です。
