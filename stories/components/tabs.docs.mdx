# Tabs

Accessible tabbed navigation for compact operational panels.

## Contract

| Surface | API | Values | Default | Rule |
| --- | --- | --- | --- | --- |
| Root class | `tr-tabs` | `data-size`, `data-orientation` | `md`, `horizontal` | Provides context for list, triggers, and panels. |
| List class | `tr-tabs-list` | `role="tablist"` | required | Receives `aria-orientation` for vertical tabs. |
| Trigger class | `tr-tabs-trigger` | `role="tab"`, `data-active` | required | Renders native buttons. |
| Panel class | `tr-tabs-panel` | `role="tabpanel"`, `data-active` | required | Hidden when inactive. |
| Activation | `activationMode` | `automatic`, `manual` | `automatic` | Use manual when panels are expensive to show. |
| Selection | `value`, `defaultValue` | `string` | none | Use exactly one selected-value mode. |

## Install

| Surface | Install | Import |
| --- | --- | --- |
| CSS / HTML | `pnpm add @tinyrack/ui tailwindcss` | `@tinyrack/ui/core/core.css` and `@tinyrack/ui/components/tabs/tabs.css` |
| React | `pnpm add @tinyrack/ui react react-dom` | `@tinyrack/ui/components/tabs/react` |

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/tabs/tabs.css";
```

## Usage

<ComponentExampleTabs
  title="Basic tabs"
  description="Use Tabs for compact panels that switch between related operational views."
  preview={(
    <Tabs defaultValue="overview" activationMode="automatic">
      <TabsList aria-label="Rack sections">
        <TabsTrigger value="overview">Overview</TabsTrigger>
        <TabsTrigger value="logs">Logs</TabsTrigger>
      </TabsList>
      <TabsPanel value="overview">Rack A is healthy.</TabsPanel>
      <TabsPanel value="logs">03:18 backup completed.</TabsPanel>
    </Tabs>
  )}
  sources={[
    {
      label: 'React',
      language: 'tsx',
      code: String.raw`import '@tinyrack/ui/core/core.css';
import '@tinyrack/ui/components/tabs/tabs.css';
import { Tabs, TabsList, TabsPanel, TabsTrigger } from '@tinyrack/ui/components/tabs/react';

<Tabs defaultValue="overview" activationMode="automatic">
  <TabsList aria-label="Rack sections">
    <TabsTrigger value="overview">Overview</TabsTrigger>
    <TabsTrigger value="logs">Logs</TabsTrigger>
  </TabsList>
  <TabsPanel value="overview">Rack A is healthy.</TabsPanel>
  <TabsPanel value="logs">03:18 backup completed.</TabsPanel>
</Tabs>;`,
    },
    {
      label: 'HTML',
      language: 'html',
      code: String.raw`<div class="tr-tabs" data-size="md" data-orientation="horizontal">
  <div class="tr-tabs-list" role="tablist" aria-label="Rack sections">
    <button
      class="tr-tabs-trigger"
      role="tab"
      aria-selected="true"
      data-active="true"
    >
      Overview
    </button>
  </div>
  <div class="tr-tabs-panel" role="tabpanel" data-active="true">
    Rack A is healthy.
  </div>
</div>`,
    },
  ]}
/>

## API

| Prop | Values | Note |
| --- | --- | --- |
| `value` | `string` | Controlled selected tab. |
| `defaultValue` | `string` | Initial selected tab for uncontrolled usage. |
| `onValueChange` | `(value) => void` | Called after click or keyboard activation. |
| `size` | `sm`, `md`, `lg` | Default `md`; sets `data-size`. |
| `orientation` | `horizontal`, `vertical` | Default `horizontal`; sets `data-orientation`. |
| `activationMode` | `automatic`, `manual` | Default `automatic`. |
| native props | div and button attributes | Passed through to root, list, trigger, and panel primitives. |
