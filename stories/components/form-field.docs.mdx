# Field

Composition primitives for labels, native controls, and supporting messages.

## Contract

| Surface | API | Values | Default | Rule |
| --- | --- | --- | --- | --- |
| Field class | `tr-field` | `data-size` | `md` | Sets vertical rhythm for a field group. |
| Label class | `tr-label` | native label props | none | Callers provide `htmlFor` or wrap the control. |
| Message class | `tr-form-message` | `data-variant` | `neutral` | Use `error` for validation support text. |
| React props | `Field.size` | `sm`, `md`, `lg` | `md` | Match the nested control size. |
| React props | `FormMessage.variant` | `neutral`, `error` | `neutral` | Mirrors `data-variant`. |

## Install

| Surface | Install | Import |
| --- | --- | --- |
| CSS / HTML | `pnpm add @tinyrack/ui tailwindcss` | `@tinyrack/ui/core/core.css` and `@tinyrack/ui/components/form/form.css` |
| React | `pnpm add @tinyrack/ui react react-dom` | `@tinyrack/ui/components/form/react` |

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/form/form.css";
```

## Usage

<ComponentExampleTabs
  title="Basic field"
  description="A field groups the label, native control, and supporting message rhythm."
  preview={(
    <Field size="md">
      <Label htmlFor="rack-name">Rack name</Label>
      <Input id="rack-name" placeholder="rack-a-01" />
      <FormMessage>Use a stable rack identifier.</FormMessage>
    </Field>
  )}
  sources={[
    {
      label: 'React',
      language: 'tsx',
      code: String.raw`import '@tinyrack/ui/core/core.css';
import '@tinyrack/ui/components/form/form.css';
import { Field, FormMessage, Input, Label } from '@tinyrack/ui/components/form/react';

<Field size="md">
  <Label htmlFor="rack-name">Rack name</Label>
  <Input id="rack-name" placeholder="rack-a-01" />
  <FormMessage>Use a stable rack identifier.</FormMessage>
</Field>;`,
    },
    {
      label: 'HTML',
      language: 'html',
      code: String.raw`<div class="tr-field" data-size="md">
  <label class="tr-label" for="rack-name">Rack name</label>
  <input class="tr-input" data-size="md" id="rack-name" />
  <p class="tr-form-message" data-variant="neutral">
    Use a stable rack identifier.
  </p>
</div>`,
    },
  ]}
/>

## API

| Prop | Values | Note |
| --- | --- | --- |
| `Field.size` | `sm`, `md`, `lg` | Default `md`; sets `data-size`. |
| `Label.htmlFor` | `string` | Native label association. |
| `FormMessage.variant` | `neutral`, `error` | Default `neutral`; sets `data-variant`. |
| native props | div, label, paragraph attributes | Passed through to each primitive. |
