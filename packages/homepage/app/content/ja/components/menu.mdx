## コントラクト

surface には `Root/Trigger/Portal/Positioner/Popup` を使い、内部に `Item` または `Separator` を構成します。nested menu は独自の `Portal/Positioner/Popup` surface を `SubmenuRoot` と `SubmenuTrigger` で囲みます。

## API

`Root` は controlled / uncontrolled の open state、orientation、modality、focus loop、dismissal を管理します。`Trigger` が menu を開き、`createHandle` が分離した trigger と型付き payload を root に接続します。

`Portal`、`Backdrop`、`Positioner`、`Popup`、`Arrow`、任意の `Viewport` が collision-aware layer を構成します。`Group`、`GroupLabel`、`Separator` は内容を整理します。command には `Item`、navigation には `LinkItem` を使い、link を選択時に閉じる場合は `closeOnClick` を指定します。

独立した選択には `CheckboxItem` と `CheckboxItemIndicator` を使います。単一選択では `RadioGroup` に `RadioItem` と `RadioItemIndicator` を配置します。`SubmenuRoot` と `SubmenuTrigger` は nested `Portal` / `Positioner` / `Popup` を接続します。Base UI が roving focus、typeahead、keyboard activation、disabled item の skip、portal の配置、outside/Escape dismissal を担当します。
