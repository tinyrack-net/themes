## 実装

Spacing の値は TypeScript メタデータ、ランタイム CSS 変数、接頭辞付き Tailwind ユーティリティとして利用できます。

```html
<div class="grid gap-tinyrack-md p-tinyrack-lg">
  <div class="flex items-center gap-tinyrack-sm">...</div>
  <section style="margin-top: var(--tinyrack-space-xl)">...</section>
</div>
```

```ts
import { tinyrackSpacing } from '@tinyrack/ui/core';

const panelPadding = tinyrackSpacing.lg;
const sectionGap = tinyrackSpacing.xl;
```

コンポーネント CSS では `--tinyrack-space-*` を、Tailwind レイアウトでは `gap/p/m-tinyrack-*` ユーティリティを使います。
