# AlertDialog

A modal confirmation surface for destructive or irreversible actions.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Storybook controls cover `label`, `open`, `disabled`.
