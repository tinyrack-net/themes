# Badge

Compact status text for operational metadata, health states, and dense labels.

## Contract

| Surface | API | Values | Default | Rule |
| --- | --- | --- | --- | --- |
| CSS class | `tr-badge` | span element | required | Use on inline status text only. |
| Size attribute | `data-size` | `sm`, `md` | `sm` | Use `sm` for table metadata and `md` near controls. |
| Variant attribute | `data-variant` | `neutral`, `primary`, `danger` | `neutral` | Variants describe state, not decoration. |
| React prop | `size` | `sm`, `md` | `sm` | Mirrors `data-size`. |
| React prop | `variant` | `neutral`, `primary`, `danger` | `neutral` | Mirrors `data-variant`. |
| Native props | `HTMLAttributes<HTMLSpanElement>` | span props | pass-through | Preserve inline semantics. |

## Install

| Surface | Install | Import |
| --- | --- | --- |
| CSS / HTML | `pnpm add @tinyrack/ui tailwindcss` | `@tinyrack/ui/core/core.css` and `@tinyrack/ui/components/badge/badge.css` |
| React | `pnpm add @tinyrack/ui react react-dom` | `@tinyrack/ui/components/badge/react` |

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/badge/badge.css";
```

## API

| Prop | Values | Note |
| --- | --- | --- |
| `size` | `sm`, `md` | Default `sm`; sets `data-size`. |
| `variant` | `neutral`, `primary`, `danger` | Default `neutral`; sets `data-variant`. |
| `className` | `string` | Merged with `tr-badge`. |
| native span props | `HTMLAttributes<HTMLSpanElement>` | Passed through to `<span>`. |
