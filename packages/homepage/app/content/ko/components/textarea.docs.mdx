## 계약

Textarea는 native ref, name, FormData, reset, disabled, read-only, required, invalid 상태를 그대로 보존합니다.

Use `uiSize="sm"` for shorter multiline controls in dense surfaces.

## API

`Textarea`는 모든 native textarea prop, ref, event, class, style을 전달합니다. 기본 resize 방향은 vertical이며 native validity가 기준으로 유지됩니다.
