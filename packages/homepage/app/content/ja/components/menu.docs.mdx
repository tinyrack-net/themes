## コントラクト

surface には `Root/Trigger/Portal/Positioner/Popup` を使い、内部に `Item` または `Separator` を構成します。nested menu は独自の `Portal/Positioner/Popup` surface を `SubmenuRoot` と `SubmenuTrigger` で囲みます。

## API

Base UI が roving focus、typeahead、keyboard activation、portal の配置、outside/Escape による解除を担当します。1 つの popup が複数の trigger 間でコンテンツを切り替える場合だけ `Viewport` が必要で、ここでは完全な public anatomy をコピーできるように表示しています。
