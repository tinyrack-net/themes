# Tooltip

Short supporting text shown on hover or keyboard focus.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `data-tr-tooltip` | `Tooltip` | Delayed hover and focus | Keep one trigger and one content part together. |
| Part | `data-tr-tooltip-trigger` | `TooltipTrigger` | `aria-describedby` relationship | Trigger must remain focusable. |
| Part | `tr-tooltip-content`, `role="tooltip"` | `TooltipContent` | Native popover layer | Keep content short and non-interactive. |
| Behavior | `data-open-delay`, `data-close-delay` | `openDelay`, `closeDelay` | 500 ms, 100 ms | Escape closes the active tooltip. |
