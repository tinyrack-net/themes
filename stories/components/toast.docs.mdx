# Toast

Transient feedback with stable IDs, actions and pause-aware timers.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `createToastManager(root)` | `ToastProvider`, `useToast` | `show`, `update`, `dismiss` | Use stable IDs when feedback changes in place. |
| Part | `tr-toast-viewport` | `ToastViewport` | Six logical viewport positions | Viewport stays inside the provided Document or ShadowRoot. |
| Variant | `data-variant` | Toast option `variant` | `neutral`, `info`, `success`, `warning`, `danger` | Danger uses alert urgency; others use status. |
| Behavior | Manager timer | Shared manager | Default 5000 ms | Hover, focus and inactive windows pause timeouts. |

## Usage

<ComponentExampleTabs id="toast-basic" title="Show feedback" description="Use the hook below a provider and render one logical viewport." preview={<div className="flex justify-center"><ToastProvider><ToastDemo variant="success" /><ToastViewport /></ToastProvider></div>} sources={[
  { label: 'React', language: 'tsx', code: String.raw`import { ToastProvider, ToastViewport, useToast } from '@tinyrack/ui/components/toast/react';

function SaveButton() {
  const toast = useToast();
  return <button onClick={() => toast.show({ title: 'Saved', variant: 'success' })}>Save</button>;
}

<ToastProvider>
  <SaveButton />
  <ToastViewport />
</ToastProvider>;` },
  { label: 'HTML', language: 'html', code: String.raw`<div class="tr-toast-viewport" data-tr-toast-viewport data-position="block-end-inline-end" popover="manual" aria-label="Notifications"></div>
<script type="module">
  import { createToastManager } from '@tinyrack/ui/components/toast/dom';
  const toast = createToastManager(document);
  toast.show({ title: 'Saved', variant: 'success' });
</script>` },
]} />

## API

| Component | Main props | Note |
| --- | --- | --- |
| `createToastManager` | `show`, `update`, `dismiss`, `destroy` | Root-scoped and ShadowRoot-safe. |
| `ToastProvider`, `ToastViewport`, `useToast` | React context and logical position | React delegates rendering and timers to the DOM manager. |
