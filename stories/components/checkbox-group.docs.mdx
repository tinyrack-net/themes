# CheckboxGroup

Coordinates the value and disabled state of related checkboxes.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Storybook controls cover `disabled`, `selectedValues`, `label`.

## Install
