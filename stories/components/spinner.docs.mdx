# Spinner

Announced indeterminate progress with size and semantic color axes.

## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | `md` |
| `variant` | `current`, `muted`, `primary`, `danger` | `current` |
| `label` | accessible status text | `Loading` |
| `decorative` | `true`, `false` | `false` |

## API

By default Spinner exposes `role="status"`; use a task-specific `label`. Set `decorative` inside an already named loading control or status region so only one element is announced.
