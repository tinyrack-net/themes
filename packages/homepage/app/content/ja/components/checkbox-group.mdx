## コントラクト

| 軸 | 動作 |
| --- | --- |
| value | `defaultValue` で非制御、`value` と `onValueChange` で制御します。 |
| form | 各 checkbox に同じ `name` を指定します。外部の `form` 所有者を使う場合も、チェックされた各項目が自身の `value` を提供します。 |
| availability | group の `disabled` は子項目へ伝播し、個別の checkbox には `readOnly` を指定できます。 |
| accessibility | group に名前を付けます。`TRCheckbox.Root` を label で囲むか、native input ID を `htmlFor` で参照すると、ビジュアル checkbox role の名前になります。 |
| parent | `allValues` を渡して `parent` checkbox をレンダリングすると、checked、empty、mixed 状態を表現できます。 |
| reset | native form reset は `defaultValue` を復元します。制御された group は引き続き `value` が状態を所有します。 |
| validation | 最小または最大選択数が必要な場合は、制御された値と TRForm/TRField のルールを組み合わせます。 |

  ['metrics', 'Metrics'],
  ['alerts', 'Alerts'],
  ['backups', 'Automated backups'],
];

function FeatureForm() {
  const [attempted, setAttempted] = useState(false);
  const [selected, setSelected] = useState(['metrics']);
  const [submitted, setSubmitted] = useState('');
  const invalid = attempted && (selected.length < 1 || selected.length > 2);

  return (
    <TRForm onSubmit={(event) => {
      event.preventDefault();
      setAttempted(true);
      const values = new FormData(event.currentTarget).getAll('features');
      setSubmitted(values.length >= 1 && values.length <= 2 ? values.join(', ') : '');
    }}>
      <TRCheckboxGroup
        aria-describedby={invalid ? 'feature-error' : undefined}
        aria-invalid={invalid || undefined}
        aria-label="Included features"
        onValueChange={(values) => { setSelected(values); setSubmitted(''); }}
        value={selected}
      >
        {featureOptions.map(([value, label]) => {
          const inputId = 'feature-' + value;
          const labelId = inputId + '-label';
          return <div key={value}>
            <TRCheckbox.Root
              aria-labelledby={labelId}
              id={inputId}
              name="features"
              value={value}
            >
              <TRCheckbox.Indicator aria-hidden="true">✓</TRCheckbox.Indicator>
            </TRCheckbox.Root>
            <label htmlFor={inputId} id={labelId}>{label}</label>
          </div>;
        })}
      </TRCheckboxGroup>
      {invalid ? <p id="feature-error" role="alert">
        {selected.length < 1 ? 'Select at least one feature.' : 'Select no more than two features.'}
      </p> : null}
      <TRButton type="submit">Save features</TRButton>
      <output aria-live="polite">{submitted}</output>
    </TRForm>
  );
}` }]}
/>

## API

`TRCheckboxGroup`、`TRCheckboxGroupProps`、`TRCheckboxGroupState` は `@tinyrack/ui/components/checkbox-group` から export されます。`onValueChange` は次の `string[]` と Base UI event details を受け取ります。固有の `value` prop とアクセシブルな名前を持つ `TRCheckbox.Root` children をレンダリングします。
