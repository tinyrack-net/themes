## Contract

Use TRNavigationMenu for a site header with a brand, a small number of direct destinations, and optional Product or Resources flyouts. The desktop composition keeps Base UI roving keyboard navigation; the mobile composition exposes the same destinations in a TRDrawer.

## API

Every compound part and prop/state type is exported from `@tinyrack/ui/components/navigation-menu`; styles remain opt-in through the matching CSS subpath.

TRNavigationMenu is intended for pointer-and-keyboard site navigation with enough horizontal room. Keep flyout groups task-oriented and preserve direct links for destinations that need no preview. At narrow breakpoints, swap the composition for a TRDrawer with the same destination set instead of compressing the horizontal keyboard model.
