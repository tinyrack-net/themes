# Elevation

Elevation separates temporary UI that sits above the page. Keep normal cards and sections flat; organize them with surface color and borders.

## Implementation

Popover, Toast, and Dialog already apply the correct elevation through their component CSS. Use a public shadow token only when building a custom floating surface; do not add it to normal layout sections.

```css
.rack-popover {
  box-shadow: var(--tinyrack-shadow-raised);
}
```

```html
<div popover class="rounded-tinyrack-lg shadow-tinyrack-raised">
  Rack actions
</div>
```

Use `tinyrackShadows.raised` and `tinyrackShadows.overlay` when tooling or generated styles need the same values in TypeScript.
