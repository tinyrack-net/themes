## コントラクト

| 軸 | 動作 |
| --- | --- |
| 状態 | `defaultChecked` を使う非制御方式、または `checked` と `onCheckedChange` を使う制御方式 |
| セマンティクス | switch を表示ラベルに関連付けます。複数の選択肢をまとめて送信する場合は checkbox を使います。 |
| フォーム | hidden native input を通じて `form`、`name`、`value`、`uncheckedValue`、`required` をサポートします。 |

## API

`TRSwitch`、`TRSwitchRoot`、`TRSwitchThumb`、`TRSwitchRootProps`、`TRSwitchThumbProps`、`TRSwitchRootState`、`TRSwitchThumbState` は `@tinyrack/ui/components/switch` から export されます。CSS は opt-in です。表示ラベルを `Root` に関連付け、Space で切り替えます。変更できない focusable value には `readOnly`、利用できない control には `disabled` を使います。hidden native input が `form`、`name`、`value`、`uncheckedValue`、`required` を保持し、フォーム送信、reset、検証をサポートします。
