# IconButton

An icon-only action that shares Button visuals and requires an accessible name.

## Contract

Provide `aria-label` or `aria-labelledby`. Appearance, variant, size, loading, disabled, refs, and events match Button.

## API

`IconButton` accepts the Button contract but requires exactly one accessible naming prop. Its CSS subpath includes Button CSS.
