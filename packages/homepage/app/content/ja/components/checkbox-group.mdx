## コントラクト

| 軸 | 動作 |
| --- | --- |
| value | uncontrolled with `defaultValue`, controlled with `value` and `onValueChange` |
| form | give each checkbox the same `name`; every checked item contributes its `value` |
| availability | group-level `disabled` cascades; individual checkboxes can be `readOnly` |
| accessibility | name the group and every `TRCheckbox.Root`; a native-input `htmlFor` alone does not name the visual checkbox role |
| validation | compose controlled values with TRForm/TRField rules when the product requires a minimum or maximum selection count |

  title="native form の値と選択制限"
  description="機能を 1 つまたは 2 つ選びます。送信は繰り返される native form の値を読み、空または上限超過の選択を拒否し、復旧後にエラーを消します。"

  ['metrics', 'Metrics'],
  ['alerts', 'Alerts'],
  ['backups', 'Automated backups'],
];

function FeatureForm() {
  const [attempted, setAttempted] = useState(false);
  const [selected, setSelected] = useState(['metrics']);
  const [submitted, setSubmitted] = useState('');
  const invalid = attempted && (selected.length < 1 || selected.length > 2);

  return (
    <TRForm onSubmit={(event) => {
      event.preventDefault();
      setAttempted(true);
      const values = new FormData(event.currentTarget).getAll('features');
      setSubmitted(values.length >= 1 && values.length <= 2 ? values.join(', ') : '');
    }}>
      <TRCheckboxGroup
        aria-describedby={invalid ? 'feature-error' : undefined}
        aria-invalid={invalid || undefined}
        aria-label="Included features"
        onValueChange={(values) => { setSelected(values); setSubmitted(''); }}
        value={selected}
      >
        {featureOptions.map(([value, label]) => {
          const inputId = 'feature-' + value;
          const labelId = inputId + '-label';
          return <div key={value}>
            <TRCheckbox.Root
              aria-labelledby={labelId}
              id={inputId}
              name="features"
              value={value}
            >
              <TRCheckbox.Indicator aria-hidden="true">✓</TRCheckbox.Indicator>
            </TRCheckbox.Root>
            <label htmlFor={inputId} id={labelId}>{label}</label>
          </div>;
        })}
      </TRCheckboxGroup>
      {invalid ? <p id="feature-error" role="alert">
        {selected.length < 1 ? 'TRSelect at least one feature.' : 'TRSelect no more than two features.'}
      </p> : null}
      <TRButton type="submit">Save features</TRButton>
      <output aria-live="polite">{submitted}</output>
    </TRForm>
  );
}` }]}
/>

## API

`TRCheckboxGroup` とその prop/state type は `@tinyrack/ui/components/checkbox-group` から export されます。各ビジュアル checkbox role に固有の `value` prop とアクセシブルな名前を持つ `TRCheckbox.Root` children をレンダリングします。
