# Spacing

A compact relationship scale where proximity explains whether interface elements belong together, repeat as a component, or begin a new task.

## Implementation

Spacing values are public TypeScript metadata, not Tinyrack-prefixed CSS variables or utilities. In rendered layouts, use the matching standard Tailwind spacing utility or read the metadata in tooling.

```html
<div class="grid gap-3 p-4">
  <div class="flex items-center gap-2">...</div>
  <section class="mt-6">...</section>
</div>
```

```ts
import { tinyrackSpacing } from '@tinyrack/ui/core';

const panelPadding = tinyrackSpacing.lg;
const sectionGap = tinyrackSpacing.xl;
```

Do not write `gap-tinyrack-md` or `--tinyrack-spacing-md`; those runtime contracts are not exported.
