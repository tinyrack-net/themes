## 계약

`Provider`는 관련 trigger group 전체의 열기·닫기 지연을 조정합니다.
`Root`는 `defaultOpen` 또는 제어형 `open`/`onOpenChange` 상태를 지원하고
`Positioner`는 side, alignment, offset, collision 처리를 담당합니다. popup content는
보조적이고 비상호작용적이어야 하며 action은 TRPopover나 TRDialog에 배치하세요.

## API

Base UI가 hover/focus timing, accessible description 연결, portal positioning,
collision 처리와 Escape 닫기를 담당합니다. 모든 compound part와 prop/state type은
`@tinyrack/ui/components/tooltip`에서 export되며 대응하는 CSS는 opt-in입니다.
icon-only trigger에는 자체 `aria-label`을 제공하고, 터치 사용자는 content를
hover할 수 없으므로 tooltip만을 필수 정보의 유일한 출처로 삼지 마세요.
