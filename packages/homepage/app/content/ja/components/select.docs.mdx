## コントラクト

| 軸 | 動作 |
| --- | --- |
| value | uncontrolled with `defaultValue`, controlled with `value` and `onValueChange` |
| disclosure | `Trigger` owns focus; `Portal`, `Positioner`, and `Popup` own the floating layer |
| collection | groups, labels, separators, disabled items, text, and indicators describe the menu anatomy |

trigger が密度の高いコントロールサーフェスに属する場合は `Select.Trigger uiSize="sm"` を使います。

## API

trigger、popup、collection、item、arrow、backdrop、scroll-arrow の完全な anatomy と対応する prop/state type は `@tinyrack/ui/components/select` から export されます。`Trigger` にアクセシブルな名前を付け、必要に応じて `value` と `open` を独立して制御します。利用できない control には `disabled`、変更できない focusable value には `readOnly` を使います。
