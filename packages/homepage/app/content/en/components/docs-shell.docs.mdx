## Contract

`Root`, `Header`, `Brand`, `Actions`, `Sidebar`, `Main`, and `Outline` compose a responsive shell. `Header` spans the viewport above the sidebar and content in `docs` and `splash` layouts. Router state arrives through props only.

## Examples

Use `splash` for product landing pages that retain the global header and `standalone` for embedded API references without navigation chrome.

## API

`layout` supports `docs`, `splash`, and `standalone`; mobile disclosure and scroll restoration remain inside UI.
