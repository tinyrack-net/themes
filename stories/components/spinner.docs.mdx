# Spinner

Announced indeterminate progress with size and semantic color axes.

## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | `md` |
| `variant` | `current`, `muted`, `primary`, `danger` | `current` |
| `label` | accessible status text | `Loading` |

## API

`Spinner` always exposes `role="status"`; use a task-specific `label` instead of relying on the default in product UI.
