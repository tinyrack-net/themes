## 主なプロパティ

`Root`、`Header`、`Brand`、`Actions`、`Sidebar`、`Main`、`Outline` でレスポンシブなシェルを構成します。`docs` はデスクトップナビゲーションとフォーカス管理されたモバイルドロワーを提供し、`splash` はグローバルヘッダーだけを残し、`standalone` は両方のナビゲーション領域を取り除きます。ルーターの状態はプロパティだけで渡されます。

## API

`Root` には `currentPath` と `locationKey` が必要です。シェル内の見出しへのスクロールには `hash`、スクロール復元には `POP`、`PUSH`、`REPLACE` のいずれかを取る `navigationKind`、保留中の進捗と `aria-busy` の表示には `pendingPath` を渡します。`layout` は `docs`、`splash`、`standalone` から選びます。

`Header`、`Brand`、`Actions`、`Sidebar`、`Outline` はネイティブプロパティ、クラス名、スタイル、`ref` を保持します。`Main` は内部コンテンツ用の `contentClassName` と、スクロール領域のアクセシブルな名前に使う `viewportLabel` も受け取ります。

制限されたプレビューやアプリケーション領域にシェルを埋め込む場合は、`--tr-docs-shell-block-size` を設定してください。既定値は `100dvh` です。
