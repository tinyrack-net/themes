## コントラクト

`TRContextMenu` は rack row、file、canvas などのコンテンツにコマンドを接続します。専用 trigger が常にコマンド一覧を開く場合は `TRMenu` を使います。この Tinyrack wrapper は Base UI の配置、フォーカス、解除、React 19 ref、ネイティブ prop、event、`render` contract を保ちます。Playground control は表示される `label`、item の `variant`、`disabledItem` 状態に対応し、open 状態は control に表示せず同期します。

## API

すべての part、prop type、state type は `@tinyrack/ui/components/context-menu` から export されます。スタイルは `@tinyrack/ui/components/context-menu.css` から任意で読み込みます。

### `TRContextMenu.Root`

制御または非制御の open 状態、方向、フォーカス循環、解除、root disabled 動作を管理します。

### `TRContextMenu.Trigger`

右クリックまたは長押しの対象を定義します。Shift+F10 とプラットフォームの Context Menu キーを使うにはフォーカス可能にします。

### `TRContextMenu.Portal`

overlay anatomy を `document.body` またはカスタム container に移動します。外部アニメーションで node を保持する場合は `keepMounted` を使います。

### `TRContextMenu.Backdrop`

popup の背後に透明な modal 操作 layer を提供します。

### `TRContextMenu.Positioner`

root popup を pointer または keyboard 座標に、nested popup を submenu trigger の隣に配置します。明示した `anchor` は pointer anchor より優先されます。

### `TRContextMenu.Popup`

`menu` をレンダリングし、keyboard navigation と typeahead を管理し、制約付き scroll viewport を提供します。

### `TRContextMenu.Arrow`

popup と anchor の間に任意の矢印をレンダリングします。

### `TRContextMenu.Group`

関連するコマンドをまとめます。

### `TRContextMenu.GroupLabel`

コマンド group に label を付けます。

### `TRContextMenu.Item`

コマンドを実行します。必要に応じて `disabled`、`closeOnClick`、Tinyrack の `variant="danger"` を使います。

### `TRContextMenu.LinkItem`

`href` を持つ semantic link コマンドをレンダリングします。移動時に menu も閉じる場合は `closeOnClick` を設定します。

### `TRContextMenu.CheckboxItem`

制御または非制御の checked 状態を持つ boolean コマンドです。

### `TRContextMenu.CheckboxItemIndicator`

item が checked のとき checkbox 選択コンテンツをレンダリングします。

### `TRContextMenu.RadioGroup`

相互排他的なコマンド群の選択値を管理します。

### `TRContextMenu.RadioItem`

radio group 内の一つの値を表します。

### `TRContextMenu.RadioItemIndicator`

item が選択されたとき radio 選択コンテンツをレンダリングします。

### `TRContextMenu.SubmenuRoot`

nested menu の open 状態と Escape 動作を管理します。

### `TRContextMenu.SubmenuTrigger`

pointer または keyboard navigation で nested menu を開きます。

### `TRContextMenu.Separator`

コマンド group を視覚的に分けます。

タッチ利用者が右クリックできない場合に同じ context-menu action を dispatch する明示的なボタンを用意します。
