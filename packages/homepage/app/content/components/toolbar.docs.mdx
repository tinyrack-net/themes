# Toolbar

Groups related controls with roving keyboard focus.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Playground controls cover `label`, `disabled`, `orientation`, and `loopFocus`.

## API

Every compound part and prop/state type is exported from `@tinyrack/ui/components/toolbar`; styles remain opt-in through the matching CSS subpath. `Root` defaults to horizontal arrow-key focus with looping enabled; vertical toolbars use Up/Down. `Group` names related controls, `Input` keeps normal text-editing keys, and disabled groups/items suppress activation. Keep a focusable disabled Button visibly unavailable, and give every Toolbar and nested Group an accessible name.
