## コントラクト

| 軸 | 値 | 既定値 | 用途 |
| --- | --- | --- | --- |
| `variant` | `secondary`, `primary`, `danger` | `secondary` | Communicates action intent. |
| `appearance` | `solid`, `outline`, `ghost` | `solid` | Changes emphasis without changing intent. |
| `size` | `sm`, `md`, `lg` | `md` | Matches the surrounding control density. |
| state | `disabled`, `loading` | interactive | Uses native disabled behavior and `aria-busy`. |

## API

- `Button` は Base UI button props、ネイティブ event、`className`、`style`、React 19 `ref` を転送します。
- `type` の既定値は `button` です。form 内で意図的に `type="submit"` を設定します。
- `loading` は control を無効化し、`aria-hidden` spinner をレンダリングし、名前付き button に `aria-busy` を設定します。これにより nested loading notification を避けます。
- Secondary outline と ghost appearance は、低コントラストの border token ではなく読みやすい foreground contrast のため `--tinyrack-text` を使います。
