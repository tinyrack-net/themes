## 계약

Navigation은 재귀적인 `group`, `page`, `link` 데이터를 받습니다. `renderLink`는 Base UI render contract를 통해 활성 라우터 링크를 주입합니다.

## 예시

중첩 그룹에 활성 페이지나 대기 중인 페이지가 포함되어 있으면 자동으로 열립니다.

## API

`currentPath`와 `pendingPath`는 단순한 값입니다. 이 컴포넌트는 React Router를 import하거나 route를 계산하지 않습니다.
