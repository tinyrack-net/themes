# Code

Inline code styling for commands, paths, environment variables, and compact API names.

## Contract

| Surface | API | Values | Default | Rule |
| --- | --- | --- | --- | --- |
| CSS class | `tr-code` | code element | required | Use only for inline code, not block examples. |
| React wrapper | `Code` | `<code>` | required for React | Merges `tr-code` with native props. |
| Native props | `HTMLAttributes<HTMLElement>` | code props | pass-through | Keep short inline content readable. |

## Install

| Surface | Install | Import |
| --- | --- | --- |
| CSS / HTML | `pnpm add @tinyrack/ui tailwindcss` | `@tinyrack/ui/core/core.css` and `@tinyrack/ui/components/code/code.css` |
| React | `pnpm add @tinyrack/ui react react-dom` | `@tinyrack/ui/components/code/react` |
| React MDX | `pnpm add @tinyrack/ui @mdx-js/react` | `@tinyrack/ui/mdx/react` maps markdown inline code. |
| Astro MDX | `pnpm add @tinyrack/ui astro @astrojs/mdx` | `@tinyrack/ui/mdx/astro` maps markdown inline code. |

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/code/code.css";
```

## API

| Prop | Values | Note |
| --- | --- | --- |
| `className` | `string` | Merged with `tr-code`. |
| native code props | `HTMLAttributes<HTMLElement>` | Passed through to `<code>`. |
