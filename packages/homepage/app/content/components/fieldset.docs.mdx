# Fieldset

Groups related controls under one native, accessible legend.

## Contract

`Root` supplies native fieldset grouping and disabled propagation. `Legend` names the group; each child control still needs its own visible label.

## API

`Root` and `Legend` are exported individually and through `Fieldset`; nested form controls retain their native submission behavior.
