## コントラクト

`TRDocsNavigation` は、名前付きのナビゲーション landmark 内に再帰的な `group`、`page`、`link` 項目をレンダリングします。現在または保留中の path を含むグループは、ルート状態の変更後も自動的に開きます。ほかのグループはポインターまたはキーボードで開閉できます。

## アクセシビリティ

ルートは `label` で名前を付けた `<nav>` です。現在のページには `aria-current="page"`、保留中のリンクには `data-pending` と装飾用 spinner が設定されます。グループ trigger は Collapsible が提供するネイティブ button で、`Enter`、`Space`、通常の tab ナビゲーションに対応します。項目 label は明確にし、既定の `Documentation` が適切でない場合は `label` をローカライズしてください。

## API

### `TRDocsNavigation`

| Prop | Type | 既定値 | 用途 |
| --- | --- | --- | --- |
| `items` | `readonly TRDocsNavigationItem[]` | 必須 | 再帰的なナビゲーションデータです。 |
| `currentPath` | `string` | 必須 | 現在のページとして示す完全一致の path です。 |
| `pendingPath` | `string` | `undefined` | 保留中として示す遷移先の完全一致 path です。 |
| `defaultGroupsOpen` | `boolean` | `false` | 現在または保留中の path を含まないグループも初期状態で開きます。 |
| `label` | `string` | `'Documentation'` | ナビゲーション landmark のアクセシブルな名前です。 |
| `renderLink` | `(item, state) => ReactElement` | ネイティブ anchor | フレームワークのルーターリンクを注入します。`state` は `active` と `pending` を含みます。 |
| `onNavigate` | `(item) => void` | `undefined` | page または link のアクティベート後に実行します。モバイル drawer を閉じる用途などに使えます。 |

`TRDocsNavigation` はネイティブ `nav` prop、`className`、`style`、イベント、data attribute、React 19 の `ref` も転送します。

### `TRDocsNavigationItem`

`page` 項目は `label`、`path`、`type: 'page'` を持ちます。`link` 項目には任意の `external` が加わります。外部リンクに target や専用表示が必要な場合は、`renderLink` でこのメタデータを使用してください。`group` 項目は `label`、`type: 'group'`、再帰的な `children` を持ちます。

公開されるデータ type は `TRDocsNavigationItem`、`TRDocsNavigationGroup`、`TRDocsNavigationPage`、`TRDocsNavigationLink` です。`TRDocsNavigationLinkState` は `renderLink` に渡す `active` と `pending` の boolean を含み、`TRDocsNavigationProps` はコンポーネントの完全な prop type です。

## スタイル

core token の後にコンポーネント stylesheet を import してください。`--tr-docs-navigation-gap`、`--tr-docs-navigation-item-gap`、`--tr-docs-navigation-group-color`、`--tr-docs-navigation-link-color`、`--tr-docs-navigation-link-hover-background`、`--tr-docs-navigation-link-hover-color`、`--tr-docs-navigation-link-active-border-color` を上書きできます。長い label は狭い sidebar やモバイル drawer 内で折り返されます。

## SSR

サーバー出力には現在の path と初期状態で開く祖先グループが含まれます。同じ `items`、`currentPath`、`pendingPath` で hydrate してください。hydration 後のルート変更では、ナビゲーションツリーを置き換えずに新しく該当するグループを開きます。
