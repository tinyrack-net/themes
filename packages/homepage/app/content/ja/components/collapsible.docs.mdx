## コントラクト

| 軸 | 値 | 既定値 |
| --- | --- | --- |
| state | controlled `open` or uncontrolled `defaultOpen` | closed |
| availability | `disabled` | interactive |
| lifecycle | unmount, `keepMounted`, or `hiddenUntilFound` | unmount |

## API

Base UI は button の意味、`aria-expanded`、panel の関係、focus、transition-state data attribute を提供します。Tinyrack CSS は `--tr-collapsible-duration`、`--tinyrack-duration-normal`、`--tinyrack-ease-standard` で panel をアニメーションし、reduced motion が選ばれた場合は装飾的な motion を取り除きます。
