# Menubar

Coordinates application menus with desktop-style arrow-key navigation.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Playground controls cover `disabled`, `orientation`, `loopFocus`, and the accessible `label`.
