# Radio

Mutually exclusive native choices with vertical or horizontal group layout.

## Contract

| Surface | API | Values | Default | Rule |
| --- | --- | --- | --- | --- |
| Group class | `tr-radio-group` | fieldset element | required for groups | Use fieldset semantics for related choices. |
| Group orientation | `data-orientation` | `vertical`, `horizontal` | `vertical` | Match the space where the choices appear. |
| Wrapper class | `tr-radio` | label element | required | Wraps the native input and visible label. |
| Input class | `tr-radio-input` | native radio | required | Radios share a native `name`. |
| Control class | `tr-radio-control` | decorative span | required | Mark with `aria-hidden="true"`. |
| Size attribute | `data-size` | `sm`, `md`, `lg` | `md` | Shared with `Checkbox` and `Switch`. |

## Install

| Surface | Install | Import |
| --- | --- | --- |
| CSS / HTML | `pnpm add @tinyrack/ui tailwindcss` | `@tinyrack/ui/core/core.css` and `@tinyrack/ui/components/form/form.css` |
| React | `pnpm add @tinyrack/ui react react-dom` | `@tinyrack/ui/components/form/react` |

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/form/form.css";
```

## API

| Prop | Values | Note |
| --- | --- | --- |
| `RadioGroup.orientation` | `vertical`, `horizontal` | Default `vertical`; sets `data-orientation`. |
| `Radio.size` | `sm`, `md`, `lg` | Default `md`; sets `data-size`. |
| `Radio.invalid` | `boolean` | Sets `aria-invalid` on input and `data-invalid` on wrapper. |
| `children` | `ReactNode` | Renders `tr-radio-label` when present. |
| native radio props | `InputHTMLAttributes<HTMLInputElement>` except `children`, `size`, `type` | Passed through to the native radio input. |
| native fieldset props | `FieldsetHTMLAttributes<HTMLFieldSetElement>` | Passed through to `RadioGroup`. |
