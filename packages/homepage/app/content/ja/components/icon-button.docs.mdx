## コントラクト

`aria-label` または `aria-labelledby` を指定します。appearance、variant、size、loading、disabled、ref、event は Button と同じです。

      aria-label={appearance + ' ' + variant + ' ' + size}

## API

`IconButton` は Button contract を受け入れますが、アクセシブルな naming prop が 1 つだけ必要です。CSS subpath には Button CSS が含まれます。
