# ContextMenu

A right-click menu with selection, groups, links, and submenus.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Storybook controls cover `label`, `open`, `disabledItem`.
