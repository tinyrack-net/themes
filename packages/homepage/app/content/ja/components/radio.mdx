## コントラクト

| 軸 | 動作 |
| --- | --- |
| 構成 | `TRRadio.Root` は必ず `TRRadioGroup` 内に配置し、`TRRadio.Indicator` は root 内で任意に使う視覚 anatomy です |
| 選択 | group が `value`、`defaultValue`、`onValueChange` を所有し、各 root は一意の `value` を 1 つ提供します |
| 操作 | label または radio をクリックして選択し、矢印キーで group 内の focus と選択を移動します |
| 状態 | root は Base UI data attribute で checked、unchecked、invalid、read-only、disabled、required、focus、field 状態を公開します |
| form | `name`、`form`、group 単位の `required` は `TRRadioGroup` に指定し、非表示 native input が必要な場合だけ root の `inputRef` を使います |

Radio を単独の boolean control として使わないでください。1 つの独立した選択をオン・オフする場合は `TRCheckbox` または `TRSwitch` を使います。

## プレイグラウンド

`disabled` control は primary item に適用され、`readOnly` は group 全体に適用されます。Preview で 2 つの item を選択でき、Reset は primary の選択を復元します。

  const [value, setValue] = useState('standard');
  const id = useId();
  const plans = [
    ['standard', 'Standard · Community support'],
    ['priority', 'Priority · 4-hour response'],
    ['critical', 'Critical · 24/7 response'],
  ];

  return (
    <TRRadioGroup aria-label="Support plan" name="support-plan" onValueChange={(nextValue) => setValue(String(nextValue))} value={value}>
      {plans.map(([planValue, label]) => (
        <label htmlFor={id + '-' + planValue} key={planValue}>
          <TRRadio.Root id={id + '-' + planValue} value={planValue}><TRRadio.Indicator aria-hidden="true" /></TRRadio.Root>
          {label}
        </label>
      ))}
      <output aria-live="polite">Selected plan: {value}</output>
    </TRRadioGroup>
  );
}` }]}
/>

## API

`TRRadio` は `Root` と `Indicator`、それぞれの prop type と state type、`TRRadioUiSize` を export します。`Root` は Base UI の `render`、関数形式の `className`/`style`、root `ref`、非表示 input の `inputRef`、native attribute、event handler を保持します。1 つの option だけ利用不可にする場合は root に `readOnly` または `disabled` を指定し、すべての option を利用不可にする場合は同じ group prop を優先します。`Indicator` は選択中だけ mount され、`keepMounted` で維持できます。

選択 event は個別の root ではなく `TRRadioGroup.onValueChange` に属します。Group に accessible name を、すべての root に accessible label を付けてください。Consumer token には `--tr-radio-size`、`--tr-radio-indicator-size`、base/hover/checked/invalid の border と background token、indicator background が含まれます。
