## 계약

이 Tinyrack wrapper는 Base UI 동작, React 19 ref, native prop, event와 `render` contract를 유지합니다. Playground는 orientation, Root disabled, controlled toggle state를 다루고 아래 example은 focus loop와 세부 disabled 동작을 보여 줍니다. Bold를 누르면 controlled preview arg가 갱신되고 Reset은 Bold pressed 상태와 모든 control을 기본값으로 복원합니다.

## API

### Anatomy

```tsx
<TRToolbar.Root aria-label="Editor controls">
  <TRToolbar.Group aria-label="Formatting"><TRToolbar.Button>Bold</TRToolbar.Button></TRToolbar.Group>
  <TRToolbar.Separator />
  <TRToolbar.Link href="#help">Help</TRToolbar.Link>
  <TRToolbar.Input aria-label="Document title" />
</TRToolbar.Root>
```

전체 anatomy는 `Root`, `Group`, `Button`, `Link`, `Input`, and `Separator`입니다. 각 part는 `TRToolbar.Part`와 named `TRToolbarPart` export로 제공되며 대응 props/state type도 export됩니다.

### Root props

| Prop | Type | 기본값 | 용도 |
| --- | --- | --- | --- |
| `orientation` | `horizontal \| vertical` | `horizontal` | Left/Right 또는 Up/Down roving navigation을 선택합니다. |
| `loopFocus` | `boolean` | `true` | 양 끝에서 keyboard focus를 순환합니다. |
| `disabled` | `boolean` | `false` | toolbar의 button과 input을 비활성화합니다. |
| `render` | `ReactElement \| function` | `<div>` | 동작과 ref를 보존하면서 Root element를 교체합니다. |

`Group`은 이름 있는 `group`을 렌더링하며 내부 button과 input을 비활성화할 수 있습니다. `Button`은 기본적으로 `type="button"`이고 `focusableWhenDisabled`를 지원합니다. `false`이면 disabled command를 roving focus에서 제외합니다. `Input`도 같은 disabled-focus 정책과 일반 text-editing key를 유지합니다. `Link`는 의도적으로 항상 navigation이 가능하므로 disabled Root/Group 안에서도 활성 상태입니다. `Separator`는 Root와 반대 orientation이 기본입니다. horizontal toolbar에서는 Input을 최대 하나만 마지막에 배치하세요.

모든 part는 `className`, `style`, native attribute, event handler, React 19 ref와 Base UI의 state 기반 `render` contract를 보존합니다. 모든 Root와 Group에 accessible name을 제공하세요.

### Appearance

| Token | Fallback | 용도 |
| --- | --- | --- |
| `--tr-toolbar-background` | `--tinyrack-surface` | Toolbar surface. |
| `--tr-toolbar-border` | `--tinyrack-border` | Toolbar border. |
| `--tr-toolbar-color` | `--tinyrack-text` | Toolbar text. |
| `--tr-toolbar-control-background` | `transparent` | Idle button surface. |
| `--tr-toolbar-control-background-hover` | `--tinyrack-surface-hover` | Enabled button/link hover. |
| `--tr-toolbar-control-background-pressed` | `--tinyrack-surface-pressed` | Active/pressed control. |
| `--tr-toolbar-input-background` | `--tinyrack-surface` | Input surface. |
| `--tr-toolbar-input-border` | `--tinyrack-control-border` | Input border. |
| `--tr-toolbar-separator-color` | `--tinyrack-border` | Separator color. |

`@tinyrack/ui/components/toolbar.css`를 명시적으로 import하세요. foundation token이 light/dark color, geometry, focus indicator와 disabled opacity를 제공합니다.
