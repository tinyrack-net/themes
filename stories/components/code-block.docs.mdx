# CodeBlock

SSR-safe block code with optional client-side Shiki highlighting.

## Contract

| Surface | API | Values | Default | Rule |
| --- | --- | --- | --- | --- |
| CSS class | `tr-code-block` | `<pre><code>` | required | Always readable without JavaScript. |
| Language attribute | `data-language` | `string` | unset | Shows the language label when provided. |
| Wrap attribute | `data-wrap` | `true` | unset | Opt in only for prose-friendly snippets. |
| React wrapper | `CodeBlock` | `children`, `language`, `wrap` | SSR-safe | Does not import Shiki. |
| Client wrapper | `ShikiCodeBlock` | `code`, `language`, `theme`, `wrap` | `github-dark` | Requires optional `shiki` peer. |

## Install

| Surface | Install | Import |
| --- | --- | --- |
| CSS / HTML | `pnpm add @tinyrack/ui tailwindcss` | `@tinyrack/ui/core/core.css` and `@tinyrack/ui/components/code-block/code-block.css` |
| React plain block | `pnpm add @tinyrack/ui react react-dom` | `@tinyrack/ui/components/code-block/react` |
| React Shiki block | `pnpm add shiki` | `@tinyrack/ui/components/code-block/shiki-react` |
| React MDX | `pnpm add @tinyrack/ui @mdx-js/react shiki` | `@tinyrack/ui/mdx/react` maps fenced code. |
| Astro MDX | `pnpm add @tinyrack/ui astro @astrojs/mdx shiki` | `@tinyrack/ui/mdx/astro` maps fenced code. |

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/code-block/code-block.css";
```

    {

  code={'const answer = 1;'}
  language="ts"
  theme="github-dark"
/>;`,
    },
  ]}
/>

## API

| Prop | Values | Note |
| --- | --- | --- |
| `children` | `ReactNode` | Plain `CodeBlock` content. |
| `code` | `string` | `ShikiCodeBlock` source string. |
| `language` | `string` or Shiki bundled language | Sets `data-language`; Shiki uses it for tokenization. |
| `theme` | Shiki bundled theme | Default `github-dark`. |
| `wrap` | `boolean` | Adds `data-wrap="true"`. |
| `className` | `string` | Merged with `tr-code-block`. |
