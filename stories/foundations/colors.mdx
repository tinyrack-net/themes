# Colors

<p className="tr-doc-lead">
  Role-based color tokens for compact light and dark operational surfaces.
</p>

## Role model

| Role | Tokens | Use |
| --- | --- | --- |
| Layer | `canvas`, `surface`, `surfaceMuted` | Separate page, default surface, and quiet control zones without inventing new ramps. |
| Content | `text`, `textMuted` | Keep primary content stable and use muted text for metadata. |
| Action | `primary`, `error` | Reserve high-contrast fills for the current Button API. |
| State | `border`, `focus` | Use for durable edges and visible keyboard focus. |

## Semantic reference

| Token | Role | Light | Dark |
| --- | --- | --- | --- |
| `--tinyrack-canvas` | Page background | `#ffffff` | `#030303` |
| `--tinyrack-surface` | Default panels and controls | `#ffffff` | `#0a0a0a` |
| `--tinyrack-surface-muted` | Quiet fills and secondary controls | `#f5f5f5` | `#262626` |
| `--tinyrack-text` | Primary content | `#171717` | `#fafafa` |
| `--tinyrack-text-muted` | Metadata and helper text | `#737373` | `#a3a3a3` |
| `--tinyrack-border` | Component edge | `#d4d4d4` | `#404040` |
| `--tinyrack-focus` | Keyboard focus ring | `#737373` | `#a3a3a3` |
| `--tinyrack-primary` | Primary action fill | `#171717` | `#fafafa` |
| `--tinyrack-error` | Danger action fill | `#dc2626` | `#f87171` |
