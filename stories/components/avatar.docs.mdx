# Avatar

Identity imagery with deterministic fallback content, size, and shape.

## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | `md` |
| `shape` | `circle`, `square` | `circle` |
| parts | `Root`, `Image`, `Fallback` | compound API |
