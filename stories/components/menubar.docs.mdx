# Menubar

Coordinates application menus with desktop-style arrow-key navigation.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Storybook controls cover `disabled`, `orientation`, `label`.
