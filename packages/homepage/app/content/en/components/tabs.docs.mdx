## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | `md` |
| `orientation` | `horizontal`, `vertical` | `horizontal` |
| activation | manual or `activateOnFocus` | manual |
| focus | looping or bounded | looping |
| state | controlled `value`, including `null`, or uncontrolled `defaultValue` | first enabled tab |

## API

Base UI supplies roles, focus movement, selection, controlled state, and activation behavior. Manual activation is the default; opt into `activateOnFocus` only when panel changes are immediate and inexpensive. The default documentation composition uses the selected tab border alone. `Indicator` remains exported for an explicitly branded composition that needs a separate active marker.
