## コントラクト

この Tinyrack wrapper は Base UI の動作、React 19 refs、ネイティブ scrolling、`render` contract を保ちます。Playground control は vertical、horizontal、two-axis content に対応します。

## API

`Root` は表示サイズと overflow edge threshold を設定します。既定の `surface` variant は既存の border と padding contract を保ち、`plain` は shell と埋め込み canvas 向けに両方を取り除きます。`autoHide` の既定値は `false` で、有効にすると idle track が隠れ、hover、scroll、focus で再表示されます。`Viewport` はネイティブの wheel、touch、trackpad、keyboard scroll を担うため、focus 可能な viewport にアクセシブルな名前を付けます。`Content` が scrollable layout を保持し、必要な各 axis に `Scrollbar` と `Thumb` を 1 つずつ追加し、2 本の track が交わる場所だけに `Corner` をレンダリングします。`Scrollbar` は `keepMounted` にも対応し、すべての part が ref、ネイティブ props、Base UI `render` contract を保ちます。
