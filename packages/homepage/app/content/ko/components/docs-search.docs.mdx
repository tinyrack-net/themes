## 계약

`onSearch(query, signal)`이 데이터 소스를 관리합니다. TRDialog는 Ctrl/⌘+K, 로딩, 빈 결과, fallback, 강조, 선택, 포커스 및 요청 취소 동작을 관리합니다.

## 예시

기본 검색 색인을 사용할 수 없을 때 결과가 줄어든 이유를 설명하려면 `fallback`을 설정하세요.

## API

표시되는 모든 메시지는 지역화할 수 있습니다. `fallback`은 기본 Pagefind 소스를 사용할 수 없음을 알립니다.
