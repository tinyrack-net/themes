# Code Block

Block-level source display with optional Shiki highlighting and explicit wrapping.

## Contract

| Axis | Values | Default |
| --- | --- | --- |
| `language` | Shiki bundled language | plain text |
| `theme` | Shiki bundled theme | `github-dark` |
| `wrap` | `true`, `false` | `false` |

## Install

<ComponentInstall surfaces={[{ label: 'React', install: 'pnpm add @tinyrack/ui react react-dom', imports: ["import '@tinyrack/ui/core.css';", "import '@tinyrack/ui/components/code-block.css';", "import { CodeBlock } from '@tinyrack/ui/components/code-block';"] }]} />

## API

Highlighting happens after hydration; the original source remains SSR-safe and visible before Shiki loads.
