## コントラクト

前後の遷移先はデータとして渡します。`renderLink` はコンポーネントをルーティング状態に結合せず、ルーター要素を注入します。

## 例

ローカライズされたドキュメント列の先頭または末尾では、片方の遷移先だけを渡します。

## API

方向とランドマークのラベルはローカライズできます。遷移先がない場合は空のプレースホルダーを表示せずに縮小されます。
