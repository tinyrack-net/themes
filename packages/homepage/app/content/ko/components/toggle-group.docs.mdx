## 계약

| Axis | Contract |
| --- | --- |
| selection | single by default; opt into `multiple` |
| state | controlled `value` or uncontrolled `defaultValue`; both use string arrays and default to `[]` |
| orientation | horizontal by default; vertical when requested |
| keyboard | arrow focus loops by default; set `loopFocus={false}` for bounded focus |
| availability | enabled by default; disable the group or an individual item |

## API

ToggleGroup은 Base UI state, arrow-key focus, orientation, ref, native props, event, `render`를 보존합니다. 안정적인 item `value`를 제공하고 제어형·비제어형 selection을 string array로 유지하세요. 가로에서는 Left/Right, 세로에서는 Up/Down arrow를 사용하며 `loopFocus={false}`는 group 가장자리에서 멈춥니다. Group `disabled`는 모든 item에 적용되지만 개별 Toggle은 독립적으로 비활성화할 수 있습니다. group에 접근 가능한 이름을 제공하세요. selection이 이름 있는 form field여야 할 때는 CheckboxGroup 또는 RadioGroup을 사용합니다.
