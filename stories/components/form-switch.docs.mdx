# Switch

Boolean native input rendered with switch semantics and shared choice spacing.

## Contract

| Surface | API | Values | Default | Rule |
| --- | --- | --- | --- | --- |
| Wrapper class | `tr-switch` | label element | required | Wraps the native checkbox and visible label. |
| Input class | `tr-switch-input` | checkbox with `role="switch"` | required | Native input remains focusable. |
| Track class | `tr-switch-track` | decorative span | required | Mark with `aria-hidden="true"`. |
| Thumb class | `tr-switch-thumb` | decorative span | required | Render inside the track. |
| Size attribute | `data-size` | `sm`, `md`, `lg` | `md` | Shared with `Checkbox` and `Radio`. |
| Invalid attribute | `data-invalid` | `true` | unset | Mirrors `aria-invalid` when `invalid` is set. |

## Install

| Surface | Install | Import |
| --- | --- | --- |
| CSS / HTML | `pnpm add @tinyrack/ui tailwindcss` | `@tinyrack/ui/core/core.css` and `@tinyrack/ui/components/form/form.css` |
| React | `pnpm add @tinyrack/ui react react-dom` | `@tinyrack/ui/components/form/react` |

```css
@import "tailwindcss";
@import "@tinyrack/ui/core/core.css";
@import "@tinyrack/ui/components/form/form.css";
```

## API

| Prop | Values | Note |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | Default `md`; sets `data-size`. |
| `invalid` | `boolean` | Sets `aria-invalid` on input and `data-invalid` on wrapper. |
| `checked` / `defaultChecked` | `boolean` | Supports controlled and uncontrolled switch state. |
| `children` | `ReactNode` | Renders `tr-switch-label` when present. |
| native checkbox props | `InputHTMLAttributes<HTMLInputElement>` except `children`, `size`, `type` | Passed through to the native checkbox input. |
