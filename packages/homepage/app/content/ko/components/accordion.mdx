## 계약

| 축 | 값 | 기본값 |
| --- | --- | --- |
| selection | controlled `value` or uncontrolled `defaultValue` | `[]` |
| expansion | single or `multiple` | single |
| availability | root `disabled` or item `disabled` | interactive |
| lifecycle | unmount, `keepMounted`, or `hiddenUntilFound` | unmount |

## API

Base UI가 확장, 제어형·비제어형 상태, trigger 의미, panel 관계를 담당합니다. TRAccordion trigger는 일반 document tab 순서를 유지하며, 더 이상 권장하지 않는 화살표 키 roving-focus props는 이 가이드에 포함하지 않습니다.
