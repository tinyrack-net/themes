## コントラクト

`Root` はネイティブ fieldset grouping と disabled の伝播を提供します。`Legend` が group に名前を付け、各 child control には独自の visible label が必要です。

## API

`Root` と `Legend` は個別にも `Fieldset` 経由でも export されます。ネストした form control はネイティブの送信動作を保ちます。
