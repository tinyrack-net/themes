# AlertDialog

A modal confirmation surface for destructive or irreversible actions.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Playground controls cover `label`, `open`, `disabled`.

            <AlertDialog.Description>This action cannot be undone.</AlertDialog.Description>
            <div className="tr-alert-dialog-actions">
              <AlertDialog.Close>Cancel</AlertDialog.Close>
              <AlertDialog.Close onClick={() => setResult('Rack deleted')}>

              </AlertDialog.Close>
            </div>
          </AlertDialog.Popup>
        </AlertDialog.Viewport>
      </AlertDialog.Portal>
    </AlertDialog.Root>
    <output aria-live="polite">{result}</output>
  </div>;

## API

Every compound part and prop/state type is exported from `@tinyrack/ui/components/alert-dialog`; styles remain opt-in through the matching CSS subpath. Group confirmation controls in `tr-alert-dialog-actions` to keep the actions in a right-aligned horizontal row.
