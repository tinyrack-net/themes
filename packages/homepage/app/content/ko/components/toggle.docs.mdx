| --- | --- | --- |
| state | controlled `pressed` or uncontrolled `defaultPressed` | off |
| availability | enabled or `disabled` | enabled |
| label | visible text or an accessible name for icon-only use | required |

  preview={<Stories.ToggleInteractiveExample />}
  sources={[{ label: 'React', language: 'tsx', code: String.raw`import { Bold } from 'lucide-react';
import { useState } from 'react';
import { TRToggle } from '@tinyrack/ui/components/toggle';

export function ToggleExample() {
  const [pressed, setPressed] = useState(false);

  return (
    <div className="grid justify-items-start gap-3">
      <TRToggle onPressedChange={setPressed} pressed={pressed}>
        <Bold aria-hidden="true" className="h-4 w-4" />
        Bold
      </TRToggle>
      <output aria-live="polite">
        Bold: {pressed ? 'on' : 'off'}
      </output>
    </div>
  );
}` }]}
/>

## API

TRToggle은 Base UI button 동작, ref, native props, event, `render`를 보존합니다. `onPressedChange`와 제어형 `pressed` 또는 비제어형 `defaultPressed`를 사용하세요. native Space/Enter activation과 `disabled` 의미는 button 동작을 따릅니다. icon-only toggle에는 `aria-label`이 필요하고 TRToggleGroup item에는 안정적인 `value`도 필요합니다. polymorphic render는 button keyboard 의미를 보존해야 합니다. form 제출이나 required 검증에 값이 참여해야 하면 TRCheckbox 또는 TRSwitch를 사용하세요.
