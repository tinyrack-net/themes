## API

탐색에는 `TRLink`, 명령에는 `TRButton`을 사용하세요. 활성 link는 Enter로 실행되며 Space는 native anchor처럼 scroll합니다. Disabled link는 `aria-disabled`를 노출하고 native 목적지와 tab stop을 제거하며 pointer, keyboard, programmatic activation을 막습니다. 외부 link가 새 browsing context를 열면 보이는 text로 알리고 `target="_blank"`와 적절한 `rel`을 함께 사용하세요.

| Prop | 타입 / 기본값 | 용도 |
| --- | --- | --- |
| `href` | native anchor `href` | 목적지를 지정하며 disabled일 때 생략됩니다. |
| `variant` | `default \| muted \| danger` / `default` | 의미 기반 text color를 선택합니다. |
| `underline` | `always \| hover \| none` / `hover` | underline 표시 방식을 선택합니다. |
| `disabled` | `boolean` / `false` | 탐색을 막고 순차 focus에서 제거합니다. |
| `render` | Base UI render element 또는 callback | nested anchor 없이 React Router 등의 link를 조합합니다. |
| native props | anchor attributes, events, `className`, `style`, `ref` | current state, 외부/download 속성, handler, style, React 19 ref를 전달합니다. |

### Style tokens

`--tr-link-color`, `--tr-link-font-weight`, `--tr-link-decoration-thickness`, `--tr-link-underline-offset`, `--tr-link-hover-opacity`, `--tr-link-disabled-opacity`, `--tr-link-radius`, `--tr-link-focus-color`, `--tr-link-focus-width`, `--tr-link-focus-offset`, `--tr-link-duration`으로 Tinyrack fallback을 유지하면서 component를 조정할 수 있습니다.
