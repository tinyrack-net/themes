## 계약

| 축 | 동작 |
| --- | --- |
| 구성 | `TRRadio.Root`는 항상 `TRRadioGroup` 안에 배치하며, `TRRadio.Indicator`는 root 안에서 선택적으로 사용하는 시각 anatomy입니다 |
| 선택 | group이 `value`, `defaultValue`, `onValueChange`를 소유하고 각 root는 고유한 `value` 하나를 제공합니다 |
| 상호작용 | label이나 radio를 클릭해 선택하고 화살표 키로 group 안의 focus와 선택을 이동합니다 |
| 상태 | root는 Base UI data attribute로 checked, unchecked, invalid, read-only, disabled, required, focus와 field 상태를 노출합니다 |
| form | `name`, `form`, group 단위 `required`는 `TRRadioGroup`에 지정하고 숨겨진 native input이 필요할 때만 root의 `inputRef`를 사용합니다 |

Radio를 독립적인 boolean control로 사용하지 마세요. 하나의 독립 선택을 켜고 끌 수 있어야 한다면 `TRCheckbox` 또는 `TRSwitch`를 사용하세요.

import { TRRadio } from '@tinyrack/ui/components/radio';

import '@tinyrack/ui/components/radio.css';

export function SupportPlanField() {
  const [value, setValue] = useState('standard');
  const id = useId();
  const plans = [
    ['standard', 'Standard · Community support'],
    ['priority', 'Priority · 4-hour response'],
    ['critical', 'Critical · 24/7 response'],
  ];

  return (
    <TRRadioGroup aria-label="Support plan" name="support-plan" onValueChange={(nextValue) => setValue(String(nextValue))} value={value}>
      {plans.map(([planValue, label]) => (
        <label htmlFor={id + '-' + planValue} key={planValue}>
          <TRRadio.Root id={id + '-' + planValue} value={planValue}><TRRadio.Indicator aria-hidden="true" /></TRRadio.Root>
          {label}
        </label>
      ))}
      <output aria-live="polite">Selected plan: {value}</output>
    </TRRadioGroup>
  );
}` }]}
/>

## API

`TRRadio`는 `Root`와 `Indicator`, 각각의 prop type과 state type, `TRRadioUiSize`를 export합니다. `Root`는 Base UI의 `render`, 함수형 `className`/`style`, root `ref`, 숨겨진 input의 `inputRef`, native attribute와 event handler를 보존합니다. 하나의 option만 사용할 수 없을 때 root에 `readOnly` 또는 `disabled`를 지정하고 모든 option을 사용할 수 없다면 같은 group prop을 우선 사용하세요. `Indicator`는 선택된 동안에만 mount되며 `keepMounted`로 유지할 수 있습니다.

선택 event는 개별 root가 아니라 `TRRadioGroup.onValueChange`에 속합니다. Group에 accessible name을, 모든 root에 accessible label을 제공하세요. Consumer token에는 `--tr-radio-size`, `--tr-radio-indicator-size`, base/hover/checked/invalid border와 background token, indicator background가 포함됩니다.
