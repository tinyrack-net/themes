## Contract

`shape` accepts `text`, `rectangle`, or `circle` and defaults to `text`. Skeletons shimmer by default; set `animate={false}` for a static placeholder. Dimensions come from layout or `style`.

## API

An unlabeled TRSkeleton has `aria-hidden="true"`; adding `aria-label`, `aria-labelledby`, or an explicit status role opts into one live region. The shimmer is disabled automatically when the user prefers reduced motion.

`--tinyrack-skeleton-fill` and `--tinyrack-skeleton-highlight` are theme-aware semantic tokens. Override `--tr-skeleton-fill` or `--tr-skeleton-highlight` locally when a loading surface needs component-scoped contrast tuning.
