## Contract

| Axis | Values | Default |
| --- | --- | --- |
| state | controlled `open` or uncontrolled `defaultOpen` | closed |
| availability | `disabled` | interactive |
| lifecycle | unmount, `keepMounted`, or `hiddenUntilFound` | unmount |

## API

Base UI supplies button semantics, `aria-expanded`, the panel relationship, focus, and transition-state data attributes. Tinyrack CSS animates the panel with `--tr-collapsible-duration`, `--tinyrack-duration-normal`, and `--tinyrack-ease-standard`, then removes decorative motion when reduced motion is preferred.
