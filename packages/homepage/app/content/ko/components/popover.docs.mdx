## 계약

제어형 `open` 또는 비제어형 `defaultOpen`을 사용하세요. `Positioner`는 Base UI의 side, alignment, offset, collision prop을 받습니다.

      <Popover.Portal>

          align="end"
          alignOffset={12}

        >
          <Popover.Popup>

            <Popover.Close>Close</Popover.Close>
          </Popover.Popup>
        </Popover.Positioner>
      </Popover.Portal>
    </Popover.Root>

## API

Base UI가 anchor 위치 지정, collision 회피, focus 동작, portal, dialog 관계, outside/Escape 닫기를 담당합니다. 보조 액션에는 표면을 non-modal로 유지하고 뒤의 page를 막아야 하는 작업에는 Dialog를 사용하세요.
