## コントラクト

`TRDocsSearch.Trigger` は単独のボタン、`TRDocsSearch.Dialog` は制御されたポータル検索画面です。ダイアログは Ctrl/⌘+K、読み込み中、空の結果、エラー、`fallback`、キーボード選択、フォーカス復帰、古いリクエストのキャンセルを管理します。データソースは `onSearch(query, signal)` が管理し、キャンセル可能な処理には受け取った `AbortSignal` を渡してください。

## プレイグラウンド

コントロールでは `disabled` と `compact` のトリガー、3 種類のサイズ、表示ラベルを変更できます。ダイアログを開いてから **リセット** を押すと、コントロールが復元され、内部の操作状態がリセットされます。

### 無効なトリガー

検索を利用できない場合は、ダイアログのショートカットを含むすべての起動経路を無効にします。

<ComponentExampleTabs id="docs-search-disabled" data-docs-example-item-count="1" title="利用できない検索" preview={<Stories.DocsSearchDisabled />} sources={[{ label: 'React', language: 'tsx', code: String.raw`import { useRef, useState } from 'react';
import '@tinyrack/ui/components/docs-search.css';
import { TRDocsSearch } from '@tinyrack/ui/components/docs-search';

export function DisabledSearch() {
  const [open, setOpen] = useState(false);
  const triggerRef = useRef<HTMLButtonElement>(null);

  return <>
    <TRDocsSearch.Trigger
      aria-label="検索できません"
      disabled
      label="ドキュメントを検索"
      onClick={() => setOpen(true)}
      ref={triggerRef}
    />
    <TRDocsSearch.Dialog
      enableShortcut={false}
      onOpenChange={setOpen}
      onSearch={async () => []}
      onSelect={() => setOpen(false)}
      open={open}
      returnFocusRef={triggerRef}
    />
  </>;
}` }]} />

- フォーカスはコンボボックスに留まり、`ArrowDown` と `ArrowUp` でアクティブな結果を移動します。`Enter` は結果を選択し、`Escape` はダイアログを閉じます。
- `enableShortcut={false}` はグローバルショートカットを無効にします。埋め込み型のプレビューや無効なトリガーですべての起動経路を止める場合に使用します。
- `compact` はアクセシブルな名前を保ちながら表示ラベルとショートカットを隠します。`uiSize="sm" | "md" | "lg"` の各サイズでコンパクトなトリガーは正方形を保ちます。
- `Dialog` は `document.body` にポータルされます。閉じた後にトリガーへフォーカスを戻すには `returnFocusRef` を渡します。
- 閉じたダイアログとトリガーは SSR でも安全で、開くまではポータルを描画せずにハイドレーションされます。
- 主要な検索インデックスを利用できない場合は `fallback` を設定し、縮小されたローカル結果が表示される理由を伝えます。検索リクエストが失敗した場合は、「使用方法」のソースで示すローカライズ済みの `messages.error` 状態を使用します。

## API

| エクスポート | 主なプロパティ | 用途 |
| --- | --- | --- |
| `TRDocsSearch` | `Trigger`, `Dialog` | 複合名前空間です。 |
| `TRDocsSearch.Trigger` | `compact`, `label`, `shortcutLabel`, `uiSize`、`button` のプロパティ、`ref` | 表示型またはコンパクトな検索ボタンです。`disabled` はボタン操作を防ぎます。 |
| `TRDocsSearch.Dialog` | `open`, `onOpenChange`, `onSearch`, `onSelect`, `messages`, `fallback`, `enableShortcut`, `popupRef`, `returnFocusRef` | 制御された検索ダイアログと結果一覧です。 |
| `TRDocsSearchResult` | `id`, `url`, `title`, `section`, `excerpt`、任意の一致範囲 | `Dialog` が描画する検索結果です。一致範囲の終了オフセットは含まれません。 |
| `TRDocsSearchMessages` | `title`, `trigger`, `placeholder`, `close`, `results`, `idle`, `loading`, `empty`, `error`, `fallback` | 表示文言とアクセシブルな文言をすべてローカライズします。 |

`onSearch` は `(query: string, signal: AbortSignal)` を受け取り、読み取り専用の結果を格納した `Promise` を返します。リクエストが失敗した場合は `messages.error` を表示し、キャンセルまたは置換されたリクエストが新しい結果を上書きすることはありません。
