## API

`TRAlert.Root` forwards native `div` props and React 19 `ref`; it does not assign a live-region role automatically because announcement urgency is application context. `TRAlert.Title` and `TRAlert.Description` support the Base UI `render` contract when their default `div` and `p` elements do not fit the surrounding semantics.
