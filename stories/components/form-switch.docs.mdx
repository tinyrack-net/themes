# Switch

Boolean native input rendered with switch semantics and shared choice spacing.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `label.tr-switch` | `Switch` | native checkbox with `role="switch"` | Use for an immediate on/off setting. |
| Part | `tr-switch-input` | native input props | `type="checkbox"`, `role="switch"`; required part | The native input remains focusable and operable. |
| Part | `tr-switch-track`, `tr-switch-thumb` | generated track and thumb | decorative spans; required parts | Mark the visual track `aria-hidden="true"`. |
| Part | `tr-switch-label` | `children` | `ReactNode`; omitted by default | Label the setting, not the action. |
| Size | `data-size` | `size` | `sm`, `md`, `lg`; default `md` | Shares the choice-control scale with Checkbox and Radio. |
| State | `data-invalid`, `aria-invalid` | `invalid` | `true` or unset; default unset | Pair invalid state with an associated error message. |
| Native | `checked`, `defaultChecked`, `disabled` | native checkbox props | controlled or uncontrolled; unchecked by default | Preserve native focus and Space-key behavior. |

## Guidance

<ComponentGuidance
  whenToUse={[
    'Use Switch for a binary setting that takes effect immediately, such as enabling automatic deployment.',
    'Write the visible label as the setting name so on and off remain understandable without changing copy.',
  ]}
  avoid={[
    'Do not use Switch for selecting an item, accepting terms, or submitting a choice that takes effect later.',
    'Do not make the decorative track focusable or rely on track color as the only state indicator.',
  ]}
  accessibility={[
    'Keep the native checkbox with role="switch" in the tab order and preserve Space-key toggling.',
    'Expose checked, disabled, and aria-invalid on the native input and associate help or error text with aria-describedby.',
    'Use a fieldset with a legend when several switches share one group question.',
  ]}
/>

## API

| Prop | Values | Note |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg` | Default `md`; sets `data-size`. |
| `invalid` | `boolean` | Sets `aria-invalid` on input and `data-invalid` on wrapper. |
| `checked` / `defaultChecked` | `boolean` | Supports controlled and uncontrolled switch state. |
| `children` | `ReactNode` | Renders `tr-switch-label` when present. |
| native checkbox props | `InputHTMLAttributes<HTMLInputElement>` except `children`, `size`, `type` | Passed through to the native checkbox input. |
