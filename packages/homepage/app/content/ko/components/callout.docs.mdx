## 계약

`note`, `tip`, `caution`, `danger`는 작성한 MDX를 애플리케이션 프레임워크와 결합하지 않고 해당 TRAlert 의미에 매핑됩니다. 설명에는 문단, 목록, 코드 블록 같은 MDX 블록 콘텐츠를 사용할 수 있으며 잘못된 HTML 중첩이 발생하지 않습니다.

## 예시

기본 영어 제목이 적절하지 않다면 지역화된 `title`을 사용하세요.

## API

`TRCallout`은 기본 `div` props와 React 19 `ref`를 전달합니다. `title`로 기본 제목을 지역화하거나 교체할 수 있으며, 작성한 children에는 인라인 React 콘텐츠나 블록 수준 MDX를 사용할 수 있습니다.
