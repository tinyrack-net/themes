# Combobox

Searchable single-value selection and freeform autocomplete.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `data-tr-combobox` | `Combobox` | `select` or `freeform`; default `select` | One value only. |
| Part | `data-tr-combobox-input`, `role="combobox"` | `ComboboxInput` | Search input | Owns active descendant and open state. |
| Part | `tr-combobox-content` | `ComboboxContent` | Native popover | Matches the input width. |
| Part | `role="listbox"`, `role="option"` | `ComboboxList`, `ComboboxOption` | Custom option content | Supply `textValue` for non-text content. |
| State | Hidden input, data attributes | Controlled or uncontrolled value, inputValue and open | Optional form `name` | Invalid and disabled state remain explicit. |

<ComponentGuidance whenToUse={['Use select mode for a known set and freeform mode for autocomplete.']} avoid={['Do not use for multiple selection.']} accessibility={['Keep the input label, listbox relationship and active descendant intact.']} />

## API

| Component | Main props | Note |
| --- | --- | --- |
| `Combobox` | `mode`, `value`, `inputValue`, `open`, `name`, `autoSelectOnBlur` | Every state supports controlled and uncontrolled use. |
| `ComboboxOption` | `value`, `textValue`, `disabled` | Custom visual content remains searchable. |
