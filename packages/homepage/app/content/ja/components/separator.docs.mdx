## コントラクト

`orientation` は `horizontal` または `vertical` を受け取り、既定値は `horizontal` です。

## API

Vertical separators default to `role="separator"`. If `role="presentation"` is supplied, `aria-orientation` is omitted to avoid invalid decorative ARIA.
