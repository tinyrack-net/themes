## 계약

| 축 | 동작 |
| --- | --- |
| value | uncontrolled with `defaultValue`, controlled with `value` and `onValueChange` |
| keyboard | arrow keys move selection; Tab enters and leaves the group once |
| form | `name` and `required` apply to the group; individual radios provide values |

## API

`RadioGroup`과 prop/state type은 `@tinyrack/ui/components/radio-group`에서 export됩니다. `Radio.Root`와 `Radio.Indicator`로 조합하세요. root는 ref, native props, state 기반 `className`, `style`, `render`를 보존하며 화살표 키로 선택을 이동하고 Tab은 그룹에 한 번 진입·이탈합니다.
