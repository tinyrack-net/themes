## Contract

Pass h2/h3 heading data and the current heading. `renderLink` may inject a router link while native hash navigation remains the fallback.

## Examples

Update `currentHeading` from an intersection observer while keeping hash navigation controlled by the adapter.

## API

Depth is limited to `2 | 3`. The component renders a desktop rail and native mobile `details` disclosure from the same data.
