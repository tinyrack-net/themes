## 계약

이 Tinyrack wrapper는 Base UI 동작, React 19 ref, native props, event, `render` contract를 보존합니다. Playground control은 잘못된 범위를 정규화하고 `label`, `min`, `max`, `value`, number formatting, 모든 명시적 semantic variant를 다룹니다.

## API

현재 `min`/`max` 범위 안에 `value`를 유지하고 표시 단위에는 `format`, 맥락에 맞는 알림에는 `getAriaValueText`를 사용하세요. `neutral`, `info`, `success`, `warning`, `danger`는 명시적이며 Tinyrack은 암묵적인 low/high threshold API를 추가하지 않습니다.
