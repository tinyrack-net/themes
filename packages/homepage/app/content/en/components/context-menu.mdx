## Contract

`TRContextMenu` attaches commands to content such as a rack row, file, or canvas. Use `TRMenu` when a dedicated trigger always opens the command list. This Tinyrack wrapper preserves Base UI positioning, focus, dismissal, React 19 refs, native props, events, and the `render` contract. Playground controls cover the visible `label`, item `variant`, and `disabledItem` state; open state stays synchronized without appearing as a control.

## API

Every part, prop type, and state type is exported from `@tinyrack/ui/components/context-menu`. Styles remain opt-in through `@tinyrack/ui/components/context-menu.css`.

### `TRContextMenu.Root`

Owns controlled or uncontrolled open state, orientation, focus looping, dismissal, and root disabled behavior.

### `TRContextMenu.Trigger`

Defines the right-click or long-press target. Make it focusable to support Shift+F10 and the platform Context Menu key.

### `TRContextMenu.Portal`

Moves overlay anatomy to `document.body` or a custom container. Use `keepMounted` when an external animation needs the nodes retained.

### `TRContextMenu.Backdrop`

Provides the transparent modal interaction layer behind the popup.

### `TRContextMenu.Positioner`

Places the root popup at pointer or keyboard coordinates and nested popups beside their submenu trigger. An explicit `anchor` overrides pointer anchoring.

### `TRContextMenu.Popup`

Renders the `menu`, owns keyboard navigation and typeahead, and provides the constrained scrolling viewport.

### `TRContextMenu.Arrow`

Renders an optional arrow between a popup and its anchor.

### `TRContextMenu.Group`

Groups related commands.

### `TRContextMenu.GroupLabel`

Labels a command group.

### `TRContextMenu.Item`

Runs a command. Use `disabled`, `closeOnClick`, and the Tinyrack `variant="danger"` option as needed.

### `TRContextMenu.LinkItem`

Renders a semantic link command with `href`; set `closeOnClick` when navigation should also dismiss the menu.

### `TRContextMenu.CheckboxItem`

Represents a boolean command with controlled or uncontrolled checked state.

### `TRContextMenu.CheckboxItemIndicator`

Renders checkbox selection content while its item is checked.

### `TRContextMenu.RadioGroup`

Owns the selected value for a set of mutually exclusive commands.

### `TRContextMenu.RadioItem`

Represents one value in a radio group.

### `TRContextMenu.RadioItemIndicator`

Renders radio selection content while its item is selected.

### `TRContextMenu.SubmenuRoot`

Owns a nested menu's open state and Escape behavior.

### `TRContextMenu.SubmenuTrigger`

Opens a nested menu through pointer or keyboard navigation.

### `TRContextMenu.Separator`

Visually separates command groups.

Provide an explicit button that dispatches the same context-menu action when touch users cannot right-click.
