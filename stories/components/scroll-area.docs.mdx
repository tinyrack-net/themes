# ScrollArea

A custom scrollbar surface that preserves native scrolling.

## Contract

This Tinyrack wrapper preserves Base UI behavior, React 19 refs, native props, events, and the `render` contract. Storybook controls cover `content`, `orientation`.

## Install

<ComponentInstall surfaces={[{ label: 'React', install: 'pnpm add @tinyrack/ui react react-dom', imports: ["import '@tinyrack/ui/core.css';", "import '@tinyrack/ui/components/scroll-area.css';", "import { ScrollArea } from '@tinyrack/ui/components/scroll-area';"] }]} />
