## 계약

이 컴포넌트는 선택 동작만 담당합니다. locale route 계산, navigation 및 저장은 사용하는 애플리케이션이 담당합니다.

## 예시

제어형 locale을 변경하기 전에 공유 content key로 대상 URL을 확인하세요.

## API

`value`와 `onValueChange`는 필수입니다. 각 option에는 렌더링되는 label을 위한 BCP 47 `language`를 포함할 수 있습니다.
