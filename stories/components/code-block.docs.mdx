# CodeBlock

SSR-safe block code with optional client-side Shiki highlighting.

## Contract

| Category | CSS / HTML | React | Values & default | Guidance |
| --- | --- | --- | --- | --- |
| Component | `tr-code-block` on `<pre><code>` | `<CodeBlock>` | Required base surface; SSR-safe | Keep code readable without JavaScript or syntax highlighting. |
| Component | Base class plus generated token spans | `<ShikiCodeBlock>` | Client-highlighted; theme defaults to `github-dark` | Add the optional `shiki` peer only when token highlighting is valuable. |
| Behavior | `data-language` | `language` | Any string for CodeBlock; bundled language for ShikiCodeBlock; default unset | Provide a language when the label or tokenization helps readers. |
| Behavior | `data-wrap="true"` | `wrap` | `boolean`; default `false` | Leave commands and source scrollable; wrap prose-like output deliberately. |
| Native | Native `<pre>` attributes | `HTMLAttributes<HTMLPreElement>` | Passed through | Preserve whitespace and copyable source text. |

## Usage

<ComponentExampleTabs
  id="code-block-basic"
  title="Plain block and Shiki upgrade"
  description="The preview is the SSR-safe base rendering; open the React Shiki tab for the optional client-highlighted equivalent."
  previewLayout="stretch"
  preview={(
    <CodeBlock language="sh">{'pnpm install\npnpm verify'}</CodeBlock>
  )}
  sources={[
    {
      label: 'React',
      language: 'tsx',
      code: String.raw`import '@tinyrack/ui/core/core.css';
import '@tinyrack/ui/components/code-block/code-block.css';
import { CodeBlock } from '@tinyrack/ui/components/code-block/react';

<CodeBlock language="sh">{'pnpm install\\npnpm verify'}</CodeBlock>;`,
    },
    {
      label: 'HTML',
      language: 'html',
      code: String.raw`<pre class="tr-code-block" data-language="sh">
  <code>pnpm install
pnpm verify</code>
</pre>`,
    },
    {
      label: 'React Shiki block',
      language: 'tsx',
      code: String.raw`import '@tinyrack/ui/core/core.css';
import '@tinyrack/ui/components/code-block/code-block.css';
import { ShikiCodeBlock } from '@tinyrack/ui/components/code-block/shiki-react';

<ShikiCodeBlock
  code={'pnpm install\npnpm verify'}
  language="sh"
  theme="github-dark"
/>;`,
    },
  ]}
/>

## Guidance

<ComponentGuidance
  whenToUse={[
    'Present multiline commands, configuration, logs, or source that readers may copy.',
    'Use ShikiCodeBlock when language-aware highlighting materially improves a longer example.',
  ]}
  avoid={[
    'Do not use a block for a short literal embedded in prose; use Code instead.',
    'Do not enable wrapping for source where indentation or aligned columns carry meaning.',
  ]}
  accessibility={[
    'Keep the pre and code structure so whitespace, keyboard selection, and copied text remain predictable.',
    'Treat syntax color as enhancement only; the plain SSR output must stay readable, and long unwrapped lines must scroll inside the block.',
  ]}
/>

## API

| Prop | Values | Note |
| --- | --- | --- |
| `children` | `ReactNode` | Plain `CodeBlock` content. |
| `code` | `string` | `ShikiCodeBlock` source string. |
| `language` | `string` or Shiki bundled language | Sets `data-language`; Shiki uses it for tokenization. |
| `theme` | Shiki bundled theme | Default `github-dark`. |
| `wrap` | `boolean` | Adds `data-wrap="true"`. |
| `className` | `string` | Merged with `tr-code-block`. |
