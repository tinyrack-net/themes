## Contract

| Axis | Behavior |
| --- | --- |
| ownership | selection is owned by the nearest `RadioGroup` |
| identity | every item needs a stable `value`; labels should target a unique `id` |
| state | selected, unselected, read-only, and disabled |

Use `uiSize="sm"` for a smaller radio in dense control surfaces.

## API

`Root`, `Indicator`, and their prop and state types are exported from `@tinyrack/ui/components/radio`. Use them only inside `RadioGroup`. Keep the complete label row at least 24px tall so the visible text enlarges the pointer target beyond the radio glyph.
